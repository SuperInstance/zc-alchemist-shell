## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that applies the fleet's Deadband Protocol to a list of float values.

The Deadband Protocol has three priority levels:
P0: Map negative space (what NOT to do) — here, filter out values that are NaN or infinite.
P1: Find safe channels (where you CAN be) — here, keep only values within a specified safe range [min_val, max_val].
P2: Optimize within channels (best path) — here, round the remaining values to the nearest step size.

Parameters:
- values: list of floats (may contain None, NaN, inf)
- min_val: float, minimum safe value
- max_val: float, maximum safe value
- step: float, step size for rounding (default 1.0)

Return:
- list of filtered and rounded floats, in original order.

Strict priority: Never skip to P2. Handle edge cases gracefully.
```

### **DeepSeek-chat output:**
```python
import math

def deadband_filter(values, min_val, max_val, step=1.0):
    """
    Apply Deadband Protocol filtering to a list of float values.
    
    P0: Remove NaN and infinite values (negative space).
    P1: Keep only values within safe range [min_val, max_val].
    P2: Round remaining values to nearest step.
    
    Args:
        values: List of floats (may contain None, NaN, inf).
        min_val: Minimum safe value.
        max_val: Maximum safe value.
        step: Rounding step size (default 1.0).
    
    Returns:
        List of filtered and rounded floats, preserving order.
    """
    if not isinstance(values, list):
        raise TypeError("values must be a list")
    
    filtered = []
    for v in values:
        # P0: Skip non-float, NaN, infinite
        if not isinstance(v, (int, float)):
            continue
        if math.isnan(v) or math.isinf(v):
            continue
        
        # P1: Check safe range
        if min_val <= v <= max_val:
            # P2: Round to nearest step
            rounded = round(v / step) * step
            filtered.append(rounded)
    
    return filtered
```

## **3. Function 2: `parse_fleet_log`**

**Prompt:**
```
Write a Python function `parse_fleet_log` that parses a structured fleet log line.

Log format: 
"YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE | TILE_ID"

Where:
- YYYY-MM-DD HH:MM:SS is a timestamp
- AGENT is an agent name (e.g., "Navigator", "Alchemist")
- LEVEL is one of: "INFO", "WARN", "ERROR", "DEBUG"
- MESSAGE is a free-text string (may contain pipes)
- TILE_ID is an optional alphanumeric ID (may be empty)

The function should:
- Split the line by " | " (space-pipe-space).
- Return a dict with keys: timestamp, agent, level, message, tile_id.
- If tile_id is missing, set to None.
- Strip whitespace from all fields.
- If the line doesn't match the expected format, return None.

Handle edge cases: empty string, too few/many segments, malformed timestamp.
```

### **DeepSeek-chat output:**
```python
from datetime import datetime

def parse_fleet_log(line):
    """
    Parse a structured fleet log line into a dictionary.
    
    Format: "YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE | TILE_ID"
    
    Args:
        line (str): Log line.
    
    Returns:
        dict with keys: timestamp, agent, level, message, tile_id
        or None if parsing fails.
    """
    if not line or not isinstance(line, str):
        return None
    
    parts = [part.strip() for part in line.split(" | ")]
    if len(parts) < 4:
        return None
    
    # Extract required fields
    timestamp_str = parts[0]
    agent = parts[1]
    level = parts[2]
    message = parts[3]
    
    # Tile_id is optional
    tile_id = parts[4] if len(parts) > 4 else None
    
    # Validate timestamp format
    try:
        timestamp = datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        return None
    
    # Validate level
    if level not in {"INFO", "WARN", "ERROR", "DEBUG"}:
        return None
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "level": level,
        "message": message,
        "tile_id": tile_id
    }
```
