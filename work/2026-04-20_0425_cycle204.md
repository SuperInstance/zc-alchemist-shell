### **2.1 deadband_filter()**

**Spec:** Given a list of float values and a deadband threshold, return a new list where any value within ±threshold of zero is set to 0.0, otherwise keep original value.

**DeepSeek-chat output:**
```python
def deadband_filter(values, threshold):
    """
    Apply a deadband filter to a list of float values.
    
    Any value within +/- threshold of zero is set to 0.0.
    
    Args:
        values (list of float): Input values.
        threshold (float): Deadband threshold (positive).
        
    Returns:
        list of float: Filtered values.
        
    Example:
        >>> deadband_filter([-0.5, -0.1, 0.0, 0.2, 1.5], 0.3)
        [-0.5, 0.0, 0.0, 0.0, 1.5]
    """
    if not isinstance(values, list):
        raise TypeError("values must be a list")
    if not isinstance(threshold, (int, float)):
        raise TypeError("threshold must be numeric")
    if threshold < 0:
        raise ValueError("threshold must be non-negative")
        
    result = []
    for v in values:
        if not isinstance(v, (int, float)):
            raise TypeError(f"Value {v} is not numeric")
        if -threshold <= v <= threshold:
            result.append(0.0)
        else:
            result.append(float(v))
    return result
```

**Groq Llama-70b output:**
```python
def deadband_filter(values, threshold):
    """
    Apply deadband filtering to a list of values.
    
    Args:
        values: List of floats/ints.
        threshold: Float, deadband width.
        
    Returns:
        List of floats with deadband applied.
        
    Example:
        >>> deadband_filter([-0.5, -0.1, 0.0, 0.2, 1.5], 0.3)
        [-0.5, 0.0, 0.0, 0.0, 1.5]
    """
    if threshold < 0:
        raise ValueError("Threshold must be non-negative")
    
    filtered = []
    for val in values:
        try:
            v = float(val)
        except (TypeError, ValueError):
            raise TypeError(f"Invalid value in list: {val}")
        
        if abs(v) <= threshold:
            filtered.append(0.0)
        else:
            filtered.append(v)
    return filtered
```

### **2.2 parse_fleet_log()**

**Spec:** Parse a fleet log line formatted as `"YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE"` into a dict with keys: timestamp, agent, level, message.

**DeepSeek-chat output:**
```python
import datetime

def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line into a dictionary.
    
    Expected format: "YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE"
    
    Args:
        log_line (str): A single log line.
        
    Returns:
        dict: Parsed components with keys: timestamp, agent, level, message.
        
    Example:
        >>> parse_fleet_log("2026-04-20 04:30:00 | Alchemist | INFO | Cycle 204 started")
        {'timestamp': datetime.datetime(2026, 4, 20, 4, 30, 0),
         'agent': 'Alchemist',
         'level': 'INFO',
         'message': 'Cycle 204 started'}
    """
    if not log_line or not isinstance(log_line, str):
        raise ValueError("log_line must be a non-empty string")
    
    parts = log_line.split(" | ", 3)
    if len(parts) != 4:
        raise ValueError(f"Invalid log format. Got {len(parts)} parts, expected 4")
    
    timestamp_str, agent, level, message = parts
    
    try:
        timestamp = datetime.datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        raise ValueError(f"Invalid timestamp format: {timestamp_str}")
    
    return {
        "timestamp": timestamp,
        "agent": agent.strip(),
        "level": level.strip(),
        "message": message.strip()
    }
```

```python
from datetime import datetime

def parse_fleet_log(log_line):
    """
    Parse fleet log line.
    
    Format: "YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE"
    
    Args:
        log_line: String log line.
        
    Returns:
        Dictionary with timestamp (datetime), agent, level, message.
        
    Example:
        >>> parse_fleet_log("2026-04-20 04:30:00 | Alchemist | INFO | Cycle 204 started")
        {'timestamp': datetime(2026, 4, 20, 4, 30, 0),
         'agent': 'Alchemist',
         'level': 'INFO',
         'message': 'Cycle 204 started'}
    """
    if not log_line:
        return {}
    
    segments = log_line.split(" | ", 3)
    if len(segments) < 4:
        raise ValueError(f"Malformed log line: {log_line}")
    
    date_str, agent, level, msg = segments
    
    try:
        dt = datetime.strptime(date_str, "%Y-%m-%d %H:%M:%S")
    except Exception as e:
        raise ValueError(f"Bad timestamp {date_str}: {e}")
    
    return {
        "timestamp": dt,
        "agent": agent,
        "level": level,
        "message": msg
    }
```
