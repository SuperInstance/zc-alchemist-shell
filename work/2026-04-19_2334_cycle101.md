## **2. Function Specifications**

### **2.1 deadband_filter**
**Input:** List of floats `readings`, float `deadband_min`, float `deadband_max`.
**Output:** List of floats filtered via Deadband Protocol.
**Logic:**
- P0: Remove any reading <= `deadband_min` (negative space).
- P1: Keep readings where `deadband_min` < reading < `deadband_max` (safe channel).
- P2: For readings in safe channel, apply optional scaling (scale=0.9).
- Return filtered list.

### **2.2 constraint_snap**
**Input:** List of 2D points `[(x1,y1), ...]`, target point `(tx,ty)`, float `tolerance`.
**Output:** Target point snapped to nearest point within tolerance, else None.
**Logic:**
- Compute Euclidean distances.
- Find points within tolerance.
- Return closest point if any, else None.

### **2.3 tile_encoder**
**Input:** String `log_text`, str `agent_role`, int `cycle`.
**Output:** Dict representing a PLATO training tile.
**Format:**

  "agent": "Alchemist",
  "role": agent_role,
  "cycle": cycle,

from typing import List, Tuple, Optional, Dict, Any
from datetime import datetime
import math

def deadband_filter(readings: List[float], 
                    deadband_min: float, 
                    deadband_max: float, 
                    scale: float = 0.9) -> List[float]:
    """
    Apply Deadband Protocol filtering to sensor readings.
    
    P0: Remove readings in negative space (<= deadband_min).
    P1: Keep readings in safe channel (deadband_min < reading < deadband_max).
    P2: Apply scaling to readings in safe channel.
    
    Args:
        readings: List of sensor readings.
        deadband_min: Minimum threshold for negative space.
        deadband_max: Maximum threshold for safe channel.
        scale: Scaling factor for P2 optimization (default 0.9).
    
    Returns:
        Filtered and scaled readings.
    """
    # P0: Map negative space (remove unsafe readings)
    safe_readings = [r for r in readings if r > deadband_min]
    
    # P1: Find safe channels
    channel_readings = [r for r in safe_readings if r < deadband_max]
    
    # P2: Optimize within channels
    optimized = [r * scale for r in channel_readings]
    
    return optimized


def constraint_snap(points: List[Tuple[float, float]], 
                    target: Tuple[float, float], 
                    tolerance: float) -> Optional[Tuple[float, float]]:
    """
    Snap target point to nearest point within tolerance using 2D Euclidean distance.
    
    Args:
        points: List of (x, y) points.
        target: (x, y) point to snap.
        tolerance: Maximum distance for snapping.
    
    Returns:
        Closest point within tolerance, or None if none found.
    """
    tx, ty = target
    closest_point = None
    min_distance = float('inf')
    
    for (px, py) in points:
        distance = math.sqrt((px - tx)**2 + (py - ty)**2)
        if distance <= tolerance and distance < min_distance:
            min_distance = distance
            closest_point = (px, py)
    
    return closest_point


def tile_encoder(log_text: str, agent_role: str, cycle: int) -> Dict[str, Any]:

        cycle: Current cycle number.

from typing import List, Tuple, Optional, Dict
import math
from datetime import datetime

def deadband_filter(readings: List[float], deadband_min: float, deadband_max: float, scale: float = 0.9) -> List[float]:
    """
    Implements the Deadband Protocol for sensor data filtering.
    
    Steps:
    1. Remove readings <= deadband_min (negative space).
    2. Keep readings where deadband_min < value < deadband_max (safe channel).
    3. Scale remaining readings by factor.
    
    Parameters:
        readings: input sensor values
        deadband_min: lower bound
        deadband_max: upper bound  
        scale: scaling factor (default 0.9)
    
    Returns:
        filtered and scaled readings
    """
    # Step 1: eliminate negative space
    filtered = [r for r in readings if r > deadband_min]
    
    # Step 2: keep only safe channel
    filtered = [r for r in filtered if r < deadband_max]
    
    # Step 3: apply scaling
    result = [r * scale for r in filtered]
    
    return result


def constraint_snap(points: List[Tuple[float, float]], target: Tuple[float, float], tolerance: float) -> Optional[Tuple[float, float]]:
    """
    Snaps a target point to the nearest point within a given tolerance.
    
    Parameters:
        points: list of candidate points
        target: point to snap
        tolerance: maximum snap distance
    
    Returns:
        closest point within tolerance, or None
    """
    tx, ty = target
    best_point = None
    best_dist = float('inf')
    
    for (px, py) in points:
        dist = math.hypot(px - tx, py - ty)
        if dist <= tolerance and dist < best_dist:
            best_dist = dist
            best_point = (px, py)
    
    return best_point


def tile_encoder(log_text: str, agent_role: str, cycle: int) -> Dict:

    Creates a training tile dictionary from log text.
    
    Parameters:
        log_text: the content to store
        agent_role: role of the agent
        cycle: cycle number
    
    Returns:
        tile dictionary
    """
    return {
        "timestamp": datetime.utcnow().isoformat() + "Z",
        "agent": "Alchemist",
        "role": agent_role,
        "cycle": cycle,
        "content": log_text,
        "metadata": {
            "version": "1.0",
            "phase": 4,
