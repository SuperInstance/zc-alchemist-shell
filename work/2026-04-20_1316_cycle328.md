## **2. Function 1: Simple Utility**

**Prompt:** "Write a Python function `clean_text` that takes a string, removes all non-alphanumeric characters except spaces, converts to lowercase, and strips extra whitespace."

### **DeepSeek-chat Output:**
```python
import re

def clean_text(text: str) -> str:
    """
    Clean a text string by removing non-alphanumeric characters (except spaces),
    converting to lowercase, and normalizing whitespace.
    
    Args:
        text (str): Input string
        
    Returns:
        str: Cleaned string
    """
    if not isinstance(text, str):
        raise TypeError("Input must be a string")
    
    # Keep alphanumeric and spaces
    cleaned = re.sub(r'[^a-zA-Z0-9\s]', '', text)
    # Convert to lowercase
    cleaned = cleaned.lower()
    # Normalize whitespace: replace multiple spaces with single, strip
    cleaned = re.sub(r'\s+', ' ', cleaned).strip()
    
    return cleaned
```

### **Groq Llama-3.3-70b-Specdec Output:**
```python

    """
    Remove non-alphanumeric characters except spaces,
    convert to lowercase, and strip extra whitespace.
    """
```

### **Comparison:**

- **Style:** Both follow PEP 8. DeepSeek's `re.sub(r'\s+', ' ', cleaned)` vs Groq's `' '.join(text.split())` — latter is more Pythonic for whitespace normalization.

**Verdict:** DeepSeek more robust (validation, documentation). Groq more concise and Pythonic in whitespace handling.

## **3. Function 2: Data Processing**

**Prompt:** "Write a Python function `parse_log_line` that takes a log line string in format 'TIMESTAMP LEVEL [MODULE] MESSAGE' and returns a dict with keys timestamp, level, module, message. Handle malformed lines gracefully."

### **DeepSeek-chat Output:**
```python
from datetime import datetime
import re

def parse_log_line(log_line: str) -> dict:
    """
    Parse a structured log line into its components.
    
    Expected format: 'TIMESTAMP LEVEL [MODULE] MESSAGE'
    Example: '2023-10-05 14:30:22 INFO [network] Connection established'
    
    Args:
        log_line (str): Log line string
        
    Returns:
        dict: Parsed components or error dict
    """
    if not log_line or not isinstance(log_line, str):
        return {"error": "Invalid input", "original": log_line}
    
    pattern = r'^(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) (\w+) \[([^\]]+)\] (.+)$'
    match = re.match(pattern, log_line)
    
    if not match:
        return {
            "error": "Format mismatch",
            "original": log_line,
            "expected_format": "TIMESTAMP LEVEL [MODULE] MESSAGE"
        }
    
    timestamp_str, level, module, message = match.groups()
    
    try:
        timestamp = datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        timestamp = timestamp_str  # Keep as string if parsing fails
    
    return {
        "timestamp": timestamp,
        "level": level.upper(),
        "module": module,
        "message": message.strip()
    }
```

## **4. Function 3: Algorithm Implementation**

**Prompt:** "Write a function `find_cycle_start` that takes a linked list head node (class Node with .val and .next) and returns the node where cycle begins, or None if no cycle. Use Floyd's cycle detection algorithm."

### **DeepSeek-chat Output:**
```python
class Node:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def find_cycle_start(head: Node) -> Node | None:
    """
    Find the starting node of a cycle in a linked list using Floyd's algorithm.
    
    Args:
        head (Node): Head node of the linked list
        
    Returns:
        Node | None: Starting node of cycle, or None if no cycle
    """
    if not head or not head.next:
        return None
    
    slow = fast = head
    
    # First phase: detect if cycle exists
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            break
    else:
        return None  # No cycle
    
    # Second phase: find cycle start
    slow = head
    while slow != fast:
        slow = slow.next
        fast = fast.next
    
    return slow
```
