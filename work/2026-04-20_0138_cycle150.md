### **Function 1: Deadband Filter**

**Prompt Details:**
- Name: `apply_deadband_protocol`
- Description: Given a list of float values, classify each into P0 (negative space), P1 (safe channel), or P2 (optimization zone) based on deadband thresholds.
- Input: `values: List[float], p0_threshold: float = -10.0, p1_min: float = 0.0, p1_max: float = 100.0`
- Output: `List[Tuple[float, str]]` where str is "P0", "P1", or "P2"
- Constraints: Values below p0_threshold are P0. Values between p1_min and p1_max (inclusive) are P1. Values above p1_max are P2. Values between p0_threshold and p1_min are undefined (treat as P0 for safety).

**DeepSeek-chat Output:**
```python
from typing import List, Tuple

def apply_deadband_protocol(
    values: List[float],
    p0_threshold: float = -10.0,
    p1_min: float = 0.0,
    p1_max: float = 100.0
) -> List[Tuple[float, str]]:
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of float values.
    
    Args:
        values: List of float values to classify.
        p0_threshold: Values below this are P0 (negative space).
        p1_min: Lower bound of P1 (safe channel), inclusive.
        p1_max: Upper bound of P1 (safe channel), inclusive.
        
    Returns:
        List of tuples (original_value, classification).
        Classification is "P0", "P1", or "P2".
    """
    result = []
    for v in values:
        if v < p0_threshold:
            result.append((v, "P0"))
        elif p1_min <= v <= p1_max:
            result.append((v, "P1"))
        elif v > p1_max:
            result.append((v, "P2"))
        else:  # v between p0_threshold and p1_min (undefined zone)
            result.append((v, "P0"))  # Default to P0 for safety
    return result


if __name__ == "__main__":
    # Example usage
    test_values = [-15.5, -5.0, 0.0, 50.0, 100.0, 150.0]
    classified = apply_deadband_protocol(test_values)
    for val, cls in classified:
        print(f"{val:7} -> {cls}")
```

def apply_deadband_protocol(values: List[float], 
                           p0_threshold: float = -10.0,
                           p1_min: float = 0.0,
                           p1_max: float = 100.0) -> List[Tuple[float, str]]:
    """
    Classifies each value in `values` according to the Deadband Protocol.
    
    P0: value < p0_threshold (negative space)
    P1: p1_min <= value <= p1_max (safe channel)
    P2: value > p1_max (optimization zone)

    as a safety default (undefined region).
    """

    for val in values:
        if val < p0_threshold:
            classifications.append((val, "P0"))
        elif p1_min <= val <= p1_max:
            classifications.append((val, "P1"))
        elif val > p1_max:
            classifications.append((val, "P2"))
        else:  # p0_threshold <= val < p1_min
            classifications.append((val, "P0"))  # fallback to P0

### **Function 2: Tile Validator**

**Prompt Details:**
- Name: `validate_training_tile`
- Description: Validate a training tile dictionary against the fleet's expected schema for submission to PLATO room server.
- Input: `tile: Dict[str, Any]`
- Output: `Tuple[bool, List[str]]` where bool is validity, list is error messages.
- Constraints: Required keys: `agent_id`, `cycle`, `phase`, `content`, `timestamp`. `content` must be non-empty string. `cycle` and `phase` must be non-negative ints. `timestamp` must be ISO 8601 string.

**DeepSeek-chat Output:**
```python
from typing import Dict, Any, Tuple, List
from datetime import datetime

def validate_training_tile(tile: Dict[str, Any]) -> Tuple[bool, List[str]]:
    """
    Validate a training tile against fleet schema.
    
    Required keys: agent_id, cycle, phase, content, timestamp.
    
    Args:
        tile: Dictionary containing tile data.
        
    Returns:
        Tuple of (is_valid, list_of_error_messages).
    """
    required_keys = {"agent_id", "cycle", "phase", "content", "timestamp"}
    errors = []
    
    # Check presence of all required keys
    missing_keys = required_keys - set(tile.keys())
    if missing_keys:
        errors.append(f"Missing required keys: {sorted(missing_keys)}")
    
    # Validate agent_id
    if "agent_id" in tile and not isinstance(tile["agent_id"], str):
        errors.append("agent_id must be a string")
    
    # Validate cycle
    if "cycle" in tile:
        if not isinstance(tile["cycle"], int):
            errors.append("cycle must be an integer")
        elif tile["cycle"] < 0:
            errors.append("cycle must be non-negative")
    
    # Validate phase
    if "phase" in tile:
        if not isinstance(tile["phase"], int):
            errors.append("phase must be an integer")
        elif tile["phase"] < 0:
            errors.append("phase must be non-negative")
    
    # Validate content
    if "content" in tile:
        if not isinstance(tile["content"], str):
            errors.append("content must be a string")
        elif len(tile["content"].strip()) == 0:
            errors.append("content must be non-empty")
    
    # Validate timestamp
    if "timestamp" in tile:
        if not isinstance(tile["timestamp"], str):
            errors.append("timestamp must be a string")
        else:
            try:
                datetime.fromisoformat(tile["timestamp"].replace('Z', '+00:00'))
            except ValueError:
                errors.append("timestamp must be valid ISO 8601 format")
    
    is_valid = len(errors) == 0
    return (is_valid, errors)
