## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

The function should take two arguments:
1. `signal` (float): An input value.
2. `deadband_range` (tuple of two floats): The (lower, upper) bounds of a "deadband" or safe channel.

It should return a tuple: (`priority_level`, `action`).

DOCTRINE:
- P0 (Map negative space): If `signal` is BELOW the lower bound, priority is 0, action is "avoid_lower".
- P1 (Find safe channels): If `signal` is WITHIN the deadband range (inclusive), priority is 1, action is "hold_channel".
- P2 (Optimize within channels): If `signal` is ABOVE the upper bound, priority is 2, action is "optimize_upper".

Include a clear docstring and type hints. Add a brief example in the docstring.
```

### **DeepSeek-chat Output:**
```python
from typing import Tuple

def deadband_filter(signal: float, deadband_range: Tuple[float, float]) -> Tuple[int, str]:
    """
    Apply the Deadband Protocol to a signal.

    The Deadband Protocol defines three priority levels:
    - P0: Signal below safe channel -> avoid lower region
    - P1: Signal within safe channel -> hold position
    - P2: Signal above safe channel -> optimize upper region

    Args:
        signal: Input value to evaluate.
        deadband_range: Tuple (lower_bound, upper_bound) defining the safe channel.

    Returns:
        Tuple of (priority_level, action_description).

    Example:
        >>> deadband_filter(5.0, (4.0, 6.0))
        (1, 'hold_channel')
    """
    lower, upper = deadband_range

    if signal < lower:
        return (0, "avoid_lower")
    elif lower <= signal <= upper:
        return (1, "hold_channel")
    else:  # signal > upper
        return (2, "optimize_upper")
```

## **3. Function 2: `parse_fleet_context`**

**Prompt to both models:**
```
Write a Python function called `parse_fleet_context` that parses a markdown string (like the FLEET-CONTEXT.md file) and extracts key sections into a dictionary.

The function should:
1. Take a string `md_content` (the markdown text).
2. Use simple markdown parsing (rely on heading levels `##` and `###`). Assume consistent formatting.
3. Extract the text under these specific top-level (`##`) headings: "Who We Are", "Fleet Doctrine", "Your Purpose", "Key Technologies", "Architecture", "Rules".
4. Return a dict where keys are the heading names (without markdown) and values are the concatenated text of that section (including any subsections). Strip leading/trailing whitespace.
5. If a heading is not found, its value should be an empty string.

Include error handling for malformed input. Include a docstring and type hints.
```

### **DeepSeek-chat Output:**
```python
import re
from typing import Dict

def parse_fleet_context(md_content: str) -> Dict[str, str]:
    """
    Parse a fleet context markdown string and extract major sections.

    Args:
        md_content: Markdown text from a file like FLEET-CONTEXT.md.

    Returns:
        Dictionary mapping section names to their content.
        Expected keys: "Who We Are", "Fleet Doctrine", "Your Purpose",
                      "Key Technologies", "Architecture", "Rules".

    Example:
        context_text = \"\"\"## Who We Are\\nCocapn fleet.\\n## Fleet Doctrine\\nP0...\"\"\"
        result = parse_fleet_context(context_text)
        # result[\"Who We Are\"] == \"Cocapn fleet.\"
    """
    # Define the sections we want to extract
    target_sections = [
        "Who We Are",
        "Fleet Doctrine",
        "Your Purpose",
        "Key Technologies",
        "Architecture",
        "Rules"
    ]

    result = {section: "" for section in target_sections}

    # Split by top-level (##) headings
    # Pattern matches "## Heading" and captures heading text
    pattern = r'^##\s+(.+?)$'

    lines = md_content.split('\n')
    current_section = None
    current_content = []

    for line in lines:
        match = re.match(pattern, line)
        if match:
            # Save previous section if it's one we care about
            if current_section in target_sections:
                result[current_section] = '\n'.join(current_content).strip()

            # Start new section
            current_section = match.group(1).strip()
            current_content = []
        elif current_section is not None:
            current_content.append(line)

    # Don't forget the last section
    if current_section in target_sections:
        result[current_section] = '\n'.join(current_content).strip()

    return result
```
