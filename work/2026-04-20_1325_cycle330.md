## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input:
- `readings`: list of floats (sensor readings)
- `deadband`: float (threshold, positive)

Protocol logic (P0/P1/P2):
- P0 (negative space): If any reading is NaN or infinite, raise ValueError.
- P1 (safe channels): Filter readings where abs(reading) <= deadband. These are "in channel".
- P2 (optimize): From in‑channel readings, return the one closest to zero (most neutral).

If no readings are in channel, return None.

Include a docstring and a simple example in the function.
```

### **DeepSeek‑chat output:**
```python
def deadband_filter(readings, deadband):
    """
    Apply Deadband Protocol (P0/P1/P2) to sensor readings.

    Parameters
    ----------
    readings : list of float
        Raw sensor readings.
    deadband : float
        Positive threshold defining the safe channel.

    Returns
    -------
    float or None
        The in‑channel reading closest to zero, or None if none are in channel.

    Raises
    ------
    ValueError
        If any reading is NaN or infinite (P0 violation).

    Example
    -------
    >>> deadband_filter([1.2, -0.5, 3.0, -0.1], 1.0)
    -0.1
    """
    # P0: Map negative space
    for r in readings:
        if not isinstance(r, (int, float)) or not math.isfinite(r):
            raise ValueError(f"Invalid reading: {r}")

    # P1: Find safe channels
    in_channel = [r for r in readings if abs(r) <= deadband]

    # P2: Optimize within channels
    if not in_channel:
        return None
    # Return reading with minimum absolute value
    return min(in_channel, key=abs)
```

import math

def deadband_filter(readings, deadband):
    """
    Implements the Deadband Protocol (P0/P1/P2) for sensor data.

    P0: Check for invalid data (NaN, infinity).
    P1: Identify readings within the deadband (safe channel).
    P2: Select the reading closest to zero from the safe channel.

    Args:
        readings (list): List of numerical readings.
        deadband (float): Positive deadband threshold.

    Returns:
        float or None: The optimal reading within deadband, or None.

    Raises:
        ValueError: If any reading is NaN or infinite.

    Example:
        >>> deadband_filter([0.3, -0.7, 1.5, -0.2], 1.0)
        -0.2
    """
    # P0: Negative space detection
    for r in readings:
        if math.isnan(r) or math.isinf(r):
            raise ValueError(f"Invalid reading detected: {r}")

    # P1: Safe channel identification
    safe = [r for r in readings if abs(r) <= deadband]

    # P2: Optimization
    if not safe:
        return None
    # Find element with smallest absolute value
    return min(safe, key=lambda x: abs(x))
```

**Comparison:**
- **Correctness:** Both implement P0/P1/P2 correctly. DeepSeek uses `math.isfinite`, Groq uses separate `isnan`/`isinf`. Both raise `ValueError`.
- **Style:** DeepSeek’s docstring includes a concrete example with output. Groq’s docstring is slightly more structured (Args/Returns/Raises). Both are clear.
- **Edge cases:** Both handle empty safe channel → `None`. DeepSeek adds a type check (`isinstance(r, (int, float))`), which may be overly strict if readings are `numpy` types.
- **Verdict:** Tie. Both produce functionally equivalent, clean code.

**Prompt:**

Write a Python function `parse_fleet_log` that parses a semi‑structured fleet log line.

Log format (example):

- Handle extra spaces gracefully.

Assume the log always contains at least a timestamp and agent fields.

Include error handling for malformed lines (return None).
```

### **DeepSeek‑chat output:**
```python
def parse_fleet_log(log_line):
    """
    Parse a fleet log line into a structured dictionary.

    Parameters
    ----------
    log_line : str
        A log line in format "TIMESTAMP | KEY1: VAL1 | KEY2: VAL2 ..."

    Returns
    -------
    dict or None
        Dictionary with lowercased keys, or None if line is malformed.

    Example
    -------
    >>> parse_fleet_log("2026‑04‑20 13:45:23 | AGENT: Alchemist | CYCLE: 330")
    {'timestamp': '2026‑04‑20 13:45:23', 'agent': 'Alchemist', 'cycle': '330'}
    """
    if not log_line or '|' not in log_line:
        return None

    try:
        parts = [p.strip() for p in log_line.split('|')]
        result = {}
        for part in parts:
            if ':' in part:
                key, val = part.split(':', 1)
                result[key.strip().lower()] = val.strip()
            else:
                # First part is timestamp if no colon
                if 'timestamp' not in result:
                    result['timestamp'] = part.strip()
        return result
    except Exception:
        return None
```
