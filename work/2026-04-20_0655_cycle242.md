**Prompt for both models:**
"Write a Python function called `validate_deadband_protocol` that takes a list of strings `steps`. The function must validate that the list correctly follows the Deadband Protocol: it must contain only the strings "P0", "P1", or "P2", and they must appear in strict priority order P0 -> P1 -> P2 (e.g., ["P0", "P1", "P2"] is valid, ["P0", "P2"] is invalid because it skips P1, ["P1", "P0"] is invalid due to order). Return a tuple (bool, str). If valid, return (True, ""). If invalid, return (False, "error message") describing the first violation found."

**DeepSeek-chat Output:**
```python
import re

def validate_deadband_protocol(steps: list[str]) -> tuple[bool, str]:
    """Validate a sequence of Deadband Protocol steps."""
    valid_steps = {"P0", "P1", "P2"}
    priority_order = {"P0": 0, "P1": 1, "P2": 2}
    
    # Check for invalid step strings
    for i, step in enumerate(steps):
        if step not in valid_steps:
            return (False, f"Invalid step '{step}' at position {i}. Must be 'P0', 'P1', or 'P2'.")
    
    # Check for correct order
    last_priority = -1
    for i, step in enumerate(steps):
        current_priority = priority_order[step]
        if current_priority < last_priority:
            return (False, f"Priority violation at position {i}: '{step}' appears after a higher-priority step.")
        if current_priority > last_priority + 1:
            skipped = [p for p in ["P0", "P1", "P2"] if priority_order[p] == last_priority + 1]
            if skipped:
                return (False, f"Priority gap at position {i}: skipped '{skipped[0]}'.")
        last_priority = current_priority
    
    return (True, "")
```

### **Function 2: `parse_fleet_log_line`**

**Prompt for both models:**
"Write a Python function `parse_fleet_log_line(line: str) -> dict`. It parses a fleet log line with format: '2026-04-20 07:30:15 [AGENT] CYCLE 123 - Message here'. Return a dict with keys: 'timestamp' (string), 'agent' (string), 'cycle' (int), 'message' (string). Use regex for robustness. Handle potential malformed lines by returning None."

**DeepSeek-chat Output:**
```python
import re

def parse_fleet_log_line(line: str) -> dict | None:
    """Parse a standardized fleet log line."""
    pattern = r'^(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) \[([^\]]+)\] CYCLE (\d+) - (.+)$'
    match = re.match(pattern, line.strip())
    
    if not match:
        return None
    
    timestamp, agent, cycle_str, message = match.groups()
    
    try:
        cycle = int(cycle_str)
    except ValueError:
        return None
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "message": message.strip()
    }
```

    """

    }
