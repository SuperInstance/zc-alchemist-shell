## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Context:
- The Deadband Protocol has three priority levels: P0 (avoid negative space), P1 (find safe channels), P2 (optimize within channels).
- Input: A list of float values `samples` representing sensor readings or decision scores.
- Parameters: `deadband_min` (float), `deadband_max` (float), `p0_threshold` (float).
- Logic:
  1. P0: If any value <= `p0_threshold`, return empty list (negative space detected).
  2. P1: Filter values to keep only those inside the deadband: `deadband_min <= v <= deadband_max`.
  3. P2: If filtered list is non-empty, return it; else return original list (no safe channel, fallback).
- Include type hints and a docstring.
- Add a simple example in the docstring.
```

### **DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(
    samples: List[float],
    deadband_min: float,
    deadband_max: float,
    p0_threshold: float
) -> List[float]:
    """
    Apply the fleet's Deadband Protocol to filter sensor readings.

    Steps:
    1. P0 (Negative space): If any sample <= p0_threshold, return [].
    2. P1 (Safe channel): Keep samples within [deadband_min, deadband_max].
    3. P2 (Optimize): If P1 result non-empty, return it; else return original samples.

    Example:
    >>> deadband_filter([1.0, 2.0, 3.0, 4.0], 2.0, 3.0, 0.5)
    [2.0, 3.0]
    >>> deadband_filter([0.1, 2.0, 3.0], 2.0, 3.0, 0.5)
    []
    >>> deadband_filter([5.0, 6.0], 2.0, 3.0, 0.5)
    [5.0, 6.0]

    Args:
        samples: List of float sensor readings.
        deadband_min: Minimum inclusive bound for safe channel.
        deadband_max: Maximum inclusive bound for safe channel.
        p0_threshold: Threshold for P0 negative space detection.

    Returns:
        Filtered list of floats.
    """
    # P0: Map negative space
    if any(s <= p0_threshold for s in samples):
        return []

    # P1: Find safe channels
    safe = [s for s in samples if deadband_min <= s <= deadband_max]

    # P2: Optimize within channels (or fallback)
    return safe if safe else samples
```

## **3. Function 2: `parse_fleet_log`**

**Prompt:**
```
Write a Python function `parse_fleet_log` that parses a structured log line from the Cocapn fleet.

Log format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"
Example: "2026-04-20T14:30:00Z | Alchemist | 333 | ACTIVE | Testing model comparison"

Rules:
- Fields are separated by " | ".
- TIMESTAMP is ISO 8601 format.
- AGENT is a string (no spaces).
- CYCLE is an integer.
- STATUS is one of: ACTIVE, IDLE, ERROR, COMPLETE.
- MESSAGE can contain any characters, including pipes (escaped as \| in original, but not in our simplified version).
- If the line does not match exactly 5 fields, raise a ValueError.
- Return a dict with keys: timestamp (datetime object), agent (str), cycle (int), status (str), message (str).
- Use datetime.fromisoformat for parsing.

Include type hints and a docstring with an example.
```

### **DeepSeek-chat output:**
```python
from datetime import datetime
from typing import Dict, Any

def parse_fleet_log(log_line: str) -> Dict[str, Any]:
    """
    Parse a structured fleet log line into a dictionary.

    Format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"
    Example: "2026-04-20T14:30:00Z | Alchemist | 333 | ACTIVE | Testing model comparison"

    Args:
        log_line: A string in the fleet log format.

    Returns:
        Dictionary with keys: timestamp (datetime), agent (str), cycle (int),
        status (str), message (str).

    Raises:
        ValueError: If line does not have exactly 5 fields or cycle is not an integer.
    """
    parts = log_line.split(" | ")
    if len(parts) != 5:
        raise ValueError(f"Log line must have exactly 5 fields, got {len(parts)}: {log_line}")

    timestamp_str, agent, cycle_str, status, message = parts

    # Validate status
    valid_statuses = {"ACTIVE", "IDLE", "ERROR", "COMPLETE"}
    if status not in valid_statuses:
        raise ValueError(f"Invalid status '{status}'. Must be one of {valid_statuses}")

    # Parse
    try:
        timestamp = datetime.fromisoformat(timestamp_str)
        cycle = int(cycle_str)
    except ValueError as e:
        raise ValueError(f"Failed to parse timestamp or cycle: {e}")

    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message.strip()
    }
```
