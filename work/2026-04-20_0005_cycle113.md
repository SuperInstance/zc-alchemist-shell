## **2. Function 1: Deadband Filter**

**Prompt:**
```
Write a Python function called `apply_deadband_protocol` that implements the Cocapn fleet's Deadband Protocol.

Input: A list of potential actions, where each action is a dictionary with keys `"name"` (str) and `"p0_violation"` (bool).
Logic:
- P0: First, filter OUT any action where `p0_violation` is True. These are the "rocks" to avoid.
- P1: From the remaining actions, if any exist, return a list of their names. These are the "safe channels."
- P2: If multiple safe channels exist, the protocol says to "optimize within channels." For this simple version, just return the list. A full implementation would choose the best one.

If all actions are P0 violations (no safe channels), return an empty list.

Include a brief docstring.
```

### **DeepSeek-chat Output:**
```python
def apply_deadband_protocol(actions):
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of potential actions.

    Args:
        actions (list): List of dicts with keys 'name' (str) and 'p0_violation' (bool).

    Returns:
        list: Names of actions that pass P0 (safe channels). Empty list if no safe actions.

    # P1: Find safe channels - extract names
    safe_channel_names = [action['name'] for action in safe_actions]

    # In a real implementation, you would apply optimization logic here
    return safe_channel_names
```

### **Groq Llama-3.3-70b Output:**

```python
    """
    Implements the Deadband Protocol (P0/P1/P2) for action selection.

    Parameters:
    actions (list of dict): Each dict must have 'name' (str) and 'p0_violation' (bool).

    Returns:
    list of str: Names of actions that are in safe channels (P1). Empty if no safe actions.
    """
    # P0: Eliminate actions with p0_violation=True

    safe_channels = [act['name'] for act in p0_filtered]

```

*   **Correctness:** Both correctly implement P0 filtering and P1 listing. Both handle the empty case.

DeepSeek uses `.get('p0_violation', False)` which is safer if the key is missing; Groq assumes the key exists (`act['p0_violation']`).
*   **Efficiency:** Groq adds an explicit `if not p0_filtered:` check, which is slightly more explicit but adds a line. DeepSeek's list comprehension for `safe_channel_names` would simply produce an empty list, which is also correct.
*   **Adherence to Spec:** Both meet the specification. Groq's output includes a comment about "Future enhancement," which shows a hint of forward-thinking.

## **3. Function 2: Training Tile Validator**

**Prompt:**
```
Write a Python function called `validate_tile` that checks if a dictionary representing a "training tile" (agent work output) is valid for submission to the fleet's PLATO room server.

A valid tile must have:
1. A `content` key (str) that is non-empty.
2. An `agent_id` key (str) that is non-empty.
3. A `cycle` key (int) that is a positive integer.
4. A `phase` key (int) between 1 and 4 (inclusive).
5. An optional `tags` key, but if present, it must be a list of strings.

The function should return a tuple: (bool, str). The bool is True if valid, False otherwise. The str is an error message if invalid, or "OK" if valid.
```

### **DeepSeek-chat Output:**
```python
def validate_tile(tile):
    """
    Validate a training tile for submission to PLATO room server.

    Args:
        tile (dict): The tile dictionary to validate.

    Returns:
        tuple: (is_valid(bool), message(str))
    """
    # Check required keys
    required_keys = ['content', 'agent_id', 'cycle', 'phase']
    for key in required_keys:
        if key not in tile:
            return False, f"Missing required key: {key}"
    
    # Check content
    if not isinstance(tile['content'], str) or not tile['content'].strip():
        return False, "Content must be a non-empty string"
    
    # Check agent_id
    if not isinstance(tile['agent_id'], str) or not tile['agent_id'].strip():
        return False, "Agent ID must be a non-empty string"
    
    # Check cycle
    if not isinstance(tile['cycle'], int) or tile['cycle'] <= 0:
        return False, "Cycle must be a positive integer"
    
    # Check phase
    if not isinstance(tile['phase'], int) or tile['phase'] < 1 or tile['phase'] > 4:
        return False, "Phase must be an integer between 1 and 4"
    
    # Check optional tags
    if 'tags' in tile:
        if not isinstance(tile['tags'], list):
            return False, "Tags must be a list if present"
        for tag in tile['tags']:
            if not isinstance(tag, str):
                return False, "All tags must be strings"
    
    return True, "OK"
```
