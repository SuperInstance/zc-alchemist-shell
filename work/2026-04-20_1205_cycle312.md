## **2. Function 1: `deadband_filter`**

**Prompt:** "Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol. The function should take a list of floating-point numbers `decisions` and return a filtered list. The protocol has three strict priorities: P0 (map negative space): remove any values <= 0.0. P1 (find safe channels): from the remaining values, keep only those between 0.0 and 100.0 (exclusive). P2 (optimize within channels): from the P1-safe values, return only those above a given `threshold` (default 50.0). The function must apply P0, then P1, then P2 in that strict order. Include a docstring and a simple example."

### **DeepSeek-chat Output:**
```python
def deadband_filter(decisions, threshold=50.0):
    """
    Apply the Cocapn fleet's Deadband Protocol to a list of decisions.

    Protocol:
    P0 (Negative Space): Remove all values <= 0.0
    P1 (Safe Channels): Keep values where 0.0 < value < 100.0
    P2 (Optimize): Keep values > threshold (default 50.0)

    Strict priority: P0 -> P1 -> P2. Never skip steps.

    Args:
        decisions (list[float]): Input list of decision values.
        threshold (float): P2 optimization threshold. Default 50.0.

    Returns:
        list[float]: Filtered decisions after applying Deadband Protocol.
    """
    # P0: Map negative space (remove <= 0.0)
    p0_safe = [d for d in decisions if d > 0.0]
    
    # P1: Find safe channels (0.0 < value < 100.0)
    p1_safe = [d for d in p0_safe if d < 100.0]
    
    # P2: Optimize within channels (value > threshold)
    p2_optimized = [d for d in p1_safe if d > threshold]
    
    return p2_optimized

## **3. Function 2: `parse_fleet_context`**

**Prompt:** "Write a Python function `parse_fleet_context` that takes a markdown string (content like FLEET-CONTEXT.md) and returns a dictionary with keys: 'agents', 'technologies', 'rules'. Extract the lists from the relevant sections. Assume sections are denoted by '##' headers. Be robust to minor formatting variations. Include type hints and error handling for missing sections."

### **DeepSeek-chat Output:**
```python
from typing import Dict, List, Optional
import re

def parse_fleet_context(markdown_text: str) -> Dict[str, List[str]]:
    """
    Parse a fleet context markdown string to extract structured information.
    
    Extracts:
    - 'agents': from "## Who We Are" section (bullet points after description)
    - 'technologies': from "## Key Technologies" section (bullet points)
    - 'rules': from "## Rules" section (bullet points)
    
    Args:
        markdown_text (str): The full markdown content.
    
    Returns:
        Dict[str, List[str]]: Dictionary with keys 'agents', 'technologies', 'rules'.
        Missing sections result in empty lists.
    """
    result = {
        "agents": [],
        "technologies": [],
        "rules": []
    }
    
    # Split by ## headers to find sections
    sections = re.split(r'\n## ', markdown_text)
    
    for section in sections:
        if not section.strip():
            continue
            
        # Get section title (first line)
        lines = section.strip().split('\n')
        title = lines[0].strip().lower()
        content = '\n'.join(lines[1:])
        
        # Determine which list to populate
        target_key = None
        if "who we are" in title:
            target_key = "agents"
        elif "key technologies" in title:
            target_key = "technologies"
        elif "rules" in title:
            target_key = "rules"
        
        if target_key:
            # Extract bullet points (lines starting with '- ' or '* ')
            bullets = re.findall(r'^[-\*] (.+)$', content, re.MULTILINE)
            # Also check for numbered items in "Who We Are" (e.g., "1. Do useful work")
            if target_key == "agents" and not bullets:
                bullets = re.findall(r'^\d+\.\s+(.+)$', content, re.MULTILINE)
            result[target_key] = [b.strip() for b in bullets if b.strip()]
    
    return result
