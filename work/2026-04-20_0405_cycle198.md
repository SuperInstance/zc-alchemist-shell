def deadband_filter(signal: float, bounds: tuple[float, float], p0_threshold: float) -> tuple[str, float]:
    """
    Implements the Deadband Protocol (P0/P1/P2) for a scalar signal.

    P0 (Negative Space): If |signal| >= p0_threshold, return ('P0', 0.0).
    P1 (Safe Channel): If signal is within `bounds`, return ('P1', signal).
    P2 (Optimize): If signal is outside bounds but |signal| < p0_threshold,
                   clamp it to the nearest bound and return ('P2', clamped_value).

    Args:
        signal: Input scalar value.
        bounds: (lower, upper) acceptable range for P1.
        p0_threshold: Absolute threshold for catastrophic P0 state.

    Returns:
        Tuple of (protocol_phase, output_signal).
    """
    lower, upper = bounds

    # P0: Negative Space (catastrophic avoidance)
    if abs(signal) >= p0_threshold:
        return ('P0', 0.0)

    # P1: Safe Channel (within acceptable bounds)
    if lower <= signal <= upper:
        return ('P1', signal)

    # P2: Optimize (clamp to nearest bound)
    clamped_value = min(max(signal, lower), upper)
    return ('P2', clamped_value)
```

    Implements the Deadband Protocol (P0/P1/P2) for a scalar signal.

    P0 (Negative Space): If |signal| >= p0_threshold, return ('P0', 0.0).
    P1 (Safe Channel): If signal is within `bounds`, return ('P1', signal).
    P2 (Optimize): If signal is outside bounds but |signal| < p0_threshold,
                   clamp it to the nearest bound and return ('P2', clamped_value).

    Args:
        signal: Input scalar value.
        bounds: (lower, upper) acceptable range for P1.
        p0_threshold: Absolute threshold for catastrophic P0 state.

    Returns:
        Tuple of (protocol_phase, output_signal).
    """
    low, high = bounds

    # Check P0 condition first (highest priority)
    if abs(signal) >= p0_threshold:
        return ("P0", 0.0)

    # Check P1 condition
    if low <= signal <= high:
        return ("P1", signal)

    # Otherwise, we are in P2: clamp to nearest bound
    # Determine which bound is closer
    dist_to_low = abs(signal - low)
    dist_to_high = abs(signal - high)

    if dist_to_low < dist_to_high:
        clamped = low
    else:
        clamped = high

    return ("P2", clamped)
```

### **Comparison for Function 1:**
*   **Correctness:** Both are logically correct and respect the P0 > P1 > P2 priority.
*   **Clarity:** DeepSeek uses `min(max(...))` for clamping, which is a standard, concise pattern. Llama calculates explicit distances, which is more verbose but perhaps more readable for the specific "nearest bound" logic.
*   **Fleet Context:** Both correctly implement the protocol as described in `FLEET-CONTEXT.md`.
*   **Edge Cases:** Both handle the P0 threshold correctly. Llama's distance calculation might have a subtlety if `signal` is exactly midway between bounds; it would clamp to `high` due to the `<=` comparison. DeepSeek's `min(max(...))` would clamp to `upper` in that same scenario (since `max(signal, lower)` returns `signal`, then `min(signal, upper)` returns `upper`). The spec does not define tie-breaking, so both are acceptable.
*   **Verdict:** **DeepSeek** is slightly more idiomatic and concise. **Llama** is more explicit in its "nearest bound" logic.
