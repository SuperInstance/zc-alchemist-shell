## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

The function should take:
- `values`: list of floats (input signal)
- `deadband_min`: float (lower bound of dead zone, default 0.1)
- `deadband_max`: float (upper bound of dead zone, default 0.9)

Protocol logic:
- P0 (negative space): Any value <= deadband_min or >= deadband_max is set to None (unsafe).
- P1 (safe channels): Values > deadband_min and < deadband_max are kept as-is (safe zone).
- P2 (optimize): Not applied here (that's downstream).

Return a list where unsafe values are replaced with None, safe values unchanged.

Include a docstring, type hints, and a simple example.
```

### **DeepSeek-chat output:**
```python
from typing import List, Optional

def deadband_filter(
    values: List[float],
    deadband_min: float = 0.1,
    deadband_max: float = 0.9
) -> List[Optional[float]]:
    """
    Apply Deadband Protocol to a list of values.

    P0: Map negative space – values outside (deadband_min, deadband_max) are unsafe -> None.
    P1: Safe channels – values inside the deadband are kept.

    Parameters:
        values: Input list of float values.
        deadband_min: Lower deadband threshold (default 0.1).
        deadband_max: Upper deadband threshold (default 0.9).

    Returns:
        List where unsafe values are replaced with None, safe values unchanged.

    Example:
        >>> deadband_filter([0.0, 0.2, 0.5, 0.95, 1.0])
        [None, 0.2, 0.5, None, None]
    """
    result = []
    for v in values:
        if v <= deadband_min or v >= deadband_max:
            result.append(None)
        else:
            result.append(v)
    return result
```

**Prompt:**
```

The function should extract:

- level (string)
- agent name (string)

Handle missing key-value pairs gracefully. Use regex if needed.

```

### **DeepSeek-chat output:**
```python
import re
from datetime import datetime
from typing import Dict, Any

def parse_fleet_log(log_line: str) -> Dict[str, Any]:
    """
    Parse a fleet log line into structured components.

    Format: "YYYY-MM-DD HH:MM:SS [LEVEL] AGENT@CYCLE - MESSAGE (KEY=VALUE, ...)"

    Parameters:
        log_line: A single log line string.

    Returns:
        Dictionary with keys: timestamp, level, agent, cycle, message, params.

    Example:
        >>> parse_fleet_log("2026-04-20 14:30:00 [INFO] Alchemist@341 - Task completed (tiles=3, errors=0)")
        {
            'timestamp': datetime.datetime(2026, 4, 20, 14, 30, 0),
            'level': 'INFO',
            'agent': 'Alchemist',
            'cycle': 341,
            'message': 'Task completed',
            'params': {'tiles': 3, 'errors': 0}
        }
    """
    pattern = r'(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) \[(\w+)\] (\w+)@(\d+) - (.+?)(?: \((.+)\))?$'
    match = re.match(pattern, log_line)
    if not match:
        raise ValueError(f"Invalid log format: {log_line}")

    timestamp_str, level, agent, cycle_str, message, params_str = match.groups()
    timestamp = datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
    cycle = int(cycle_str)

    params = {}
    if params_str:
        for pair in params_str.split(', '):
            if '=' in pair:
                key, val = pair.split('=')
                params[key.strip()] = int(val) if val.isdigit() else val

    return {
        'timestamp': timestamp,
        'level': level,
        'agent': agent,
        'cycle': cycle,
        'message': message.strip(),
        'params': params
    }
```
