### **Function 1: deadband_filter()**

**Spec:**  
Input: `values: List[float]`, `deadband: float = 0.1`  
Output: `List[float]`  
Behavior: Implements the Deadband Protocol:  
- P0: Remove any value where `abs(value) < deadband` (negative space).  
- P1: Keep values where `deadband <= abs(value) <= 1.0` (safe channel).  
- P2: For kept values, apply optimization: if value > 0, square it; if value < 0, take absolute value then square root.  
Return filtered/optimized list in original order.

---

**DeepSeek‑chat output:**
```python
from typing import List

def deadband_filter(values: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply the Deadband Protocol to a list of float values.

    Steps:
    1. P0 (negative space): Remove values where absolute value < deadband.
    2. P1 (safe channel): Keep values where deadband <= |value| <= 1.0.
    3. P2 (optimization): For kept values, if positive -> square, if negative -> sqrt(|value|).

    Example:
    >>> deadband_filter([-0.05, 0.2, -0.8, 1.5, 0.05])
    [0.04, 0.8944271909999159]
    """
    if not isinstance(values, list):
        raise TypeError("values must be a list")
    if not isinstance(deadband, (int, float)):
        raise TypeError("deadband must be a numeric value")
    if deadband < 0:
        raise ValueError("deadband must be non‑negative")

    result = []
    for v in values:
        if not isinstance(v, (int, float)):
            continue  # skip non‑numeric entries
        abs_v = abs(v)
        # P0: skip negative space
        if abs_v < deadband:
            continue
        # P1: only keep safe channel
        if abs_v <= 1.0:
            # P2: optimize
            if v > 0:
                result.append(v ** 2)
            else:
                result.append(abs_v ** 0.5)
    return result
```

```python
def deadband_filter(values: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply the Deadband Protocol (P0, P1, P2) to filter and optimize a list of floats.

    Example:
    >>> deadband_filter([0.05, -0.2, 0.8, -1.2, 0.3])
    [0.04, 0.8944271909999159, 0.09]
    """
    if not values:
        return []
    if deadband < 0:
        raise ValueError("deadband cannot be negative")

    filtered = []
    for val in values:
        try:
            val_float = float(val)
        except (TypeError, ValueError):
            continue  # ignore non‑convertible items

        abs_val = abs(val_float)
        # P0: negative space
        if abs_val < deadband:
            continue
        # P1: safe channel
        if abs_val > 1.0:
            continue

        # P2: optimization
        if val_float > 0:
            filtered.append(val_float ** 2)
        else:
            filtered.append(abs_val ** 0.5)

    return filtered
```

- **Clarity:** DeepSeek’s docstring is more detailed about P0/P1/P2. Groq’s example is slightly misleading (includes -1.2 which is outside safe channel).
- **Robustness:** DeepSeek raises errors for invalid `deadband` or non‑list input; Groq silently returns `[]` for empty input and skips non‑convertible items.
- **Fleet relevance:** DeepSeek explicitly names the steps P0/P1/P2 in comments, aligning with fleet doctrine.

### **Function 2: parse_fleet_log()**

**Spec:**  
Input: `log: str`  
Output: `Dict[str, Any]` with keys: `timestamp`, `agent`, `cycle`, `action`, `status`  
Behavior: Parse a log line like `"2026‑04‑20 18:02 UTC — Zeroclaw — Cycle 367 — ACTIVE — Submitted tile"`. Extract components. If a part is missing, set to `None`. Be tolerant of extra spaces/dashes.

---

**DeepSeek‑chat output:**
```python
from typing import Dict, Any
import re

def parse_fleet_log(log: str) -> Dict[str, Any]:
    """
    Parse a semi‑structured fleet log line into structured fields.

    Expected format (variations allowed):
    "YYYY‑MM‑DD HH:MM UTC — Agent — Cycle N — STATUS — Description"

    Example:
    >>> parse_fleet_log("2026‑04‑20 18:02 UTC — Zeroclaw — Cycle 367 — ACTIVE — Submitted tile")
    {'timestamp': '2026‑04‑20 18:02 UTC', 'agent': 'Zeroclaw', 'cycle': 367, 'status': 'ACTIVE', 'action': 'Submitted tile'}
    """
    if not isinstance(log, str):
        raise TypeError("log must be a string")
    log = log.strip()
    if not log:
        return {
            "timestamp": None,
            "agent": None,
            "cycle": None,
            "status": None,
            "action": None
        }

    # Split by '—' (em dash) or '--' (fallback)
    parts = [p.strip() for p in re.split(r'—|--', log)]
    # Ensure at least 5 placeholders
    while len(parts) < 5:
        parts.append(None)

    timestamp = parts[0] if parts[0] else None
    agent = parts[1] if parts[1] else None
    cycle = None
    status = None
    action = parts[4] if parts[4] else None

    # Extract cycle number from parts[2]
    if parts[2]:
        match = re.search(r'Cycle\s*(\d+)', parts[2], re.IGNORECASE)
        if match:
            cycle = int(match.group(1))
    # Extract status from parts[3]
    if parts[3]:
        # Common status tokens
        if any(s in parts[3].upper() for s in ['ACTIVE', 'IDLE', 'ERROR', 'PAUSED']):
            status = parts[3].strip()

    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "action": action
    }
```
