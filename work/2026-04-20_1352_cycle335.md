### **Function 1: `validate_deadband_protocol`**

**Prompt:**
```
Write a Python function `validate_deadband_protocol` that takes a list of strings `actions`. Each string is a simple action like "avoid rock", "find channel", "optimize path". The function must enforce the Deadband Protocol: P0 (avoid negative) must come before any P1 (find safe), and P1 must come before any P2 (optimize). Return `True` if the sequence is valid, `False` if a priority is violated. Assume actions are pre-classified; you can check if a string contains keywords: P0 actions have "avoid" or "map negative", P1 have "find" or "channel", P2 have "optimize" or "best". Be strict about order.
```

**DeepSeek-chat Output:**
```python
def validate_deadband_protocol(actions):
    """
    Validate that actions follow Deadband Protocol priority order.
    P0 (avoid/map negative) -> P1 (find/channel) -> P2 (optimize/best).
    """
    current_phase = 0  # 0: P0, 1: P1, 2: P2
    for action in actions:
        action_lower = action.lower()
        # Classify phase of this action
        if "avoid" in action_lower or "map negative" in action_lower:
            action_phase = 0
        elif "find" in action_lower or "channel" in action_lower:
            action_phase = 1
        elif "optimize" in action_lower or "best" in action_lower:
            action_phase = 2
        else:
            # Unrecognized action, treat as neutral? For strictness, assume it breaks order.
            # Could also ignore, but spec says pre-classified. We'll treat as invalid.
            return False
        
        # Check phase order
        if action_phase < current_phase:
            # Trying to go back to earlier phase
            return False
        # Move forward in phases if this action is ahead
        if action_phase > current_phase:
            current_phase = action_phase
    return True
```

**Groq Llama-70b Output:**
```python
def validate_deadband_protocol(actions):
    """
    Validate Deadband Protocol sequence.
    P0: actions containing "avoid" or "map negative"
    P1: actions containing "find" or "channel"
    P2: actions containing "optimize" or "best"
    Sequence must be P0 then P1 then P2, not necessarily all present.
    """
    phase = 0
    for action in actions:
        action_lower = action.lower()
        if "avoid" in action_lower or "map negative" in action_lower:
            if phase > 0:
                return False
            phase = max(phase, 0)
        elif "find" in action_lower or "channel" in action_lower:
            if phase > 1:
                return False
            phase = max(phase, 1)
        elif "optimize" in action_lower or "best" in action_lower:
            if phase > 2:
                return False
            phase = max(phase, 2)
        else:
            # Unknown action type - assume it's neutral and doesn't affect phase
            continue
    return True
```

**Analysis:**
*   **Correctness:** Both correctly enforce P0→P1→P2. DeepSeek's version forbids going back (`action_phase < current_phase`). Groq's version uses `if phase > X:` to block late-phase actions from appearing early. Both are logically sound.
*   **Style:** DeepSeek's code is slightly more compact. Groq's uses `max(phase, X)` which is clever but less direct than DeepSeek's `if action_phase > current_phase: current_phase = action_phase`.
*   **Edge Cases:** DeepSeek returns `False` on unrecognized actions (strict). Groq continues (lenient). The prompt said "assume actions are pre-classified," so DeepSeek's strict interpretation is more aligned.
*   **Winner:** **DeepSeek-chat** for stricter adherence to the "pre-classified" assumption and cleaner state update logic.

### **Function 2: `parse_fleet_tile`**

**Prompt:**
```
Write a Python function `parse_fleet_tile` that takes a string `tile_text` (a raw training tile). The tile format is loose but usually contains lines like "Agent: Alchemist", "Cycle: 335", "Phase: 4", "Task: compare models", "Content: ...". Extract these fields into a dict with keys: `agent_type`, `cycle`, `phase`, `task`, `content_hash`. Compute `content_hash` as a SHA256 hex digest of the entire `tile_text`. If a field is missing, set its value to `None`. Be robust to extra whitespace, colons, and varying line order.
```

**DeepSeek-chat Output:**
```python
import hashlib

def parse_fleet_tile(tile_text):
    """
    Parse a fleet training tile string into structured dict.
    """
    result = {
        "agent_type": None,
        "cycle": None,
        "phase": None,
        "task": None,
        "content_hash": None
    }
    
    # Compute hash
    result["content_hash"] = hashlib.sha256(tile_text.encode()).hexdigest()
    
    # Parse lines
    for line in tile_text.strip().split('\n'):
        line = line.strip()
        if not line:
            continue
        # Split on first colon
        if ':' not in line:
            continue
        key, value = line.split(':', 1)
        key = key.strip().lower()
        value = value.strip()
        
        if key == "agent":
            result["agent_type"] = value
        elif key == "cycle":
            try:
                result["cycle"] = int(value)
            except ValueError:
                result["cycle"] = None
        elif key == "phase":
            try:
                result["phase"] = int(value)
            except ValueError:
                result["phase"] = None
        elif key == "task":
            result["task"] = value
    
    return result
```

**Groq Llama-70b Output:**

```python
import hashlib

def parse_fleet_tile(tile_text):
    """
    Parse a fleet training tile.
    Expected fields: Agent, Cycle, Phase, Task.
    Returns dict with agent_type, cycle, phase, task, content_hash.
    """
    fields = {
        "agent": None,
        "cycle": None,
        "phase": None,
        "task": None
    }
    
    for line in tile_text.splitlines():
        line = line.strip()
        if ':' in line:
            key, val = line.split(':', 1)
            key = key.strip().lower()
            val = val.strip()
            if key in fields:
                if key == "cycle" or key == "phase":
                    try:
                        fields[key] = int(val)
                    except ValueError:
                        fields[key] = None
                else:
                    fields[key] = val
    
    # Build result with renamed agent -> agent_type
    result = {
        "agent_type": fields["agent"],
        "cycle": fields["cycle"],
        "phase": fields["phase"],
        "task": fields["task"],
        "content_hash": hashlib.sha256(tile_text.encode()).hexdigest()
    }
    return result
```
