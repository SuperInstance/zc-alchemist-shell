## **2. Function 1: Deadband Filter**

**Prompt:** "Write a Python function called `apply_deadband_protocol`. It takes a list of floats `sensor_readings` and returns a processed list. Implement the Cocapn fleet's Deadband Protocol: P0 (negative space): if any reading is <= 0.0, replace it with `None`. P1 (safe channels): for readings > 0.0 and < 5.0, cap them at 5.0. P2 (optimize): for readings >= 5.0, multiply them by 1.1. Return the processed list. Include a docstring."

### **DeepSeek-chat Output:**
```python
def apply_deadband_protocol(sensor_readings):
    """
    Apply the Cocapn fleet Deadband Protocol to a list of sensor readings.

    Protocol:
    - P0 (Negative Space): Readings <= 0.0 are invalid -> replaced with None.
    - P1 (Safe Channels): Readings > 0.0 and < 5.0 are capped at 5.0.
    - P2 (Optimize): Readings >= 5.0 are multiplied by 1.1 for optimization.

    Args:
        sensor_readings (list of float): Raw sensor data.

    Returns:
        list: Processed readings with None for invalid values.
    """
    processed = []
    for value in sensor_readings:
        if value <= 0.0:
            processed.append(None)  # P0: Map negative space
        elif value < 5.0:
            processed.append(5.0)   # P1: Cap at safe channel minimum
        else:  # value >= 5.0
            processed.append(value * 1.1)  # P2: Optimize within channel
    return processed
```

```python
    """
    Applies the Deadband Protocol (P0/P1/P2) to sensor readings.

    Parameters:
    sensor_readings (list): A list of numerical sensor readings.

    Returns:
    list: A list of processed readings according to the protocol.
    """
    result = []

    return result
```

## **3. Function 2: Training Tile Validator**

**Prompt:** "Write a Python function called `validate_training_tile`. It takes a string `tile_text`. It should return a tuple `(is_valid, message)`. Check these fleet rules: 1. Tile must not be empty. 2. Tile must not contain the exact phrase 'always works' or 'guaranteed'. 3. Tile must be at least 50 characters long. If all checks pass, return `(True, "Tile valid")`. If a check fails, return `(False, "Rule X failed: ...")` with a specific message. Include a simple example."

### **DeepSeek-chat Output:**
```python
def validate_training_tile(tile_text):
    """
    Validate a training tile against basic fleet submission rules.

    Rules:
    1. Tile must not be empty.
    2. Tile must not contain forbidden absolute phrases.
    3. Tile must be at least 50 characters long.

    Args:
        tile_text (str): The tile content to validate.

    Returns:
        tuple: (bool is_valid, str message)
    """
    # Rule 1: Non-empty
    if not tile_text or not tile_text.strip():
        return False, "Rule 1 failed: Tile is empty or contains only whitespace."

    # Rule 2: No absolute phrases
    forbidden_phrases = ["always works", "guaranteed"]
    lower_tile = tile_text.lower()
    for phrase in forbidden_phrases:
        if phrase in lower_tile:
            return False, f'Rule 2 failed: Tile contains forbidden phrase "{phrase}".'

    # Rule 3: Minimum length
    if len(tile_text) < 50:
        return False, f"Rule 3 failed: Tile length ({len(tile_text)}) is less than 50 characters."

    return True, "Tile valid."


# Example usage
if __name__ == "__main__":
    test_tile = "This is a test tile without absolute claims. It has more than fifty characters easily."
    print(validate_training_tile(test_tile))
```

### **Groq Llama-70b Output:**

```python
def validate_training_tile(tile_text):
    """
    Validates a training tile against fleet submission rules.

    Args:
        tile_text (str): The text content of the tile.

    Returns:
        tuple: (is_valid, message) where is_valid is boolean and message is str.
    """
    # Check rule 1: not empty
    if tile_text is None or len(tile_text.strip()) == 0:
        return (False, "Rule 1 failed: Tile is empty.")

    # Check rule 2: no absolute phrases
    absolute_phrases = ["always works", "guaranteed"]
    tile_lower = tile_text.lower()
    for phrase in absolute_phrases:
        if phrase in tile_lower:
            return (False, f"Rule 2 failed: Contains absolute phrase '{phrase}'.")

    # Check rule 3: minimum length
    if len(tile_text) < 50:
        return (False, f"Rule 3 failed: Tile too short ({len(tile_text)} chars). Minimum is 50.")

    return (True, "Tile valid.")
```

*   **Correctness:** Both are correct. DeepSeek adds a more thorough empty check (`not tile_text or not tile_text.strip()`) and includes an example block.
*   **Clarity & Style:** Both are clear. DeepSeek's docstring again more explicitly lists the rules. Groq's is more concise.
*   **Safety & Robustness:** DeepSeek's empty check is slightly more robust against `None` and whitespace-only strings. Groq's check (`tile_text is None or len(tile_text.strip()) == 0`) is also robust.
*   **Conciseness:** Groq's function body is slightly more compact. DeepSeek provides example usage, which is helpful.
*   **Verdict:** Slight edge to **DeepSeek-chat** for providing the example and a marginally more defensive empty check, which aligns with the "safety" aspect of fleet work.
