## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function `deadband_filter(readings, deadband_min, deadband_max)` that applies the Deadband Protocol to a list of numeric sensor readings.

The Deadband Protocol has three priority levels:
P0: Remove readings that are "rocks" — values outside the absolute safe range [deadband_min, deadband_max].
P1: Find "safe channels" — consecutive readings within the safe range.
P2: Optimize within each safe channel — replace each channel with its average.

Return a list of filtered values (averages per safe channel) in original order.

Include docstring and example.
```

### **DeepSeek-chat output:**
```python
def deadband_filter(readings, deadband_min, deadband_max):
    """
    Apply Deadband Protocol filtering to sensor readings.
    
    P0: Remove values outside [deadband_min, deadband_max] (rocks).
    P1: Group consecutive safe values into channels.
    P2: Replace each channel with its average.
    
    Args:
        readings: List of numeric values.
        deadband_min: Minimum safe value.
        deadband_max: Maximum safe value.
    
    Returns:
        List of averaged channel values in original order.
    
    Example:
        >>> deadband_filter([1, 2, 10, 11, 12, 30, 31], 5, 25)
        [10.5, 30.5]  # Channels: [10,11,12] avg=11, [30,31] avg=30.5
    """
    if not readings:
        return []
    
    # P0: Identify safe readings (within deadband)
    safe_mask = [deadband_min <= v <= deadband_max for v in readings]
    
    # P1: Find consecutive safe channels
    channels = []
    current_channel = []
    for i, (value, is_safe) in enumerate(zip(readings, safe_mask)):
        if is_safe:
            current_channel.append(value)
        elif current_channel:
            channels.append(current_channel)
            current_channel = []
    if current_channel:
        channels.append(current_channel)
    
    # P2: Optimize each channel by averaging
    result = [sum(channel) / len(channel) for channel in channels]
    
    return result
```

```python
def deadband_filter(readings, deadband_min, deadband_max):
    """
    Implements the Deadband Protocol for sensor data filtering.
    
    Steps:
    1. Remove readings outside the deadband range (P0).
    2. Group remaining readings into consecutive "safe channels" (P1).
    3. Compute the average of each safe channel (P2).
    
    Parameters:
        readings (list): Input sensor readings.
        deadband_min (float): Lower bound of safe range.
        deadband_max (float): Upper bound of safe range.
    
    Returns:
        list: A list of averaged values for each safe channel.
    
    Example:
        >>> readings = [1, 2, 10, 11, 12, 30, 31]
        >>> deadband_filter(readings, 5, 25)
        [11.0, 30.5]
    """
    # P0: Filter out readings outside deadband
    safe_readings = [r for r in readings if deadband_min <= r <= deadband_max]
    
    if not safe_readings:
        return []
    
    # P1: Identify safe channels (consecutive indices in original list)
    channels = []
    current_channel = []
    prev_index = -2
    
    for i, val in enumerate(readings):
        if deadband_min <= val <= deadband_max:
            if i == prev_index + 1:
                current_channel.append(val)
            else:
                if current_channel:
                    channels.append(current_channel)
                current_channel = [val]
            prev_index = i
        else:
            if current_channel:
                channels.append(current_channel)
                current_channel = []
    
    if current_channel:
        channels.append(current_channel)
    
    # P2: Compute average for each channel
    averaged_channels = [sum(ch) / len(ch) for ch in channels if ch]
    
    return averaged_channels
```

from datetime import datetime

def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line into a dictionary.
    
    Format: "YYYY-MM-DD HH:MM:SS | AGENT | CYCLE | STATUS | MESSAGE"
    
    Args:
        log_line (str): A log line.
    
    Returns:
        dict: Parsed fields or None if malformed.
    
    Example:
        >>> parse_fleet_log("2026-04-20 19:23:00 | Alchemist | 379 | ACTIVE | Testing")
        {'timestamp': datetime.datetime(2026, 4, 20, 19, 23, 0),
         'agent': 'Alchemist',
         'cycle': 379,
         'status': 'ACTIVE',
         'message': 'Testing'}
    """
    if not log_line or '|' not in log_line:
        return None
    
    parts = [part.strip() for part in log_line.split('|')]
    if len(parts) != 5:
        return None
    
    date_str, agent, cycle_str, status, message = parts
    
    try:
        timestamp = datetime.strptime(date_str, '%Y-%m-%d %H:%M:%S')
        cycle = int(cycle_str)
    except ValueError:
        return None
    
    return {
        'timestamp': timestamp,
        'agent': agent,
        'cycle': cycle,
        'status': status,
        'message': message
    }
