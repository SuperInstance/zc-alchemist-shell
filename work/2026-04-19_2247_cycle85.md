## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.
Input: A list of floating-point numbers `signal` and a float `deadband_threshold`.
Doctrine:
P0 (Negative Space): If any value in `signal` is <= 0, return an empty list [] and print "P0 Violation: Non-positive signal".
P1 (Safe Channel): If the signal's range (max-min) is > `deadband_threshold`, return the original signal. Print "P1: Wide channel, signal passed".
P2 (Optimize): If the range is <= `deadband_threshold`, apply a smoothing filter (a simple moving average with window size 3). Return the smoothed signal. Print "P2: Narrow channel, optimizing".
Handle edge cases like short lists. Include type hints and a brief docstring.
```

### **DeepSeek-chat Output:**
```python
from typing import List

def deadband_filter(signal: List[float], deadband_threshold: float) -> List[float]:
    """
    Apply the Deadband Protocol (P0/P1/P2) to a signal.

    P0: Reject signals with non-positive values.
    P1: Pass signals with variability exceeding the threshold.
    P2: Smooth signals with low variability.

    Args:
        signal: Input signal as list of floats.
        deadband_threshold: Threshold for signal range decision.

    Returns:
        Filtered (or empty) signal list.
    """
    # P0: Map negative space
    if any(x <= 0 for x in signal):
        print("P0 Violation: Non-positive signal")
        return []

    if not signal:
        print("P0 Violation: Empty signal")
        return []

    # Calculate signal range
    sig_range = max(signal) - min(signal)

    # P1: Find safe channels
    if sig_range > deadband_threshold:
        print("P1: Wide channel, signal passed")
        return signal.copy()

    # P2: Optimize within channels
    print("P2: Narrow channel, optimizing")
    # Simple moving average, window size 3
    smoothed = []
    for i in range(len(signal)):
        window = signal[max(0, i-1): i+2]  # center-aligned window
        avg = sum(window) / len(window)
        smoothed.append(avg)
    return smoothed
```

    Parameters:
    signal: List[float] - input signal values
    deadband_threshold: float - threshold for determining signal variability

    Returns:
    List[float] - processed signal according to P0/P1/P2 rules
    """
    # P0: Check for non-positive values
    for value in signal:
        if value <= 0:
            print("P0 Violation: Non-positive signal detected")
            return []

    # Handle empty signal
    if len(signal) == 0:
        print("P0 Violation: Empty signal")
        return []

    # Calculate range
    min_val = min(signal)
    max_val = max(signal)
    signal_range = max_val - min_val

    # P1: Check if range exceeds threshold
    if signal_range > deadband_threshold:
        print("P1: Wide channel - signal passed through")
        return signal[:]  # Return a copy

    # P2: Apply smoothing optimization
    print("P2: Narrow channel - applying optimization")
    window_size = 3
    smoothed_signal = []

    for i in range(len(signal)):
        start_idx = max(0, i - window_size // 2)
        end_idx = min(len(signal), i + window_size // 2 + 1)
        window = signal[start_idx:end_idx]
        window_avg = sum(window) / len(window)
        smoothed_signal.append(window_avg)

    return smoothed_signal
