### **Function 1: `deadband_filter`**

**Prompt to both models:**
"Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol. The function should take a list of floats `values` and three threshold floats `p0_thresh`, `p1_thresh`, `p2_thresh`. It should return a list of strings representing the protocol phase for each value: 'P0' if value <= p0_thresh (unsafe, negative space), 'P1' if p0_thresh < value <= p1_thresh (safe channel), 'P2' if p1_thresh < value <= p2_thresh (optimization zone), and 'OUT_OF_BOUNDS' if value > p2_thresh. Include a docstring and a simple example."

**DeepSeek-chat Output:**
```python
def deadband_filter(values, p0_thresh, p1_thresh, p2_thresh):
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of values.

    Parameters:
    values (list of float): Input values to classify.
    p0_thresh (float): Threshold for P0 (negative space). Values <= this are P0.
    p1_thresh (float): Threshold for P1 (safe channel). Values > p0_thresh and <= p1_thresh are P1.
    p2_thresh (float): Threshold for P2 (optimization). Values > p1_thresh and <= p2_thresh are P2.

    Returns:
    list of str: Protocol phase for each value ('P0', 'P1', 'P2', or 'OUT_OF_BOUNDS').
    """
    result = []
    for v in values:
        if v <= p0_thresh:
            result.append('P0')
        elif p0_thresh < v <= p1_thresh:
            result.append('P1')
        elif p1_thresh < v <= p2_thresh:
            result.append('P2')
        else:
            result.append('OUT_OF_BOUNDS')
    return result
```

```python
        elif v <= p1_thresh:
            phases.append('P1')
```

### **Function 2: `parse_fleet_log`**

**Prompt to both models:**
"Write a Python function called `parse_fleet_log`. It should take a string `log_line` with the format '[TIMESTAMP] AGENT@CYCLE: MESSAGE | key1=value1 key2=value2'. Example: '[2026-04-20T12:21Z] Alchemist@316: Submitted tile | tiles=3 accepted=2'. The function should return a dictionary with keys: 'timestamp', 'agent', 'cycle', 'message', and a 'metadata' dict containing the key-value pairs. If the format is invalid, return None. Use regex for parsing. Include a docstring."

**DeepSeek-chat Output:**
```python
import re

def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line into its components.

    Expected format: [TIMESTAMP] AGENT@CYCLE: MESSAGE | key1=value1 key2=value2

    Parameters:
    log_line (str): A single log line.

    Returns:
    dict or None: Dictionary with keys 'timestamp', 'agent', 'cycle', 'message', 'metadata'.
                  Returns None if the line doesn't match the expected pattern.
    """
    # Regex pattern to capture the main parts and metadata
    pattern = r'^\[([^\]]+)\] ([^@]+)@(\d+): ([^|]+) \| (.+)$'
    match = re.match(pattern, log_line.strip())
    if not match:
        return None

    timestamp, agent, cycle_str, message, meta_str = match.groups()

    # Parse metadata key-value pairs (space-separated)
    metadata = {}
    for pair in meta_str.split():
        if '=' in pair:
            key, value = pair.split('=', 1)
            metadata[key] = value

    return {
        'timestamp': timestamp,
        'agent': agent,
        'cycle': int(cycle_str),
        'message': message.strip(),
        'metadata': metadata
    }

# Example
if __name__ == "__main__":
    line = "[2026-04-20T12:21Z] Alchemist@316: Submitted tile | tiles=3 accepted=2"
    print(parse_fleet_log(line))
```
