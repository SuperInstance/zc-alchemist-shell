"Write a Python function called `validate_deadband_protocol`. It takes one argument, `actions`, which is a list of strings. Each string is either 'P0', 'P1', or 'P2'. **DeepSeek-chat Output:**

```python
def validate_deadband_protocol(actions):
    """
    Validate that a list of action strings follows Deadband Protocol priority.
    
    Args:
        actions (list of str): List containing 'P0', 'P1', 'P2' strings.
    
    Returns:
        bool: True if order is strictly P0 -> P1 -> P2, False otherwise.
    """
    # Define allowed phases and their order
    phase_order = {'P0': 0, 'P1': 1, 'P2': 2}
    
    # Track the highest phase we've seen
    max_seen = -1
    
    for action in actions:
        if action not in phase_order:
            # Invalid action string
            return False
        
        current_phase = phase_order[action]
        
        # Check if we're going backwards or staying in same phase is fine?
        # Strict order means we cannot go to a lower phase index after seeing a higher one.
        # Actually, we must be non-decreasing and cannot skip.
        # Simpler: Once we move to a new phase, we cannot see an older phase.
        if current_phase < max_seen:
            return False
        
        # Update max_seen if we've advanced to a new phase
        if current_phase > max_seen:
            # Check if we skipped a phase (e.g., P0 -> P2)
            if current_phase - max_seen > 1:
                return False
            max_seen = current_phase
    
    return True
```

```python
def validate_deadband_protocol(actions):
    """
    Validates if the given list of actions follows the Deadband Protocol priority order.
    The protocol requires strict ordering: all P0 actions must come before any P1 actions,
    and all P1 actions must come before any P2 actions.
    
    Args:
        actions (list of str): List of action strings ('P0', 'P1', or 'P2')
    
    Returns:
        bool: True if the order is valid, False otherwise.
    """
    # Define the expected order
    expected_order = ['P0', 'P1', 'P2']
    
    # Find the index of each action in the expected order
    # Filter out invalid actions early
    try:
        indices = [expected_order.index(action) for action in actions]
    except ValueError:
        # Invalid action found
        return False
    
    # Check if indices are non-decreasing and don't skip phases
    for i in range(1, len(indices)):
        if indices[i] < indices[i-1]:
            return False  # Went backwards
        if indices[i] > indices[i-1] + 1:
            return False  # Skipped a phase
    
    return True
```

### **Function 2: `parse_fleet_tile`**

**Prompt:**
"Write a Python function called `parse_fleet_tile`. It takes one argument, `tile_text`, a string containing a simple markdown-like format for a fleet training tile. Example input: '## Tile\\n**Model:** deepseek\\n**Score:** 85\\n**Notes:** good reasoning'. The function should extract the title (text after '## ' on the first line), model, score (as int), and notes. Return a dict with keys 'title', 'model', 'score', 'notes'. If a field is missing, use None for its value. Assume each field is on its own line. Be robust to extra whitespace. Include a docstring."

**DeepSeek-chat Output:**
```python
def parse_fleet_tile(tile_text):
    """
    Parse a fleet training tile string into a structured dictionary.
    
    Expected format (markdown-like):
    ## Title
    **Model:** model_name
    **Score:** 85
    **Notes:** some notes
    
    Args:
        tile_text (str): Raw tile text.
    
    Returns:
        dict: Contains 'title', 'model', 'score', 'notes'.
              Missing fields become None. Score converted to int.
    """
    result = {'title': None, 'model': None, 'score': None, 'notes': None}
    
    lines = tile_text.strip().split('\n')
    
    for line in lines:
        line = line.strip()
        
        # Parse title (## Title)
        if line.startswith('## '):
            result['title'] = line[3:].strip()
        
        # Parse model (**Model:** value)
        elif line.startswith('**Model:**'):
            result['model'] = line.split(':', 1)[1].strip()
        
        # Parse score (**Score:** value)
        elif line.startswith('**Score:**'):
            score_str = line.split(':', 1)[1].strip()
            try:
                result['score'] = int(score_str)
            except ValueError:
                result['score'] = None
        
        # Parse notes (**Notes:** value)
        elif line.startswith('**Notes:**'):
            result['notes'] = line.split(':', 1)[1].strip()
    
    return result
```
