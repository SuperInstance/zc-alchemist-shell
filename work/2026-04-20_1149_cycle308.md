### **Function 1: `deadband_filter`**
**Prompt:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input: A list of floats `values` representing sensor readings or scores, and three threshold floats: `p0_threshold` (danger zone), `p1_threshold` (safe boundary), `p2_threshold` (optimization target). Assume p0_threshold < p1_threshold < p2_threshold.

The protocol has three strict priorities:
P0 (Negative Space): Any value <= p0_threshold is a critical failure. The function must identify these.
P1 (Safe Channels): Values > p0_threshold and <= p1_threshold are in the operational safe zone. The function must identify these.
P2 (Optimization): Values > p1_threshold are candidates for optimization. For values in P2, if they also exceed p2_threshold, they are considered "optimal".

The function must return a dictionary with four keys:
- `p0_failures`: List of indices where value <= p0_threshold.
- `p1_safe`: List of indices where p0_threshold < value <= p1_threshold.
- `p2_optimizable`: List of indices where value > p1_threshold.
- `p2_optimal`: A subset of `p2_optimizable` indices where value > p2_threshold.

Include type hints, a clear docstring, and consider edge cases (empty list, thresholds in wrong order). Do not use external libraries.
```

from typing import List, Dict

def deadband_filter(values: List[float], p0_threshold: float, p1_threshold: float, p2_threshold: float) -> Dict[str, List[int]]:
    """
    Implements the Cocapn fleet Deadband Protocol (P0/P1/P2) on a list of values.

    Args:
        values: List of sensor readings or scores.
        p0_threshold: Values <= this are P0 failures (danger zone).
        p1_threshold: Values > p0 and <= this are P1 safe.
        p2_threshold: Values > p1 and > this are P2 optimal.

    Returns:
        Dictionary with keys:
        - p0_failures: Indices where value <= p0_threshold
        - p1_safe: Indices where p0_threshold < value <= p1_threshold
        - p2_optimizable: Indices where value > p1_threshold
        - p2_optimal: Subset of p2_optimizable where value > p2_threshold

    Raises:
        ValueError: If thresholds are not in ascending order.
    """
    if not (p0_threshold < p1_threshold < p2_threshold):
        raise ValueError("Thresholds must be in order: p0 < p1 < p2")

    p0_failures = []
    p1_safe = []
    p2_optimizable = []
    p2_optimal = []

    for idx, val in enumerate(values):
        if val <= p0_threshold:
            p0_failures.append(idx)
        elif val <= p1_threshold:
            p1_safe.append(idx)
        else:  # val > p1_threshold
            p2_optimizable.append(idx)
            if val > p2_threshold:
                p2_optimal.append(idx)

    return {
        "p0_failures": p0_failures,
        "p1_safe": p1_safe,
        "p2_optimizable": p2_optimizable,
        "p2_optimal": p2_optimal
    }
```

from datetime import datetime
from typing import Dict, Any

def parse_fleet_log(log_line: str) -> Dict[str, Any]:
    """
    Parses a standardized fleet agent log line.

    Expected format: "YYYY-MM-DD HH:MM:SS | AGENT_NAME | STATUS | CYCLE: N | MESSAGE"

    Args:
        log_line: A single log line string.

    Returns:
        Dictionary with parsed components.

    Raises:
        ValueError: If line format is invalid.
    """
    parts = log_line.split(" | ")
    if len(parts) != 5:
        raise ValueError(f"Log line must have 5 pipe-separated parts, got {len(parts)}")

    date_str, agent, status, cycle_part, message = parts

    try:
        timestamp = datetime.strptime(date_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        raise ValueError(f"Invalid timestamp format: {date_str}")

    if not cycle_part.startswith("CYCLE: "):
        raise ValueError(f"Cycle part malformed: {cycle_part}")
    try:
        cycle = int(cycle_part[7:])  # Remove "CYCLE: "
    except ValueError:
        raise ValueError(f"Cycle number invalid: {cycle_part[7:]}")

    return {
        "timestamp": timestamp,
        "agent": agent.strip(),
        "status": status.strip(),
        "cycle": cycle,
        "message": message.strip()
    }
```
