## **2. Function 1: `deadband_filter`**

**Prompt:** "Write a Python function called `deadband_filter` that takes a list of floats `values` and three float parameters: `p0_threshold`, `p1_min`, `p1_max`. Implement the Deadband Protocol: P0: Remove any values <= `p0_threshold` (negative space). P1: Keep only values where `p1_min` <= value <= `p1_max` (safe channel). P2: Return the sorted list (optimization within channel). Include a docstring and a simple example."

### **DeepSeek-chat Output:**
```python
def deadband_filter(values, p0_threshold, p1_min, p1_max):
    """
    Apply the Deadband Protocol (P0-P2) to a list of values.

    P0 (Negative Space): Filter out values <= p0_threshold.
    P1 (Safe Channel): Keep values within [p1_min, p1_max].
    P2 (Optimization): Return sorted result.

    Parameters:
    values (list of float): Input values to filter.
    p0_threshold (float): Values <= this are removed (P0).
    p1_min (float): Minimum bound for safe channel (P1).
    p1_max (float): Maximum bound for safe channel (P1).

    Returns:
    list of float: Filtered and sorted values.
    """
    # P0: Map negative space
    p0_filtered = [v for v in values if v > p0_threshold]

    # P1: Find safe channel
    p1_filtered = [v for v in p0_filtered if p1_min <= v <= p1_max]

    # P2: Optimize within channel
    return sorted(p1_filtered)
```

## **3. Function 2: `parse_fleet_log`**

**Prompt:** "Write a Python function `parse_fleet_log` that takes a string `log_line` with format 'TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE'. Return a dictionary with keys `timestamp`, `agent`, `cycle`, `status`, `message`. Handle malformed lines by returning `None`. Use `str.split()` with maxsplit=4. `cycle` should be an integer. Include error handling."

### **DeepSeek-chat Output:**
```python
def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line into a dictionary.

    Format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"

    Parameters:
    log_line (str): A single log line.

    Returns:
    dict or None: Parsed fields as dictionary, or None if malformed.
    """
    if not isinstance(log_line, str):
        return None

    parts = log_line.split(' | ', maxsplit=4)
    if len(parts) != 5:
        return None

    timestamp, agent, cycle_str, status, message = parts

    try:
        cycle = int(cycle_str)
    except ValueError:
        return None

    return {
        'timestamp': timestamp.strip(),
        'agent': agent.strip(),
        'cycle': cycle,
        'status': status.strip(),
        'message': message.strip()
    }
```
