## **2. Function 1: Deadband Filter**

**Prompt:** "Write a Rust function `apply_deadband_protocol` that takes a vector of strings representing possible actions. It must filter out any actions containing the substring 'error' (P0), then keep only actions containing 'safe' (P1), then from remaining actions select the shortest one (P2). Return a `Result<String, &'static str>` where Ok contains the selected action or an error if no action survives P0/P1."

### **DeepSeek-chat output:**
```rust
pub fn apply_deadband_protocol(actions: Vec<String>) -> Result<String, &'static str> {
    // P0: Map negative space - remove actions containing "error"
    let p0_filtered: Vec<String> = actions
        .into_iter()
        .filter(|action| !action.to_lowercase().contains("error"))
        .collect();
    
    if p0_filtered.is_empty() {
        return Err("No actions survived P0 filtering");
    }
    
    // P1: Find safe channels - keep only actions containing "safe"
    let p1_filtered: Vec<String> = p0_filtered
        .into_iter()
        .filter(|action| action.to_lowercase().contains("safe"))
        .collect();
    
    if p1_filtered.is_empty() {
        return Err("No actions survived P1 filtering");
    }
    
    // P2: Optimize within channels - select shortest action
    let selected = p1_filtered
        .into_iter()
        .min_by_key(|action| action.len())
        .expect("Should have at least one action after P1");
    
    Ok(selected)
}
```

### **Groq Llama-70b output:**
```rust
fn apply_deadband_protocol(actions: Vec<String>) -> Result<String, &'static str> {
    // P0: eliminate actions with "error"
    let after_p0: Vec<_> = actions
        .iter()
        .filter(|a| !a.contains("error"))
        .cloned()
        .collect();
    if after_p0.is_empty() {
        return Err("All actions eliminated at P0");
    }
    
    // P1: keep only actions with "safe"
    let after_p1: Vec<_> = after_p0
        .iter()
        .filter(|a| a.contains("safe"))
        .cloned()
        .collect();
    if after_p1.is_empty() {
        return Err("No safe actions found at P1");
    }
    
    // P2: pick the shortest
    let best = after_p1
        .iter()
        .min_by_key(|a| a.len())
        .unwrap()
        .clone();
    
    Ok(best)
}
```

**Comparison:**
- **Correctness:** Both implement the logic correctly.
- **Fleet patterns:** DeepSeek uses `to_lowercase()` for case-insensitive matching (more robust). Groq does exact matching.
- **Error handling:** DeepSeek provides slightly more descriptive error messages.
- **Ownership:** DeepSeek uses `into_iter()` appropriately; Groq mixes `iter()` and `cloned()`.
- **Readability:** Both are clear; DeepSeek has better comments aligning with P0/P1/P2 doctrine.

## **3. Function 2: Tile Validator**

**Prompt:** "Write a Rust function `validate_tile` that takes a JSON string and validates it as a training tile. A valid tile must have fields: `id` (string), `content` (string, non-empty), `agent_role` (string, one of: navigator, sentinel, scribe, tinker, scout, curator, mason, alchemist, herald, scholar, weaver, archivist), `cycle` (positive integer). Return `Result<(), Vec<String>>` where Ok(()) means valid, Err contains list of validation errors."

### **DeepSeek-chat output:**
```rust
use serde_json::Value;
use std::collections::HashSet;

pub fn validate_tile(json_str: &str) -> Result<(), Vec<String>> {
    let mut errors = Vec::new();
    
    let tile: Value = match serde_json::from_str(json_str) {
        Ok(v) => v,
        Err(e) => {
            errors.push(format!("Invalid JSON: {}", e));
            return Err(errors);
        }
    };
    
    // Check required fields
    if !tile["id"].is_string() {
        errors.push("Missing or non-string field: id".to_string());
    }
    
    if let Some(content) = tile["content"].as_str() {
        if content.trim().is_empty() {
            errors.push("Field 'content' is empty".to_string());
        }
    } else {
        errors.push("Missing or non-string field: content".to_string());
    }
    
    let valid_roles: HashSet<&str> = [
        "navigator", "sentinel", "scribe", "tinker", "scout", "curator",
        "mason", "alchemist", "herald", "scholar", "weaver", "archivist"
    ].iter().cloned().collect();
    
    if let Some(role) = tile["agent_role"].as_str() {
        if !valid_roles.contains(role) {
            errors.push(format!("Invalid agent_role: '{}'", role));
        }
    } else {
        errors.push("Missing or non-string field: agent_role".to_string());
    }
    
    if let Some(cycle) = tile["cycle"].as_i64() {
        if cycle <= 0 {
            errors.push("Field 'cycle' must be positive".to_string());
        }
    } else {
        errors.push("Missing or non-integer field: cycle".to_string());
    }
    
    if errors.is_empty() {
        Ok(())
    } else {
        Err(errors)
    }
}
```
