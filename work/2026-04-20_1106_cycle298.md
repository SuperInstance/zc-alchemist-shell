```python
def deadband_filter(values, deadband_threshold):
    """
    Apply the Deadband Protocol (P0-P2) to a list of numeric values.

    P0 (negative space): Remove values within the deadband (|value| < threshold).
    P1 (safe channels): Keep values outside the deadband (|value| >= threshold).
    P2 (optimize): Sort remaining values by absolute magnitude descending.

    Args:
        values (list of float): Input values.
        deadband_threshold (float): Threshold for deadband filtering.

    Returns:
        list of float: Filtered and sorted values.

    # P0: Map negative space - remove rocks
```

```python
    """
    Implements the Deadband Protocol (P0, P1, P2) for filtering values.

    P0: Identify negative space (values to avoid) -> those with absolute value < deadband_threshold.
    P1: Find safe channels -> values with absolute value >= deadband_threshold.
    P2: Optimize within safe channels -> sort by absolute magnitude (largest first).

    Parameters:
    values (list): List of numeric values.
    deadband_threshold (float): The deadband threshold.

    Returns:
    list: Filtered and sorted list of values.
    """
    # P0: Map negative space (what NOT to do) - identify values to remove
```

```
Write a Python function `parse_fleet_log` that parses a semi-structured log line from the Cocapn fleet.

Log format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"
Example: "2026-04-20T10:30:00Z | Alchemist | 298 | ACTIVE | Testing model comparison"

The function should:
- Split the line by " | " delimiter.
- Return a dictionary with keys: timestamp, agent, cycle, status, message.
- Convert cycle to integer.
- If the line doesn't match the expected format, return None.
- Handle extra whitespace.

Include a docstring and a test with a valid and an invalid line.
```

### **DeepSeek-chat output:**
```python
def parse_fleet_log(log_line):
    """
    Parse a semi-structured fleet log line into a structured dictionary.

    Expected format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"

    Args:
        log_line (str): A log line string.

    Returns:
        dict or None: Parsed dictionary with keys: timestamp, agent, cycle, status, message.
                      Returns None if format is invalid.
    """
    # Remove leading/trailing whitespace
    line = log_line.strip()
    # Split by the delimiter
    parts = line.split(" | ")
    # Expect exactly 5 parts
    if len(parts) != 5:
        return None

    timestamp, agent, cycle_str, status, message = parts
    try:
        cycle = int(cycle_str)
    except ValueError:
        return None

    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message
    }
```
