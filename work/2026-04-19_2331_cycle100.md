## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

The function should take two arguments:
1. `actions`: A list of floats representing proposed actions or values.
2. `threshold`: A float representing the P0 deadband threshold.

The Deadband Protocol has three priority levels:
- P0 (Map negative space): Any action with absolute value below `threshold` is FORBIDDEN. Set these to 0.0.
- P1 (Find safe channels): Of the remaining actions, identify those that are valid (non-zero).
- P2 (Optimize within channels): From the P1 valid actions, select the one with the maximum absolute value to execute. Return its index and value.

The function must return a tuple: `(p0_filtered_actions, p1_valid_indices, p2_selected_index, p2_selected_value)`.

Include a clear docstring and example usage.
```

### **Alchemist (DeepSeek-chat) Output:**
```python
def deadband_filter(actions, threshold):
    """
    Apply the Cocapn fleet Deadband Protocol to a list of actions.

    P0: Eliminate actions within the deadband (|action| < threshold).
    P1: Identify indices of surviving (valid) actions.
    P2: Select the valid action with maximum magnitude for execution.

    Args:
        actions (list[float]): List of proposed action values.
        threshold (float): P0 deadband threshold. Actions with |value| < threshold are zeroed.

    Returns:
        tuple: Contains four elements:
            - p0_filtered (list[float]): Actions after P0 zeroing.
            - p1_indices (list[int]): Indices where p0_filtered is non-zero.
            - p2_index (int or None): Index of selected action (in original list). None if no valid actions.
            - p2_value (float or None): Value of selected action. None if no valid actions.
    """
    # P0: Map negative space
    p0_filtered = []
    for val in actions:
        if abs(val) < threshold:
            p0_filtered.append(0.0)
        else:
            p0_filtered.append(val)

    # P1: Find safe channels
    p1_indices = [i for i, val in enumerate(p0_filtered) if val != 0.0]

    # P2: Optimize within channels
    p2_index = None
    p2_value = None
    if p1_indices:
        # Find index with maximum absolute value in the original filtered list
        candidates = [(i, p0_filtered[i]) for i in p1_indices]
        p2_index, p2_value = max(candidates, key=lambda x: abs(x[1]))

    return p0_filtered, p1_indices, p2_index, p2_value
```

```python
def deadband_filter(actions, threshold):
    """
    Implements Deadband Protocol for action selection.

    Parameters:
    actions (list): list of action values (floats)
    threshold (float): deadband limit

    Returns:
    tuple: (filtered_actions, valid_indices, selected_index, selected_value)
    """
    import numpy as np

    # Convert to numpy for easier handling
    arr = np.array(actions)
    # P0 step
    filtered = np.where(np.abs(arr) < threshold, 0.0, arr)
    # P1 step
    valid_idx = np.where(filtered != 0.0)[0].tolist()
    # P2 step
    if len(valid_idx) > 0:
        # get values at valid indices
        valid_vals = filtered[valid_idx]
        # find max absolute
        abs_vals = np.abs(valid_vals)
        max_idx_rel = np.argmax(abs_vals)
        selected_index = valid_idx[max_idx_rel]
        selected_value = filtered[selected_index]
    else:
        selected_index = None
        selected_value = None

    return filtered.tolist(), valid_idx, selected_index, selected_value
```
