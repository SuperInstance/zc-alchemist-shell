## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function `deadband_filter(values: list[float], deadband: float = 0.1) -> list[float]`.
It implements the Deadband Protocol from the Cocapn fleet:
- P0: Remove values that are within `deadband` of zero (absolute value <= deadband). These are "negative space" – ignore noise.
- P1: Keep only values that survive P0.
- P2: Scale surviving values linearly to range [-1, 1] based on original max magnitude.
Return the filtered and scaled list.
Include a docstring and type hints. Example: deadband_filter([-0.05, 0.2, 0.0, 0.8], 0.1) -> [-0.25, 1.0].
```

### **DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(values: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply the Deadband Protocol to a list of float values.
    
    P0: Remove values within deadband of zero (abs(val) <= deadband).
    P1: Keep surviving values.
    P2: Scale surviving values linearly to range [-1, 1] based on max magnitude.
    
    Args:
        values: Input list of floats.
        deadband: Threshold for zero deadband (default 0.1).
    
    Returns:
        List of filtered and scaled floats.
    
    Example:
        >>> deadband_filter([-0.05, 0.2, 0.0, 0.8], 0.1)
        [-0.25, 1.0]
    """
    # P0: Filter out values in deadband
    filtered = [v for v in values if abs(v) > deadband]
    
    if not filtered:
        return []
    
    # Find max magnitude among survivors
    max_mag = max(abs(v) for v in filtered)
    
    # P2: Scale to [-1, 1]
    scaled = [v / max_mag for v in filtered]
    
    return scaled
```

```python
def deadband_filter(values: List[float], deadband: float = 0.1) -> List[float]:
    """
    Implements the Deadband Protocol (P0, P1, P2) for signal filtering.
    
    Steps:
    1. P0: Remove values within deadband of zero (|value| <= deadband).
    2. P1: Retain remaining values.
    3. P2: Linearly scale retained values to [-1, 1] based on maximum magnitude.
    
    Args:
        values: List of input float values.
        deadband: Deadband threshold (default 0.1).
    
    Returns:
        List of filtered and scaled float values.
    
    Example:
        >>> deadband_filter([-0.05, 0.2, 0.0, 0.8], 0.1)
        [-0.25, 1.0]
    """
    # P0: Identify negative space (values to remove)
    survivors = [v for v in values if abs(v) > deadband]
    
    # If no survivors, return empty list
    if not survivors:
        return []
    
    # P2: Determine scaling factor
    max_abs = max(abs(v) for v in survivors)
    scaled = [v / max_abs for v in survivors]
    
    return scaled
```

## **3. Function 2: `parse_fleet_log`**

**Prompt:**
```
Write a Python function `parse_fleet_log(line: str) -> dict`.
Parse a Cocapn fleet log line format: "2026-04-20 12:34:56 [AGENT] [LEVEL] message @key=value key2=value2".
Return dict with keys: timestamp (datetime), agent (str), level (str), message (str), and a dict 'tags' for key=value pairs.
If line doesn't match format, return empty dict.
Use datetime.strptime. Include docstring and example.
Example: parse_fleet_log('2026-04-20 12:34:56 [Alchemist] [INFO] Task started @cycle=116 model=deepseek')
```

### **DeepSeek-chat output:**
```python
import re
from datetime import datetime
from typing import Dict, Any

def parse_fleet_log(line: str) -> Dict[str, Any]:
    """
    Parse a structured fleet log line into a dictionary.
    
    Format: "YYYY-MM-DD HH:MM:SS [AGENT] [LEVEL] message @key=value key2=value2"
    
    Args:
        line: Log line string.
    
    Returns:
        Dictionary with keys: timestamp (datetime), agent (str), level (str),
        message (str), tags (dict). Returns empty dict if format invalid.
    
    Example:
        >>> parse_fleet_log('2026-04-20 12:34:56 [Alchemist] [INFO] Task started @cycle=116 model=deepseek')
        {'timestamp': datetime(2026, 4, 20, 12, 34, 56),
         'agent': 'Alchemist',
         'level': 'INFO',
         'message': 'Task started',
         'tags': {'cycle': '116', 'model': 'deepseek'}}
    """
    pattern = r'(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) \[([^\]]+)\] \[([^\]]+)\] (.+?)(?: @(.+))?$'
    match = re.match(pattern, line.strip())
    if not match:
        return {}
    
    timestamp_str, agent, level, message, tags_str = match.groups()
    
    try:
        timestamp = datetime.strptime(timestamp_str, '%Y-%m-%d %H:%M:%S')
    except ValueError:
        return {}
    
    # Parse tags
    tags = {}
    if tags_str:
        for pair in tags_str.split():
            if '=' in pair:
                key, value = pair.split('=', 1)
                tags[key] = value
    
    return {
        'timestamp': timestamp,
        'agent': agent,
        'level': level,
        'message': message.strip(),
        'tags': tags
    }
```
