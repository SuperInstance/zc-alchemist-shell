## **2. Function 1: Deadband Filter**

**Specification:** `apply_deadband_protocol(readings: List[float], deadband: float) -> List[str]`
Takes a list of sensor readings. For each reading:
*   **P0 (Negative Space):** If reading is `NaN` or `Inf`, return `"ERROR"`.
*   **P1 (Safe Channel):** If reading is within `[-deadband, +deadband]`, return `"SAFE"`.
*   **P2 (Optimize):** Otherwise, return `"ACTIVE"`.

### **DeepSeek-chat Implementation:**
```python
from typing import List
import math

def apply_deadband_protocol(readings: List[float], deadband: float) -> List[str]:
    """
    Applies the fleet's Deadband Protocol (P0/P1/P2) to a list of sensor readings.

    Args:
        readings: List of numeric sensor values.
        deadband: Threshold for the safe channel (P1). Must be non-negative.

    Returns:
        List of protocol states: "ERROR" (P0), "SAFE" (P1), or "ACTIVE" (P2).

    Fleet Context:
        Implements strict priority: P0 (safety) is checked first, then P1 (viability).
    """
    if deadband < 0:
        raise ValueError("Deadband threshold must be non-negative.")

    results = []
    for val in readings:
        # P0: Map Negative Space - Invalid inputs
        if math.isnan(val) or math.isinf(val):
            results.append("ERROR")
            continue

        # P1: Find Safe Channel - Within operational deadband
        if -deadband <= val <= deadband:
            results.append("SAFE")
            continue

        # P2: Optimize - Outside deadband, active signal
        results.append("ACTIVE")
    return results
```

## **3. Function 2: Training Tile Validator**

**Specification:** `validate_training_tile(tile_text: str, min_length: int = 50) -> tuple[bool, List[str]]`
Validates a text block intended as a fleet training tile.
Returns `(is_valid, list_of_violations)`.
**Rules:**
1.  Tile must not be empty.
2.  Tile must meet minimum length.
3.  Tile must not contain absolute claim markers ("always", "never", "guaranteed").
4.  Tile must contain at least one reference to a fleet concept (e.g., "P0", "deadband", "ensign", "PLATO").

### **DeepSeek-chat Implementation:**
```python
from typing import Tuple, List

def validate_training_tile(tile_text: str, min_length: int = 50) -> Tuple[bool, List[str]]:
    """
    Validates a proposed training tile against fleet submission criteria.

    Args:
        tile_text: The candidate tile content.
        min_length: Minimum acceptable character length.

    Returns:
        A tuple (is_valid, violations). `violations` is a list of descriptive strings
        for any failed checks. Empty list if valid.

    Fleet Context:
        Enforces rules against absolute claims and requires fleet-specific knowledge,
        ensuring tiles contribute to context-aware training.
    """
    violations = []
    tile_lower = tile_text.lower()

    # Rule 1: Not empty
    if not tile_text.strip():
        violations.append("Tile text is empty or only whitespace.")

    # Rule 2: Minimum length
    if len(tile_text.strip()) < min_length:
        violations.append(f"Tile length ({len(tile_text.strip())}) < minimum ({min_length}).")

    # Rule 3: No absolute claims
    absolute_markers = ["always", "never", "guaranteed", "certainly", "definitely"]
    found_markers = [marker for marker in absolute_markers if marker in tile_lower]
    if found_markers:
        violations.append(f"Contains absolute claim markers: {found_markers}")

    # Rule 4: Contains fleet concept
    fleet_concepts = ["p0", "p1", "p2", "deadband", "ensign", "plato", "zeroclaw", "oracle", "jetson", "forge", "training room", "ghost tile"]
    if not any(concept in tile_lower for concept in fleet_concepts):
        violations.append("Does not reference any core fleet concept.")

    is_valid = len(violations) == 0
    return (is_valid, violations)
```
