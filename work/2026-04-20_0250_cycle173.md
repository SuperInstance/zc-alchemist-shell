### **Function 1: Deadband Filter**

**Prompt to both models:**
```
Write a Python function `apply_deadband_protocol` that takes a list of action dictionaries. Each dict has keys `action` (str) and `risk_score` (float 0-1). Implement the fleet's Deadband Protocol:
- P0 (risk_score >= 0.7): Reject action (return None for that action)
- P1 (0.3 <= risk_score < 0.7): Flag for review (return action with `status: review`)
- P2 (risk_score < 0.3): Approve (return action with `status: approved`)
Return a list of processed actions. Include type hints and a docstring.
```

**DeepSeek-chat output:**
```python
from typing import List, Dict, Optional, Union

def apply_deadband_protocol(actions: List[Dict[str, Union[str, float]]]) -> List[Dict[str, Union[str, float, None]]]:
    """
    Apply the Cocapn fleet's Deadband Protocol to a list of actions.
    
    Protocol priorities:
    - P0 (risk_score >= 0.7): Negative space - reject action
    - P1 (0.3 <= risk_score < 0.7): Safe channel - flag for review  
    - P2 (risk_score < 0.3): Optimization zone - approve
    
    Args:
        actions: List of action dicts with 'action' (str) and 'risk_score' (float 0-1)
    
    Returns:
        List of processed actions with added 'status' field and possibly None for rejected actions
    """
    processed = []
    
    for action in actions:
        # Validate input structure
        if 'action' not in action or 'risk_score' not in action:
            processed.append({**action, 'status': 'invalid', 'note': 'Missing required fields'})
            continue
            
        risk = action['risk_score']
        
        # P0: Map negative space (reject)
        if risk >= 0.7:
            processed.append({
                'action': action['action'],
                'risk_score': risk,
                'status': 'rejected',
                'protocol_phase': 'P0',
                'note': 'High risk - negative space'
            })
        
        # P1: Find safe channels (review)
        elif 0.3 <= risk < 0.7:
            processed.append({
                **action,
                'status': 'review',
                'protocol_phase': 'P1', 
                'note': 'Medium risk - requires review'
            })
        
        # P2: Optimize within channels (approve)
        else:  # risk < 0.3
            processed.append({
                **action,
                'status': 'approved',
                'protocol_phase': 'P2',
                'note': 'Low risk - optimization zone'
            })
    
    return processed
```

```python
    """
    Implements the Deadband Protocol (P0/P1/P2) for action filtering.
    
    Parameters:
    -----------
    actions : List[Dict[str, Any]]
        Each dictionary must contain:
        - 'action': str describing the action
        - 'risk_score': float between 0 and 1
        
    Returns:
    --------
    List[Dict[str, Any]]
        Each dictionary contains original keys plus:
        - 'status': 'rejected', 'review', or 'approved'
        - 'phase': 'P0', 'P1', or 'P2'
        - 'reason': str explanation
    """
    result = []
    
    for act in actions:
        # Input validation
        if not isinstance(act.get('risk_score'), (int, float)):
            act_copy = act.copy()
            act_copy.update({
                'status': 'error',
                'phase': 'invalid',
                'reason': 'risk_score must be numeric'
            })
            result.append(act_copy)
            continue
            
        risk = float(act['risk_score'])
        
        # Clamp to [0, 1] range
        if risk < 0:
            risk = 0.0
        elif risk > 1:
            risk = 1.0
        
        # Apply protocol
        if risk >= 0.7:
            status = 'rejected'
            phase = 'P0'
            reason = 'P0: Negative space - risk too high'
        elif risk >= 0.3:
            status = 'review' 
            phase = 'P1'
            reason = 'P1: Safe channel - requires manual review'
        else:
            status = 'approved'
            phase = 'P2'
            reason = 'P2: Optimization zone - proceed'
        
        act_copy = act.copy()
        act_copy.update({
            'status': status,
            'phase': phase,
            'reason': reason
        })
        result.append(act_copy)
    
    return result
```
