## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

The function should take two arguments:
1.  `readings`: A list of floats representing sequential sensor readings.
2.  `deadband_width`: A float representing the minimum change required to register a new value.

The Deadband Protocol has three phases:
P0 (Map negative space): Identify readings that would cause an error (e.g., NaN, infinite values). Log them and filter them out.
P1 (Find safe channels): From the cleaned readings, find the first reading that establishes a baseline.
P2 (Optimize within channels): For each subsequent reading, only keep it if its absolute difference from the last kept reading exceeds `deadband_width`. This "snaps" the output to a channel, reducing noise.

Return a list of filtered readings that adhere to the protocol.

Include docstring, type hints, and basic error handling.
```

### **DeepSeek-chat Implementation:**
```python
from typing import List, Optional
import math

def deadband_filter(readings: List[float], deadband_width: float) -> List[float]:
    """
    Apply the Cocapn fleet's Deadband Protocol to a sequence of sensor readings.

    Protocol:
    - P0: Filter out invalid readings (NaN, inf).
    - P1: Establish baseline from first valid reading.
    - P2: Keep only readings that change beyond the deadband width.

    Args:
        readings: List of sequential sensor readings.
        deadband_width: Minimum absolute change required to keep a new reading.

    Returns:
        Filtered list of readings adhering to the protocol.

    Example:
        >>> deadband_filter([1.0, 1.1, 1.2, 5.0, 5.1], 0.5)
        [1.0, 5.0]
    """
    if not readings:
        return []

    # P0: Map negative space - filter out invalid values
    valid_readings = []
    invalid_count = 0
    for val in readings:
        if val is None or math.isnan(val) or math.isinf(val):
            invalid_count += 1
            # In production, would log this P0 violation
            continue
        valid_readings.append(val)

    if invalid_count > 0:
        print(f"[P0] Filtered {invalid_count} invalid readings")

    if not valid_readings:
        return []

    # P1: Find safe channels - establish baseline
    filtered = [valid_readings[0]]
    baseline = valid_readings[0]

    # P2: Optimize within channels
    for val in valid_readings[1:]:
        if abs(val - baseline) >= deadband_width:
            filtered.append(val)
            baseline = val  # Update the channel baseline

    return filtered
```

```python
from typing import List
import math

def deadband_filter(readings: List[float], deadband_width: float) -> List[float]:
    """
    Implements the Deadband Protocol (P0-P2) for sensor data filtering.

    P0 (Negative Space): Removes non-finite values (NaN, inf).
    P1 (Safe Channels): Establishes initial baseline from first valid reading.
    P2 (Optimization): Only keeps readings that deviate beyond deadband_width.

    Args:
        readings: Input sensor readings.
        deadband_width: Minimum change to trigger a new kept reading.

    Returns:
        Filtered readings following deadband protocol.
    """
    # Edge case: empty input
    if not readings:
        return []

    # P0: Remove hazardous readings
    safe_readings = []
    hazards = 0
    for r in readings:
        if r is None:
            hazards += 1
            continue
        try:
            if math.isfinite(r):
                safe_readings.append(r)
            else:
                hazards += 1
        except (TypeError, ValueError):
            hazards += 1

    if hazards:
        print(f"P0: Removed {hazards} hazardous readings")

    if not safe_readings:
        return []

    # P1: Establish first safe channel
    result = [safe_readings[0]]
    current_channel = safe_readings[0]

    # P2: Optimize within channels
    for val in safe_readings[1:]:
        if abs(val - current_channel) > deadband_width:  # Note: uses > not >=
            result.append(val)
            current_channel = val

    return result
```
