### **Function 1: `validate_deadband_protocol`**

**Prompt to both models:**
"Write a Python function called `validate_deadband_protocol` that takes a list of strings `steps`. It must implement the Deadband Protocol priority rules: P0 (identify negatives) must come first, P1 (find safe channels) must come before P2 (optimize). The function returns a tuple (bool, str). Return (False, "P0 step missing") if no step contains "P0:". Return (False, "P2 before P1") if a step containing "P2:" appears before any step containing "P1:". Otherwise, return (True, ""). Make the check case-sensitive. Include a brief docstring."

**DeepSeek-chat Output:**
```python
def validate_deadband_protocol(steps: list[str]) -> tuple[bool, str]:
    """
    Validates if a list of steps follows the Deadband Protocol priority order.
    Rules:
    1. At least one step must start with "P0:" (case-sensitive).
    2. No step starting with "P2:" may appear before a step starting with "P1:".
    Returns a tuple (is_valid, error_message).
    """
    has_p0 = any(step.strip().startswith("P0:") for step in steps)
    if not has_p0:
        return False, "P0 step missing"

    seen_p1 = False
    for step in steps:
        if step.strip().startswith("P1:"):
            seen_p1 = True
        if step.strip().startswith("P2:") and not seen_p1:
            return False, "P2 before P1"

    return True, ""
```

**Groq Llama-3.3-70b-Specdec Output:**

```python
def validate_deadband_protocol(steps: list[str]) -> tuple[bool, str]:
    """
    Validate the Deadband Protocol steps.
    The protocol requires:
    - At least one P0 step
    - No P2 step before a P1 step
    """
    # Check for P0
    p0_present = any(step.startswith("P0:") for step in steps)
    if not p0_present:
        return (False, "P0 step missing")

    # Check P2 before P1
    p1_seen = False
    for step in steps:
        if step.startswith("P1:"):
            p1_seen = True
        elif step.startswith("P2:") and not p1_seen:
            return (False, "P2 before P1")

    return (True, "")
```

### **Function 2: `parse_fleet_manifest`**

**Prompt to both models:**
"Write a Python function `parse_fleet_manifest` that takes a string `manifest_text`. It should parse a simple fleet manifest format. Lines may look like: 'Agent: Oracle1 (Status: Active, Model: deepseek-chat)'. The function must return a dict with: 'agent_count' (total agents), 'active_agents' (count where Status is Active), 'models' (a set of unique model names), and 'last_updated' (the most recent date string found in lines starting with 'Updated:'). Assume dates are in YYYY-MM-DD format. Ignore malformed lines. Include a docstring."

**DeepSeek-chat Output:**
```python
def parse_fleet_manifest(manifest_text: str) -> dict:
    """
    Parses a fleet manifest text and extracts key information.
    Expected format per agent line:
        Agent: <name> (Status: <status>, Model: <model>)
    Also looks for update lines:
        Updated: YYYY-MM-DD
    Returns a dictionary with agent_count, active_agents, models, last_updated.
    """
    lines = manifest_text.strip().split('\n')
    agent_count = 0
    active_agents = 0
    models = set()
    last_updated = ""

    for line in lines:
        line = line.strip()
        # Parse agent line
        if line.startswith('Agent:'):
            agent_count += 1
            # Extract status and model using simple string parsing
            if 'Status: Active' in line:
                active_agents += 1
            # Extract model - assuming format "Model: some-model)"
            start = line.find('Model:')
            if start != -1:
                end = line.find(')', start)
                if end != -1:
                    model = line[start + len('Model:'):end].strip()
                    models.add(model)
        # Parse update line
        elif line.startswith('Updated:'):
            date_str = line.split('Updated:')[1].strip()
            # Keep the latest date
            if date_str > last_updated:
                last_updated = date_str

    return {
        'agent_count': agent_count,
        'active_agents': active_agents,
        'models': list(models),
        'last_updated': last_updated
    }
```

**Groq Llama-3.3-70b-Specdec Output:**
```python
def parse_fleet_manifest(manifest_text: str) -> dict:
    import re

    """
    Parse a fleet manifest string and return structured data.
    """
    agent_count = 0
    active_agents = 0
    models = set()
    last_updated = ""

    for line in manifest_text.splitlines():
        line = line.strip()
        # Agent line
        if line.startswith("Agent:"):
            agent_count += 1
            # Use regex to extract status and model
            status_match = re.search(r"Status:\s*(\w+)", line)
            model_match = re.search(r"Model:\s*([^),]+)", line)

            if status_match and status_match.group(1) == "Active":
                active_agents += 1

            if model_match:
                models.add(model_match.group(1).strip())

        # Updated line
        elif line.startswith("Updated:"):
            date_part = line.split("Updated:")[1].strip()
            # Simple date comparison (string lexicographic works for YYYY-MM-DD)
            if date_part > last_updated:
                last_updated = date_part

    return {
        "agent_count": agent_count,
        "active_agents": active_agents,
        "models": sorted(list(models)),  # Sorted for consistency
        "last_updated": last_updated
    }
```
