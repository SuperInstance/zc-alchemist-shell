### **Function 1: `deadband_filter`**

**Specification:** Takes a list of floats `values` and a `deadband` threshold. Returns a list of tuples `(original_value, status)`. Status is determined by:
- `P0`: value <= -deadband (negative space)
- `P1`: -deadband < value < deadband (safe channel)
- `P2`: value >= deadband (optimization zone)

#### **DeepSeek-chat Implementation:**
```python
from typing import List, Tuple

def deadband_filter(values: List[float], deadband: float = 1.0) -> List[Tuple[float, str]]:
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of numeric values.

    Args:
        values: List of input values.
        deadband: Threshold for classification. Must be non-negative.

    Returns:
        List of tuples (original_value, status), where status is 'P0', 'P1', or 'P2'.

    Raises:
        ValueError: If deadband is negative.
    """
    if deadband < 0:
        raise ValueError("deadband must be non-negative")

    result = []
    for v in values:
        if v <= -deadband:
            status = "P0"
        elif v < deadband:
            status = "P1"
        else:  # v >= deadband
            status = "P2"
        result.append((v, status))
    return result
```

### **Function 2: `parse_fleet_log`**

**Specification:** Parse a log line format: `[TIMESTAMP] AGENT:CYCLE "MESSAGE" [STATUS]`. Return a dict with keys: `timestamp`, `agent`, `cycle`, `message`, `status`.

#### **DeepSeek-chat Implementation:**
```python
import re
from typing import Dict, Optional

def parse_fleet_log(log_line: str) -> Optional[Dict[str, str]]:
    """
    Parse a structured fleet log entry.

    Expected format: [YYYY-MM-DD HH:MM:SS] AGENT:CYCLE "MESSAGE" [STATUS]

    Args:
        log_line: A single line of fleet log.

    Returns:
        Dictionary with keys: timestamp, agent, cycle, message, status.
        Returns None if the line does not match the expected pattern.
    """
    pattern = r'\[(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2})\] (\w+):(\d+) "([^"]*)" \[(\w+)\]'
    match = re.match(pattern, log_line)
    if not match:
        return None

    timestamp, agent, cycle, message, status = match.groups()
    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": int(cycle),
        "message": message,
        "status": status
    }
```
