## **3. Function 1: deadband_filter**

**Problem Description:**  
Implement `deadband_filter(values: list[float], deadband: float = 0.1) -> list[float]`.  
Apply the Deadband Protocol:  
- P0: Remove any values where `abs(value) < deadband` (map negative space — filter out noise).  
- P1: Keep only values where `deadband <= abs(value) <= 1.0` (find safe channels).  
- P2: Return the filtered list sorted by absolute magnitude (optimize within channels).  
If no values survive P1, return empty list.

---

### **DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(values: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply the Deadband Protocol to filter and sort a list of float values.
    
    Steps:
    1. P0: Remove values with absolute magnitude below deadband (noise).
    2. P1: Keep only values within safe channel [deadband, 1.0].
    3. P2: Sort remaining values by absolute magnitude.
    
    Args:
        values: List of float values to filter.
        deadband: Threshold for noise filtering (default 0.1).
    
    Returns:
        Filtered and sorted list of values meeting deadband criteria.
    
    Examples:
        >>> deadband_filter([0.05, 0.2, -0.3, 1.2, -0.09])
        [0.2, -0.3]
        >>> deadband_filter([0.01, 0.02], deadband=0.05)
        []
        >>> deadband_filter([-0.5, 0.15, 0.8, -1.1])
        [0.15, -0.5, 0.8]
    """
    if not values:
        return []
    
    # P0 + P1: Filter noise and keep safe channel
    filtered = [
        v for v in values
        if deadband <= abs(v) <= 1.0
    ]
    
    # P2: Sort by absolute magnitude
    filtered.sort(key=abs)
    
    return filtered
```

## **4. Function 2: parse_fleet_log**

**Problem Description:**  
Implement `parse_fleet_log(log_text: str) -> dict`.  
Parse a raw fleet log line in format:  
`[TIMESTAMP] AGENT: "message" | STATUS: status | CYCLE: number`  
Return dict with keys: `timestamp`, `agent`, `message`, `status`, `cycle`.  
If format doesn't match, return dict with all values as `None`.  
Timestamp may be in `YYYY-MM-DD HH:MM:SS` or `HH:MM:SS` format.

---

### **DeepSeek-chat output:**
```python
import re
from typing import Optional, Dict

def parse_fleet_log(log_text: str) -> Dict[str, Optional[str]]:
    """
    Parse a structured fleet log line into its components.
    
    Expected format:
    [TIMESTAMP] AGENT: "message" | STATUS: status | CYCLE: number
    
    Args:
        log_text: A single line of fleet log.
    
    Returns:
        Dictionary with keys: timestamp, agent, message, status, cycle.
        All values are strings except cycle which is int.
        If parsing fails, all values are None.
    
    Examples:
        >>> parse_fleet_log('[2026-04-20 11:32:00] Oracle1: "Task completed" | STATUS: active | CYCLE: 304')
        {'timestamp': '2026-04-20 11:32:00', 'agent': 'Oracle1', 'message': 'Task completed', 'status': 'active', 'cycle': 304}
        
        >>> parse_fleet_log('[11:32:00] JetsonClaw1: "GPU temp high" | STATUS: warning | CYCLE: 305')
        {'timestamp': '11:32:00', 'agent': 'JetsonClaw1', 'message': 'GPU temp high', 'status': 'warning', 'cycle': 305}
        
        >>> parse_fleet_log('invalid log line')
        {'timestamp': None, 'agent': None, 'message': None, 'status': None, 'cycle': None}
    """
    pattern = r'\[(.+?)\]\s+(\w+):\s+"(.+?)"\s+\|\s+STATUS:\s+(\w+)\s+\|\s+CYCLE:\s+(\d+)'
    match = re.match(pattern, log_text.strip())
    
    if not match:
        return {
            'timestamp': None,
            'agent': None,
            'message': None,
            'status': None,
            'cycle': None
        }
    
    timestamp, agent, message, status, cycle_str = match.groups()
    
    return {
        'timestamp': timestamp,
        'agent': agent,
        'message': message,
        'status': status,
        'cycle': int(cycle_str)
    }
```
