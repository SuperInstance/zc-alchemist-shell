## **2. Function 1: `validate_deadband_protocol`**

**Prompt:**
```
Write a Python function called `validate_deadband_protocol` that validates a sequence of actions against the Cocapn Fleet's Deadband Protocol.

The Deadband Protocol has three strict priority levels:
P0: "Map negative space" (identify what NOT to do)
P1: "Find safe channels" (identify where you CAN operate)
P2: "Optimize within channels" (find the best path within safe channels)

Rule: You must complete at least one P0 action before any P1 action, and at least one P1 action before any P2 action. You cannot skip a priority level.

Input:
- `actions`: A list of strings. Each string is an action description that starts with its priority level, e.g., "P0: Check for system errors", "P1: Scan network ports", "P2: Optimize database query".

Output:
- Return a tuple `(is_valid, message)`.
- `is_valid`: Boolean. True if the sequence respects the protocol.
- `message`: String. If valid, "Sequence valid". If invalid, explain the violation, e.g., "P1 action found before P0", "P2 action found before P1".

Assume the action string always starts with "P0:", "P1:", or "P2:".
```

### **DeepSeek-chat Output:**
```python
def validate_deadband_protocol(actions):
    """
    Validate a sequence of actions against the Deadband Protocol.
    Returns (is_valid, message).
    """
    seen_p0 = False
    seen_p1 = False
    
    for i, action in enumerate(actions):
        if action.startswith("P0:"):
            seen_p0 = True
        elif action.startswith("P1:"):
            if not seen_p0:
                return (False, f"P1 action found before P0 at index {i}")
            seen_p1 = True
        elif action.startswith("P2:"):
            if not seen_p0:
                return (False, f"P2 action found before P0 at index {i}")
            if not seen_p1:
                return (False, f"P2 action found before P1 at index {i}")
        else:
            # In case of malformed priority tag
            return (False, f"Invalid priority tag in action at index {i}: {action}")
    
    return (True, "Sequence valid")
```

## **3. Function 2: `parse_fleet_tile`**

**Prompt:**
```
Write a Python function called `parse_fleet_tile` that parses a fleet training tile (a text string) and extracts metadata.

The tile format is a Markdown-like string:
```
# Cycle [number] — [Role]
**Phase:** [phase number]
**Task:** [task description]
---
[Content body here...]
```

The function should:
1. Take a string `tile_text` as input.
2. Extract:
   - `cycle` (int): The cycle number.
   - `role` (str): The agent's role (e.g., "Alchemist").
   - `phase` (int): The phase number.
   - `task` (str): The task description.
   - `content` (str): Everything after the "---" separator.
3. Return a dictionary with these keys. If a field is missing or the format is invalid, set its value to `None`.
4. Use robust string parsing, not complex regex.
```

### **DeepSeek-chat Output:**
```python
def parse_fleet_tile(tile_text):
    """
    Parse a fleet training tile string into a metadata dictionary.
    """
    result = {
        "cycle": None,
        "role": None,
        "phase": None,
        "task": None,
        "content": None
    }
    
    lines = tile_text.strip().split('\n')
    
    # Parse first line for cycle and role
    if lines and lines[0].startswith('# Cycle'):
        first_line = lines[0]
        # Expected format: "# Cycle 135 — Alchemist"
        parts = first_line.split('—')
        if len(parts) == 2:
            cycle_part = parts[0].replace('# Cycle', '').strip()
            result["cycle"] = int(cycle_part) if cycle_part.isdigit() else None
            result["role"] = parts[1].strip()
    
    # Parse subsequent lines for phase and task
    for i, line in enumerate(lines[1:], start=1):
        line_stripped = line.strip()
        if line_stripped.startswith('**Phase:**'):
            phase_part = line_stripped.replace('**Phase:**', '').strip()
            result["phase"] = int(phase_part) if phase_part.isdigit() else None
        elif line_stripped.startswith('**Task:**'):
            result["task"] = line_stripped.replace('**Task:**', '').strip()
        elif line_stripped == '---':
            # Content is everything after this line
            result["content"] = '\n'.join(lines[i+1:]).strip()
            break
    
    return result
```
