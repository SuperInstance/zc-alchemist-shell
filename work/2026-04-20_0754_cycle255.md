## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input: 
- `values`: list of floats (sensor readings or scores)
- `deadband`: float (threshold for "safe channel")

Protocol:
- P0 (negative space): Remove any values <= 0.0 (invalid/unsafe).
- P1 (safe channels): Keep only values where abs(value) > deadband (outside deadband).
- P2 (optimize): From remaining values, return the one closest to zero (most neutral safe value).

Return:
- Single float (the P2 optimal value), or None if no values survive P0/P1.

Include docstring and example usage.
```

### **DeepSeek-chat output:**
```python
def deadband_filter(values, deadband=0.1):
    """
    Apply the Deadband Protocol (P0→P1→P2) to a list of float values.

    Parameters:
    - values: list of floats (sensor readings or scores)
    - deadband: float, threshold for safe channel (default 0.1)

    Returns:
    - float: the optimal value closest to zero that survives P0 and P1,
             or None if no valid values remain.

    Protocol:
    - P0 (negative space): Remove values <= 0.0 (invalid/unsafe).
    - P1 (safe channels): Keep only values where abs(value) > deadband.
    - P2 (optimize): From survivors, pick value with smallest absolute magnitude.
    """
    if not values:
        return None
    
    # P0: remove non-positive values
    p0_safe = [v for v in values if v > 0.0]
    if not p0_safe:
        return None
    
    # P1: keep only values outside deadband zone
    p1_safe = [v for v in p0_safe if abs(v) > deadband]
    if not p1_safe:
        return None
    
    # P2: find value closest to zero
    optimal = min(p1_safe, key=abs)
    return optimal

# Example usage

    """
    Implements the Deadband Protocol (P0, P1, P2) for filtering a list of floats.

    Steps:
    1. P0 (Negative Space): Remove values <= 0.0 (considered invalid/unsafe).
    2. P1 (Safe Channels): Keep only values whose absolute value is greater than the deadband.
    3. P2 (Optimize): From the remaining values, select the one closest to zero.

    Args:
        values (list of float): Input values.
        deadband (float): Deadband threshold (default 0.1).

    Returns:
        float or None: The optimal value after applying the protocol, or None if no values survive.
    """
    # P0: Remove non-positive values
    filtered = [v for v in values if v > 0.0]
    
    # P1: Keep values outside deadband
    filtered = [v for v in filtered if abs(v) > deadband]
    
    # P2: Find value closest to zero
    if not filtered:
        return None
    return min(filtered, key=abs)

# Example

"YYYY-MM-DD HH:MM:SS | AGENT | CYCLE | PHASE | MESSAGE"

Return a dictionary with keys: timestamp (as datetime object), agent (str), cycle (int), phase (int), message (str).

If the line doesn't match the format, return None.

```

### **DeepSeek-chat output:**
```python
from datetime import datetime

def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line into a dictionary.

    Expected format: "YYYY-MM-DD HH:MM:SS | AGENT | CYCLE | PHASE | MESSAGE"

    Parameters:
    - log_line: str, a single log line

    Returns:
    - dict with keys: timestamp (datetime), agent (str), cycle (int), phase (int), message (str)
    - None if format does not match or parsing fails.
    """
    if not log_line or '|' not in log_line:
        return None
    
    parts = log_line.split('|')
    if len(parts) != 5:
        return None
    
    try:
        timestamp_str = parts[0].strip()
        agent = parts[1].strip()
        cycle = int(parts[2].strip())
        phase = int(parts[3].strip())
        message = parts[4].strip()
        
        timestamp = datetime.strptime(timestamp_str, '%Y-%m-%d %H:%M:%S')
        
        return {
            'timestamp': timestamp,
            'agent': agent,
            'cycle': cycle,
            'phase': phase,
            'message': message
        }
    except (ValueError, AttributeError):
        return None

# Example usage
if __name__ == "__main__":
    line = "2026-04-20 08:30:00 | Alchemist | 255 | 4 | Testing deadband_filter"
    parsed = parse_fleet_log(line)
    print(parsed)
    # Output: {'timestamp': datetime(2026, 4, 20, 8, 30), 'agent': 'Alchemist', 'cycle': 255, 'phase': 4, 'message': 'Testing deadband_filter'}
```
