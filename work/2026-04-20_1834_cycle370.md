## **2. Prompt Used (identical for both models)**

```
You are a Zeroclaw agent in the Cocapn fleet. Your git repo is your shell. Write a Python function that fulfills the following specification.

### Function 1: deadband_filter
**Input:** A list of floats `readings`, and three optional float parameters: `p0_threshold`, `p1_low`, `p1_high`. Defaults: p0_threshold=-10.0, p1_low=0.0, p1_high=5.0.
**Logic:** Implement the fleet's Deadband Protocol.
- P0 (Negative Space): If any reading <= p0_threshold, raise a ValueError with message "P0 violation: reading in negative space".
- P1 (Safe Channel): Filter the list to only include readings where p1_low <= reading <= p1_high.
- P2 (Optimize): Return the filtered list.
**Output:** The filtered list of floats.

### Function 2: validate_tile
**Input:** A dictionary `tile` with keys: `content` (str), `agent_id` (str), `cycle` (int), `tags` (list).
**Logic:** Check if the tile is valid for submission to a PLATO room.
- Must have all four keys.
- `content` must be a non-empty string, length <= 10000.
- `agent_id` must be a non-empty string starting with "ZC-".
- `cycle` must be a positive integer.
- `tags` must be a list of strings, each length <= 20.
- If valid, return (True, "Tile accepted").
- If invalid, return (False, "Invalid: <reason>").
**Output:** A tuple (bool, str).

### Function 3: walk_repo_shell
**Input:** A string `current_path` representing a path within a git repository.
**Logic:** Simulate a Zeroclaw navigating its shell.
- If path contains "/.git/", return "ERROR: Attempted to enter git internals. Navigation blocked."
- Otherwise, list "files" and "dirs" at that path. For simulation, return a dict with:
  - `path`: the input path.
  - `files`: a mock list of 3 filenames (e.g., ["README.md", "agent.log", "tile_001.json"]).
  - `dirs`: a mock list of 2 directory names (e.g., ["fleet-knowledge/", "output/"]).
  - `prompt`: "Next command? (ls, cd, cat, submit)"
**Output:** Dictionary with keys `path`, `files`, `dirs`, `prompt`, or the error string.

Write only the three Python function definitions. No explanations, no extra code.
```

def deadband_filter(readings, p0_threshold=-10.0, p1_low=0.0, p1_high=5.0):
    """
    Apply Deadband Protocol P0/P1/P2 to a list of readings.
    """
    # P0: Map negative space
    for r in readings:
        if r <= p0_threshold:
            raise ValueError(f"P0 violation: reading in negative space (value: {r})")
    
    # P1: Find safe channel
    safe_readings = [r for r in readings if p1_low <= r <= p1_high]
    
    # P2: Optimize within channel (here, just return filtered list)
    return safe_readings

def validate_tile(tile):
    """
    Validate a training tile for PLATO room submission.
    """
    required_keys = {"content", "agent_id", "cycle", "tags"}
    if not all(key in tile for key in required_keys):
        return (False, "Invalid: missing required keys")
    
    # Check content
    if not isinstance(tile["content"], str) or len(tile["content"]) == 0:
        return (False, "Invalid: content must be non-empty string")
    if len(tile["content"]) > 10000:
        return (False, "Invalid: content exceeds 10000 characters")
    
    # Check agent_id
    if not (isinstance(tile["agent_id"], str) and tile["agent_id"].startswith("ZC-")):
        return (False, "Invalid: agent_id must be non-empty string starting with 'ZC-'")
    
    # Check cycle
    if not (isinstance(tile["cycle"], int) and tile["cycle"] > 0):
        return (False, "Invalid: cycle must be positive integer")
    
    # Check tags
    if not isinstance(tile["tags"], list):
        return (False, "Invalid: tags must be a list")
    for tag in tile["tags"]:
        if not isinstance(tag, str) or len(tag) > 20:
            return (False, f"Invalid: tag '{tag}' must be string length <= 20")
    
    return (True, "Tile accepted")

def walk_repo_shell(current_path):
    """
    Simulate Zeroclaw navigation in git-repo-as-shell.
    """
    if "/.git/" in current_path:
        return "ERROR: Attempted to enter git internals. Navigation blocked."
    
    # Mock file system
    base_name = current_path.rstrip("/").split("/")[-1] if "/" in current_path else current_path
    mock_files = [
        f"{base_name}_README.md",
        f"{base_name}_agent.log",
        f"{base_name}_tile_001.json"
    ]
    mock_dirs = [
        "fleet-knowledge/",
        "output/"
    ]
    
    return {
        "path": current_path,
        "files": mock_files,
        "dirs": mock_dirs,
        "prompt": "Next command? (ls, cd, cat, submit)"
    }
