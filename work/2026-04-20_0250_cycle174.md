## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.
P0: Map negative space (remove values below a lower threshold).
P1: Find safe channels (keep values within a safe band).
P2: Optimize within channels (apply a scaling factor to values in the safe band).
The function should take a list of floats, a lower threshold, an upper threshold, and a P2 scaling factor.
It should return a new list. Values below the lower threshold are removed (P0).
Values between lower and upper are kept and multiplied by the scaling factor (P1 -> P2).
Values above the upper threshold are considered "unsafe" and should be logged with a warning but kept unchanged.
Include a docstring and a simple example.
```

### **DeepSeek-chat Output:**
```python
import logging

logging.basicConfig(level=logging.WARNING)
logger = logging.getLogger(__name__)

def deadband_filter(values, lower_thresh, upper_thresh, p2_scale=1.0):
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of numeric values.

    P0 (Negative Space): Remove values below `lower_thresh`.
    P1 (Safe Channels): Keep values between `lower_thresh` and `upper_thresh`.
    P2 (Optimize): Scale values in the safe channel by `p2_scale`.

    Values above `upper_thresh` are logged as warnings but retained unchanged.

    Args:
        values (list[float]): Input list of values.
        lower_thresh (float): Lower threshold for P0 removal.
        upper_thresh (float): Upper threshold for P1 safe channel.
        p2_scale (float): Scaling factor for P2 optimization. Default 1.0.

    Returns:
        list[float]: Filtered and processed list.

    Example:
        >>> deadband_filter([-5, 2, 15, 25, 50], 0, 30, 2.0)
        WARNING: Value 50 exceeds upper threshold 30. Keeping unchanged.
        [4.0, 30.0, 50.0]  # 2*2, 15*2, 50 (warning)
    """
    result = []
    for v in values:
        if v < lower_thresh:
            continue  # P0: Remove
        elif v <= upper_thresh:
            result.append(v * p2_scale)  # P1 -> P2: Scale
        else:
            logger.warning(f"Value {v} exceeds upper threshold {upper_thresh}. Keeping unchanged.")
            result.append(v)  # Keep, but warn
    return result
```

    Parameters:
    - values: list of floats, the input data.
    - lower_threshold: float, values below this are removed (P0).
    - upper_threshold: float, values above this are considered unsafe.
    - scaling_factor: float, multiplier for values in safe channel (P2).

    Returns:
    - list of floats, processed according to P0/P1/P2.
    """
    filtered = []
    for value in values:
        if value < lower_threshold:
            # P0: map negative space (remove)
            continue
        elif lower_threshold <= value <= upper_threshold:
            # P1: safe channel, P2: optimize
            filtered.append(value * scaling_factor)
        else:
            # Above upper threshold: unsafe, keep but note
            filtered.append(value)
            print(f"Warning: Value {value} exceeds safe upper threshold {upper_threshold}")
    return filtered

*   **Fleet Context:** DeepSeek's docstring and comments explicitly reference "P0/P1/P2" and "Negative Space"/"Safe Channels", showing better absorption of fleet doctrine. Llama's docstring is more generic ("signal processing").
