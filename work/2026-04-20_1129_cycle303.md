**Prompt:**
```
Write a Python function called `validate_deadband_protocol`.
```

### **DeepSeek-chat Output:**
```python
def validate_deadband_protocol(actions):
    """
    Validate if a sequence of priority actions follows the Deadband Protocol.

    The Deadband Protocol requires strict ordering: all P0 must come before any P1,
    and all P1 before any P2. Skipping priorities (e.g., P0 -> P2) is invalid.

    Args:
        actions (list of str): List of priority strings ("P0", "P1", "P2").

    Returns:
        bool: True if sequence is valid, False otherwise.

    Examples:
        >>> validate_deadband_protocol(["P0", "P0", "P1", "P2"])
        True
        >>> validate_deadband_protocol(["P0", "P2", "P1"])
        False
        >>> validate_deadband_protocol(["P1", "P0"])
        False
    """
    # Define the valid order
    valid_order = {"P0": 0, "P1": 1, "P2": 2}
    
    # Track the highest priority seen so far
    max_seen = -1
    
    for action in actions:
        if action not in valid_order:
            # Invalid priority label
            return False
        
        current_priority = valid_order[action]
        
        # Check if we are moving backwards or skipping ahead
        if current_priority < max_seen:
            # Moving backwards (e.g., P1 then P0)
            return False
        if current_priority > max_seen + 1:
            # Skipping a priority level (e.g., P0 then P2)
            return False
        
        # Update the highest priority seen
        max_seen = current_priority
    
    return True
```
