### **Function 1: `deadband_filter`**

**Specification:**  
Implement a function `deadband_filter(decisions: List[float], deadband: float = 0.1) -> List[str]`.  
For each value in `decisions`:
- **P0 (Negative Space):** If value <= -deadband, return `"P0"`.
- **P1 (Safe Channel):** If -deadband < value < deadband, return `"P1"`.
- **P2 (Optimize):** If value >= deadband, return `"P2"`.

---

#### **DeepSeek-chat Implementation:**
```python
from typing import List

def deadband_filter(decisions: List[float], deadband: float = 0.1) -> List[str]:
    """
    Apply the fleet's Deadband Protocol to a list of decision values.

    The protocol categorizes each numeric decision into three priority bands:
    P0 (negative space): values <= -deadband (what NOT to do)
    P1 (safe channel): values between -deadband and deadband (where you CAN be)
    P2 (optimize): values >= deadband (best path within safe channels)

    Args:
        decisions: A list of floating-point decision values.
        deadband: The threshold for band classification. Must be non-negative.

    Returns:
        A list of strings, each being "P0", "P1", or "P2", corresponding to each input value.

    Raises:
        ValueError: If deadband is negative.
    """
    if deadband < 0:
        raise ValueError(f"Deadband must be non-negative, got {deadband}")

    results = []
    for val in decisions:
        if val <= -deadband:
            results.append("P0")
        elif val >= deadband:
            results.append("P2")
        else:  # -deadband < val < deadband
            results.append("P1")
    return results
```

### **Function 2: `parse_fleet_context`**

**Specification:**  
Implement a function `parse_fleet_context(markdown_text: str) -> dict`.  
Parse a markdown string in the FLEET-CONTEXT.md format and return a dictionary with keys: `agents`, `technologies`, `rules`. Each key should map to a list of strings extracted from the corresponding sections.

---

#### **DeepSeek-chat Implementation:**
```python
import re
from typing import Dict, List

def parse_fleet_context(markdown_text: str) -> Dict[str, List[str]]:
    """
    Extract structured data from a fleet context markdown document.

    Parses specific sections from a FLEET-CONTEXT.md formatted string:
    - "Who We Are" for agents
    - "Key Technologies" for technologies
    - "Rules" for rules

    Args:
        markdown_text: The full markdown text of the fleet context.

    Returns:
        A dictionary with keys 'agents', 'technologies', 'rules'.
        Each maps to a list of extracted items as strings.
        If a section is not found, its list will be empty.

    Note:
        This parser assumes the specific section headers used in FLEET-CONTEXT.md.
        It extracts list items following each header until the next header.
    """
    result = {
        "agents": [],
        "technologies": [],
        "rules": []
    }

    # Normalize line endings and split into lines
    lines = markdown_text.replace('\r\n', '\n').split('\n')
    current_section = None

    for line in lines:
        # Detect section headers
        if line.startswith("## Who We Are"):
            current_section = "agents"
            continue
        elif line.startswith("## Key Technologies"):
            current_section = "technologies"
            continue
        elif line.startswith("## Rules"):
            current_section = "rules"
            continue
        elif line.startswith("## "):  # Another section starts, stop capturing
            current_section = None
            continue

        # Capture list items (lines starting with '- ' or '**')
        if current_section:
            stripped = line.strip()
            if stripped.startswith("- "):
                item = stripped[2:].strip()
                if current_section == "agents":
                    result["agents"].append(item)
                elif current_section == "technologies":
                    # Remove markdown bold if present
                    item = re.sub(r'\*\*(.*?)\*\*', r'\1', item)
                    result["technologies"].append(item)
                elif current_section == "rules":
                    result["rules"].append(item)
            # Also capture bold items on their own line (for technologies)
            elif stripped.startswith("**") and stripped.endswith("**"):
                if current_section == "technologies":
                    item = stripped[2:-2].strip()
                    result["technologies"].append(item)

    return result
```
