### **Function 1: `validate_deadband_protocol`**

**Prompt:**
"Write a Python function called `validate_deadband_protocol`. The function takes one argument: `priority_sequence`, a list of strings. The strings can only be "P0", "P1", or "P2". The Deadband Protocol requires strict sequential priority: P0 tasks must come before P1, which must come before P2. The sequence cannot skip a priority (e.g., [P0, P2] is invalid). The sequence cannot go backwards (e.g., [P1, P0] is invalid). Repeating a priority is allowed but does not change the rule (e.g., [P0, P0, P1, P2] is valid). The function should return `True` if the sequence follows the protocol, `False` otherwise. Include type hints and a docstring."

**DeepSeek-chat Output:**
```python
from typing import List

def validate_deadband_protocol(priority_sequence: List[str]) -> bool:
    """
    Validates if a sequence of priority strings follows the Deadband Protocol.

    Rules:
    1. Only "P0", "P1", "P2" are allowed.
    2. Sequence must be non-decreasing: P0 -> P1 -> P2.
    3. Skipping a priority is invalid (e.g., P0 -> P2).
    4. Repeating a priority is allowed.

    Args:
        priority_sequence: List of priority strings.

    Returns:
        True if sequence follows protocol, False otherwise.
    """
    allowed = {"P0", "P1", "P2"}
    priority_map = {"P0": 0, "P1": 1, "P2": 2}

    # Check for invalid priorities
    if not all(p in allowed for p in priority_sequence):
        return False

    # Validate sequence
    last_seen = -1
    for priority in priority_sequence:
        current = priority_map[priority]
        if current < last_seen:
            return False  # Moving backwards
        if current > last_seen + 1:
            return False  # Skipping a priority
        last_seen = current

    return True
```
