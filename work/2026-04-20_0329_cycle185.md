### **Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements a deadband/programmable hysteresis filter.

Parameters:
- `data`: List of float input values.
- `lower`: Float, lower bound of deadband.
- `upper`: Float, upper bound of deadband. Assume lower <= upper.

Logic:
- Maintain an internal `last_output` state (start at 0).
- For each value in `data`:
  - If `lower <= value <= upper`: output `last_output` (hold last value).
  - Else: output the current `value` and update `last_output = value`.
- Return the list of filtered outputs.

Include a docstring and a simple example.
```

**DeepSeek-chat (simulated) output:**
```python
def deadband_filter(data, lower, upper):
    """
    Apply a deadband filter to a sequence of data points.

    Values within the inclusive deadband range [lower, upper] result in the
    previous output being repeated. Values outside the range pass through
    and become the new reference output.

    Args:
        data (list of float): Input values to filter.
        lower (float): Lower bound of deadband.
        upper (float): Upper bound of deadband.

    Returns:
        list of float: Filtered output sequence.
    """
    if lower > upper:
        raise ValueError("lower must be <= upper")

    output = []
    last_output = 0.0  # initial state

    for value in data:
        if lower <= value <= upper:
            output.append(last_output)
        else:
            output.append(value)
            last_output = value
    return output

# Example
```

```python
def deadband_filter(data, lower, upper):
    """
    Implements a deadband filter on a list of data.

    Parameters:
    data: list of floats - input data series
    lower: float - lower deadband threshold
    upper: float - upper deadband threshold

    Returns:
    list of floats - filtered data
    """
    filtered = []
    last_valid = 0.0

    for x in data:
        if x >= lower and x <= upper:
            # Within deadband, use last valid value
            filtered.append(last_valid)
        else:
            # Outside deadband, pass through
            filtered.append(x)
            last_valid = x
    return filtered
```

*   **Edge Cases:** DeepSeek explicitly initializes `last_output = 0.0`. Llama does the same. Neither handles the initial condition perfectly (what if 0 is not a valid neutral state?), but it meets the spec.

**DeepSeek-chat output:**

```python
import re

def parse_fleet_log_line(line):
    """
    Parse a structured fleet log line into a dictionary.

    Args:
        line (str): Log line in format "[timestamp] [LEVEL] key1=val1 ..."

    Returns:
        dict or None: Parsed data or None if malformed.
    """
    pattern = r'^\[([^\]]+)\] \[([A-Z]+)\] (.+)$'
    match = re.match(pattern, line.strip())
    if not match:
        return None

    timestamp, level, kv_pairs = match.groups()
    result = {'timestamp': timestamp, 'level': level}

    for pair in kv_pairs.split():
        if '=' not in pair:
            continue
        key, val = pair.split('=', 1)

        # Type conversion
        if val.lower() == 'true':
            val = True
        elif val.lower() == 'false':
            val = False
        else:
            try:
                if '.' in val:
                    val = float(val)
                else:
                    val = int(val)
            except ValueError:
                # Keep as string
                pass

        result[key] = val
    return result
```
