## **2. Function 1: `deadband_filter`**

**Prompt:** "Write a Python function called `deadband_filter` that takes a list of numeric values and three threshold tuples: `p0_range`, `p1_range`, `p2_range`. Each tuple is (min, max). Implement the Deadband Protocol: P0 (negative space) – values inside p0_range are rejected (return None). P1 (safe channels) – values inside p1_range are passed through unchanged. P2 (optimization) – values inside p2_range are normalized to 0-1 scale relative to p2_range. If a value falls into multiple bands, apply the highest priority (P0 > P1 > P2). Return a list of processed values (or None). Include a docstring and example usage."

### **DeepSeek-chat Output:**
```python
def deadband_filter(values, p0_range, p1_range, p2_range):
    """
    Apply Deadband Protocol filtering to a list of numeric values.
    
    Protocol priorities:
    P0 (negative space): values within p0_range -> None
    P1 (safe channels): values within p1_range -> pass through
    P2 (optimization): values within p2_range -> normalize to [0,1]
    
    Priority: P0 > P1 > P2 (if multiple bands apply)
    
    Args:
        values: list of numeric values
        p0_range: tuple (min, max) for P0 rejection band
        p1_range: tuple (min, max) for P1 pass-through band  
        p2_range: tuple (min, max) for P2 normalization band
        
    Returns:
        list: processed values (or None for P0 rejects)
    """
    p0_min, p0_max = p0_range
    p1_min, p1_max = p1_range
    p2_min, p2_max = p2_range
    
    result = []
    for val in values:
        # Check P0 (highest priority)
        if p0_min <= val <= p0_max:
            result.append(None)
            continue
            
        # Check P1
        if p1_min <= val <= p1_max:
            result.append(val)
            continue
            
        # Check P2
        if p2_min <= val <= p2_max:
            # Normalize to 0-1 within P2 range
            normalized = (val - p2_min) / (p2_max - p2_min)
            result.append(normalized)
            continue
            
        # Value outside all bands
        result.append(val)
    
    return result
```

```python
def deadband_filter(values, p0_range, p1_range, p2_range):
    """
    Implements the Deadband Protocol for numeric filtering.
    
    Parameters:
    values (list): Input numeric values.
    p0_range (tuple): (min, max) for P0 (reject).
    p1_range (tuple): (min, max) for P1 (pass-through).
    p2_range (tuple): (min, max) for P2 (normalize).
    
    Returns:
    list: Processed values with None for rejected.
    """
    p0_low, p0_high = p0_range
    p1_low, p1_high = p1_range
    p2_low, p2_high = p2_range
    
    output = []
    for v in values:
        if p0_low <= v <= p0_high:
            output.append(None)  # P0 reject
        elif p1_low <= v <= p1_high:
            output.append(v)     # P1 pass
        elif p2_low <= v <= p2_high:
            # Normalize to [0,1]
            norm_val = (v - p2_low) / (p2_high - p2_low)
            output.append(norm_val)
        else:
            output.append(v)     # outside bands
    return output
```
