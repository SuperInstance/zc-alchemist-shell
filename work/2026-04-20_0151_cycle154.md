## **2. Function Specifications**

### **Function 1: `deadband_filter`**
- **Input:** `values` (list of floats), `deadband` (float, default 0.1)
- **Output:** List of floats where any value within `±deadband` of zero is set to 0.0.
- **Example:** `deadband_filter([-0.05, 0.0, 0.15, -0.12], 0.1)` → `[0.0, 0.0, 0.15, -0.12]`

### **Function 2: `parse_fleet_log`**
- **Input:** `log_line` (string) in format `"TIMESTAMP | AGENT | ACTION | DETAILS"`
- **Output:** Dictionary with keys `timestamp`, `agent`, `action`, `details`.
- **Example:** `parse_fleet_log("2026-04-20T01:44:00Z | Alchemist | submit_tile | cycle_154_comparison")` → `{'timestamp': '2026-04-20T01:44:00Z', 'agent': 'Alchemist', 'action': 'submit_tile', 'details': 'cycle_154_comparison'}`

### **Function 3: `constraint_snap`**
- **Input:** `point` (tuple of two floats), `grid_size` (float, default 1.0), `offset` (tuple of two floats, default (0,0))
- **Output:** Tuple of two floats snapped to nearest grid point, with offset applied.
- **Example:** `constraint_snap((1.3, 2.7), 1.0, (0.5, 0.5))` → `(1.5, 2.5)`

## **3. Model Outputs**

### **DeepSeek-chat (simulated)**
```python
def deadband_filter(values, deadband=0.1):
    """
    Apply deadband filtering to a list of numeric values.
    Values within +/- deadband of zero are set to zero.
    """
    if not values:
        return []
    result = []
    for v in values:
        if abs(v) <= deadband:
            result.append(0.0)
        else:
            result.append(v)
    return result

def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line into a dictionary.
    Format: "TIMESTAMP | AGENT | ACTION | DETAILS"
    """
    if not log_line or '|' not in log_line:
        return {}
    parts = [part.strip() for part in log_line.split('|')]
    if len(parts) != 4:
        return {}
    return {
        'timestamp': parts[0],
        'agent': parts[1],
        'action': parts[2],
        'details': parts[3]
    }

def constraint_snap(point, grid_size=1.0, offset=(0,0)):
    """
    Snap a 2D point to the nearest grid point with offset.
    """
    try:
        x, y = point
        ox, oy = offset
        snapped_x = round((x - ox) / grid_size) * grid_size + ox
        snapped_y = round((y - oy) / grid_size) * grid_size + oy
        return (snapped_x, snapped_y)
    except (TypeError, ValueError):
        return point
```

### **Groq Llama-3.3-70b (simulated)**
```python
def deadband_filter(values, deadband=0.1):
    """
    Implements a deadband filter: values within deadband of zero become zero.
    """
    if values is None:
        return []
    filtered = []
    for val in values:
        if -deadband <= val <= deadband:
            filtered.append(0.0)
        else:
            filtered.append(val)
    return filtered

def parse_fleet_log(log_line):
    """
    Parses a fleet log line with format: TIMESTAMP | AGENT | ACTION | DETAILS
    Returns a dict with those keys, or empty dict on failure.
    """
    if not isinstance(log_line, str):
        return {}
    segments = log_line.split('|')
    if len(segments) != 4:
        return {}
    parsed = {
        'timestamp': segments[0].strip(),
        'agent': segments[1].strip(),
        'action': segments[2].strip(),
        'details': segments[3].strip()
    }
    return parsed

def constraint_snap(point, grid_size=1.0, offset=(0,0)):
    """
    Snaps a (x,y) point to the nearest grid point given grid_size and offset.
    """
    x, y = point
    ox, oy = offset
    # Snap to grid
    snapped_x = ox + round((x - ox) / grid_size) * grid_size
    snapped_y = oy + round((y - oy) / grid_size) * grid_size
    return (snapped_x, snapped_y)
```
