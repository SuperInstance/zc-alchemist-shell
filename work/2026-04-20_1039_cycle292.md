## **2. Function 1: deadband_filter**

**Prompt to both models:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input: 
- `values`: list of floats (e.g., sensor readings)
- `deadband`: float, the width of the deadband zone centered at zero

Protocol:
- P0 (negative space): Any value whose absolute value is less than `deadband/2` is mapped to 0.0 — this is the "dead zone".
- P1 (safe channels): Values outside the deadband are passed through unchanged.
- P2 (optimize): Not needed for this basic filter.

Return: a new list of floats with the deadband applied.

Include type hints, a docstring, and a simple example in the docstring.
```

### **DeepSeek‑chat output:**
```python
from typing import List

def deadband_filter(values: List[float], deadband: float) -> List[float]:
    """
    Apply the Deadband Protocol to a list of float values.

    The deadband is centered at zero. Any value with |value| < deadband/2
    is mapped to 0.0 (P0: negative space). Values outside the deadband
    are passed through unchanged (P1: safe channels).

    Args:
        values: List of input float values.
        deadband: Width of the deadband zone (must be >= 0).

    Returns:
        List of filtered float values.

    Example:
        >>> deadband_filter([-1.5, -0.2, 0.0, 0.3, 1.8], deadband=0.5)
        [-1.5, 0.0, 0.0, 0.0, 1.8]
        # deadband/2 = 0.25, so values in (-0.25, 0.25) become 0.0
    """
    if deadband < 0:
        raise ValueError("deadband must be non‑negative")
    
    threshold = deadband / 2.0
    result = []
    for v in values:
        if abs(v) < threshold:
            result.append(0.0)
        else:
            result.append(v)
    return result
```

### **DeepSeek‑chat output:**
```python
import re
from datetime import datetime
from typing import Dict, Any

def parse_fleet_log(log_line: str) -> Dict[str, Any]:
    """
    Parse a semi‑structured fleet log line into a structured dictionary.

    Expected format:
        "YYYY‑MM‑DD HH:MM:SS [LEVEL] key1=value1 key2=value2 ..."

    Args:
        log_line: A single log line string.

    Returns:
        Dictionary with keys:
            - 'timestamp': datetime object
            - 'level': string (e.g., 'INFO', 'WARNING')
            - plus all extracted key=value pairs as strings (values are kept as strings).

    Raises:
        ValueError if the line does not match the expected pattern.

    Example:
        >>> parse_fleet_log('2026‑04‑20 10:37:23 [WARNING] agent=Alchemist cycle=292')
        {
            'timestamp': datetime.datetime(2026, 4, 20, 10, 37, 23),
            'level': 'WARNING',
            'agent': 'Alchemist',
            'cycle': '292'
        }
    """
    pattern = r'(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) \[(\w+)\] (.*)'
    match = re.match(pattern, log_line)
    if not match:
        raise ValueError(f"Log line does not match expected format: {log_line}")
    
    timestamp_str, level, kv_part = match.groups()
    timestamp = datetime.strptime(timestamp_str, '%Y-%m-%d %H:%M:%S')
    
    result = {'timestamp': timestamp, 'level': level}
    
    # Parse key=value pairs
    kv_pattern = r'(\w+)=([^\s]+)'
    for key, value in re.findall(kv_pattern, kv_part):
        result[key] = value
    
    return result
```
