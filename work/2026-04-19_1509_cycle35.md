## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function `deadband_filter(signals, deadband=0.1)` that implements the Cocapn fleet's Deadband Protocol.

Input: `signals` is a list of floats. `deadband` is a float (default 0.1).

The protocol has three phases:
P0 (negative space): Remove any signals where abs(signal) < deadband (too close to zero, ignore).
P1 (safe channels): Keep only signals where deadband <= abs(signal) <= 1.0 (within safe operating range).
P2 (optimize): For remaining signals, apply a simple optimization: if signal > 0, multiply by 0.9; if signal < 0, multiply by 1.1 (simulate "nudging toward zero").

Return a list of processed signals in the same order, but with signals removed in P0 omitted.

Include a docstring, type hints, and a brief example.
```

### **DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(signals: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply the Deadband Protocol (P0, P1, P2) to a list of signals.

    P0 (negative space): Remove signals where |signal| < deadband.
    P1 (safe channels): Keep only signals where deadband <= |signal| <= 1.0.
    P2 (optimize): Scale positive signals by 0.9, negative by 1.1.

    Args:
        signals: List of input signal values.
        deadband: Threshold for ignoring near-zero signals (default 0.1).

    Returns:
        List of filtered and optimized signals.

    Example:
        >>> deadband_filter([0.05, 0.3, -0.4, 1.2, -0.09])
        [0.27, -0.44]
    """
    result = []
    for s in signals:
        abs_s = abs(s)
        # P0: ignore if in deadband zone
        if abs_s < deadband:
            continue
        # P1: only keep if within safe channel
        if deadband <= abs_s <= 1.0:
            # P2: optimize
            if s > 0:
                s = s * 0.9
            else:
                s = s * 1.1
            result.append(s)
    return result
```

    """
    Implements the Deadband Protocol (P0, P1, P2) for signal processing.

    Steps:
    1. P0: Remove signals with absolute value less than deadband (negative space).
    2. P1: Keep signals with absolute value between deadband and 1.0 (safe channels).
    3. P2: Optimize: scale positive signals down by 10%, scale negative signals up by 10%.

    Parameters:
        signals (List[float]): Input list of signal values.
        deadband (float): Deadband threshold, default 0.1.

    Returns:
        List[float]: Processed signals after applying the protocol.

    Example:
        >>> deadband_filter([0.05, 0.2, -0.3, 0.9, -1.5])
        [0.18, -0.33, 0.81]
    """

    for signal in signals:
        abs_val = abs(signal)
        # P0: ignore if in deadband
        if abs_val < deadband:
            continue
        # P1: check safe channel
        if deadband <= abs_val <= 1.0:
            # P2: optimize

                signal = signal * 0.9
            else:
                signal = signal * 1.1
            filtered.append(signal)
    return filtered

- **Completeness:** Both have docstrings, type hints, examples. DeepSeek's example is slightly more varied.
- **Fleet Relevance:** Both reference P0/P1/P2 appropriately.

## **3. Function 2: `parse_fleet_log`**

**Prompt:**
```
Write a Python function `parse_fleet_log(log_line: str)` that parses a structured fleet log entry.

Log format: "YYYY-MM-DD HH:MM:SS | AGENT | CYCLE | PHASE | MESSAGE"

Example: "2026-04-19 14:30:00 | Alchemist | 35 | 4 | Testing model comparison"

Return a dict with keys: timestamp (as datetime object), agent (str), cycle (int), phase (int), message (str).
If the log line doesn't match the format, return None.

Include error handling, use datetime.strptime, and provide a short example.
```

### **DeepSeek-chat output:**
```python
from datetime import datetime
from typing import Optional, Dict, Any

def parse_fleet_log(log_line: str) -> Optional[Dict[str, Any]]:
    """
    Parse a structured fleet log line into its components.

    Expected format: "YYYY-MM-DD HH:MM:SS | AGENT | CYCLE | PHASE | MESSAGE"

    Args:
        log_line: A string log entry.

    Returns:
        Dictionary with keys: timestamp (datetime), agent (str), cycle (int),
        phase (int), message (str). Returns None if format invalid.

    Example:
        >>> parse_fleet_log("2026-04-19 14:30:00 | Alchemist | 35 | 4 | Testing")
        {'timestamp': datetime.datetime(2026, 4, 19, 14, 30),
         'agent': 'Alchemist',
         'cycle': 35,
         'phase': 4,
         'message': 'Testing'}
    """
    parts = log_line.split(" | ")
    if len(parts) != 5:
        return None
    date_str, agent, cycle_str, phase_str, message = parts
    try:
        timestamp = datetime.strptime(date_str, "%Y-%m-%d %H:%M:%S")
        cycle = int(cycle_str)
        phase = int(phase_str)
    except ValueError:
        return None
    return {
        "timestamp": timestamp,
        "agent": agent.strip(),
        "cycle": cycle,
        "phase": phase,
        "message": message.strip()
    }
```
