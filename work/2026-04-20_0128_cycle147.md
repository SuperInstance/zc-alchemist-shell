Function name: deadband_filter
Description: Apply the fleet's Deadband Protocol to a list of numeric values. The protocol has three priority levels: P0 (negative space, values to exclude), P1 (safe channel, values to keep), P2 (optimize, values to prioritize). For simplicity, implement: if value < lower_bound → P0 (exclude), if lower_bound <= value <= upper_bound → P1 (keep), if value > upper_bound → P2 (keep and mark for optimization).

Input: values (list of float), lower_bound (float), upper_bound (float)
Output: dict with keys: 'p0_excluded': list, 'p1_safe': list, 'p2_optimize': list
Edge cases to consider: empty list, bounds where lower_bound > upper_bound, NaN/inf values.

Include docstring and type hints. Follow PEP 8.

**DeepSeek-chat output:**
```python
from typing import List, Dict
import math

def deadband_filter(values: List[float], lower_bound: float, upper_bound: float) -> Dict[str, List[float]]:
    """
    Apply the Deadband Protocol to filter numeric values into P0, P1, P2 categories.
    
    Args:
        values: List of numeric values to categorize.
        lower_bound: Values below this are P0 (excluded).
        upper_bound: Values above this are P2 (optimize); values between bounds are P1 (safe).
    
    Returns:
        Dictionary with keys:
        - 'p0_excluded': values < lower_bound
        - 'p1_safe': lower_bound <= value <= upper_bound  
        - 'p2_optimize': value > upper_bound
    
    Raises:
        ValueError: If lower_bound > upper_bound (invalid deadband).
    """
    if lower_bound > upper_bound:
        raise ValueError("lower_bound must be <= upper_bound for valid deadband")
    
    p0_excluded = []
    p1_safe = []
    p2_optimize = []
    
    for val in values:
        if math.isnan(val) or math.isinf(val):
            # Treat NaN/inf as P0 (exclude)
            p0_excluded.append(val)
            continue
            
        if val < lower_bound:
            p0_excluded.append(val)
        elif val > upper_bound:
            p2_optimize.append(val)
        else:
            p1_safe.append(val)
    
    return {
        'p0_excluded': p0_excluded,
        'p1_safe': p1_safe,
        'p2_optimize': p2_optimize
    }
```
