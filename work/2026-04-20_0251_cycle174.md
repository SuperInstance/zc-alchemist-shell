## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.
Input: `actions` (list of dicts). Each dict has keys: `"id"` (str), `"p0_violation"` (bool), `"p1_channel"` (str or None), `"p2_score"` (float).
Protocol:
- P0: Remove any action where `p0_violation` is True.
- P1: From remaining actions, group by `p1_channel`. Keep only the group with the most actions. If tie, pick first.
- P2: From the chosen P1 channel, select the action with the highest `p2_score`.
Return: The selected action dict, or None if no actions survive P0.
Include a clear docstring and type hints.
```

### **DeepSeek-chat Output:**
```python
from typing import List, Dict, Any, Optional
from collections import Counter

def deadband_filter(actions: List[Dict[str, Any]]) -> Optional[Dict[str, Any]]:
    """
    Apply the Deadband Protocol (P0->P1->P2) to filter actions.

    Steps:
    1. P0 (Negative Space): Remove actions with p0_violation=True.
    2. P1 (Safe Channels): Group remaining by p1_channel, select channel with most actions.
    3. P2 (Optimization): From selected channel, pick action with highest p2_score.

    Args:
        actions: List of action dicts with keys: id, p0_violation, p1_channel, p2_score.

    Returns:
        Selected action dict or None if no actions survive P0.
    """
    # P0: Remove violations
    safe_actions = [a for a in actions if not a.get("p0_violation", False)]
    if not safe_actions:
        return None

    # P1: Group by channel, find channel with max count
    channel_counts = Counter(a.get("p1_channel") for a in safe_actions)
    if not channel_counts:
        # No channels defined, treat all as same group
        target_channel = None
    else:
        # Get channel with highest count, break ties by first occurrence
        max_count = max(channel_counts.values())
        # Find first channel with that count (stable tie-break)
        for channel, count in channel_counts.items():
            if count == max_count:
                target_channel = channel
                break

    # Filter to target channel
    channel_actions = [a for a in safe_actions if a.get("p1_channel") == target_channel]

    # P2: Select max p2_score
    return max(channel_actions, key=lambda x: x.get("p2_score", -float("inf")))

from collections import defaultdict

def deadband_filter(actions: List[Dict[str, Any]]) -> Optional[Dict[str, Any]]:
    """
    Implements the Deadband Protocol (P0->P1->P2) for action selection.

    Parameters:
    -----------
    actions : List[Dict[str, Any]]
        Each dict must contain:
        - "id": str
        - "p0_violation": bool
        - "p1_channel": Optional[str]
        - "p2_score": float

    Returns:
    --------
    Optional[Dict[str, Any]]
        The selected action after applying P0, P1, and P2 filters.
        Returns None if all actions are eliminated by P0.
    """
    # P0: Eliminate actions with p0_violation == True
    filtered_actions = [action for action in actions if not action.get("p0_violation", True)]

    if not filtered_actions:
        return None

    # P1: Group by p1_channel and find the channel with the most actions
    channel_groups = defaultdict(list)
    for action in filtered_actions:
        channel = action.get("p1_channel")
        channel_groups[channel].append(action)

    # Find channel with maximum number of actions
    max_channel = None
    max_count = 0
    for channel, group in channel_groups.items():
        if len(group) > max_count:
            max_count = len(group)
            max_channel = channel

    # If all channels have 0 actions (should not happen after P0) or no channels
    if max_channel is None:
        # Fallback: treat all as same channel
        candidate_actions = filtered_actions
    else:
        candidate_actions = channel_groups[max_channel]

    # P2: Select action with highest p2_score from the chosen channel
    best_action = max(candidate_actions, key=lambda x: x.get("p2_score", float("-inf")))

    return best_action
```

### **Comparison:**
*   **Correctness:** Both implement P0→P1→P2 correctly. DeepSeek uses `Counter`, Groq uses `defaultdict`. Both handle `None` channels.
*   **Conciseness:** DeepSeek is slightly more compact (20 lines vs 25). Groq has more verbose comments.
*   **Edge Cases:** Groq explicitly handles the `max_channel is None` fallback. DeepSeek's `if not channel_counts` handles it implicitly. Both are robust.
*   **Tie-breaking:** DeepSeek correctly implements "first occurrence" tie-break. Groq does not specify tie-breaking (could be unstable).
*   **Readability:** Both are clear. Groq's parameter/return docstring is more formal.

**Verdict:** DeepSeek's implementation is more concise and correctly specifies tie-breaking. Groq's is more defensive with explicit fallback logic. Both are production-viable. **Slight edge to DeepSeek for adhering precisely to spec.**
