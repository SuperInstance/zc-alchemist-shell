## **2. Function 1: `validate_deadband_protocol`**

**Prompt:**  
```
Write a Python function `validate_deadband_protocol` that takes a list of action strings (like ["P0_check", "P1_explore", "P2_optimize"]) and returns True if the sequence respects strict P0→P1→P2 priority, False otherwise. Rules: P0 can appear anytime, P1 only after all P0s are done, P2 only after all P1s are done. Include docstring and example.
```

### **DeepSeek-chat output:**

```python
def validate_deadband_protocol(actions):
    """
    Validate that a sequence of actions follows Deadband Protocol priority.
    
    Args:
        actions: List of strings where each starts with 'P0_', 'P1_', or 'P2_'.
    
    Returns:
        bool: True if sequence respects P0→P1→P2 strict priority, False otherwise.
    
    Example:
        >>> validate_deadband_protocol(["P0_check", "P1_explore", "P2_optimize"])
        True
        >>> validate_deadband_protocol(["P1_explore", "P0_check"])
        False
    """
    seen_p1 = False
    seen_p2 = False
    
    for action in actions:
        if not action.startswith(('P0_', 'P1_', 'P2_')):
            raise ValueError(f"Invalid action prefix: {action}")
        
        if action.startswith('P1_'):
            seen_p1 = True
        elif action.startswith('P2_'):
            seen_p2 = True
        
        # Violation: P1 appears before P0 is done? Actually P0 can be anywhere.
        # Violation: P2 appears before P1 is done.
        if action.startswith('P2_') and not seen_p1:
            return False
        # P1 can appear anytime after P0s? Wait, spec says P1 only after all P0s.
        # Let's track P0 count.
    
    # Better implementation: track phase
    current_phase = 0
    for action in actions:
        if action.startswith('P0_'):
            if current_phase > 0:
                return False
        elif action.startswith('P1_'):
            if current_phase > 1:
                return False
            current_phase = max(current_phase, 1)
        elif action.startswith('P2_'):
            if current_phase > 2:
                return False
            current_phase = max(current_phase, 2)
        else:
            raise ValueError(f"Invalid action: {action}")
    
    return True
```

```python
    >>> validate_deadband_protocol(["P1_explore", "P0_scan"])
    False

    p1_done = False

                return False
```
