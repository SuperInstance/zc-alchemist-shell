### **Function 1: deadband_filter**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the fleet's Deadband Protocol (P0/P1/P2). 

Input: list of float sensor readings.
P0 (negative space): Remove values below -100.0 or above 100.0 (invalid range).
P1 (safe channels): Keep only values between -50.0 and 50.0 (operational range).
P2 (optimize): For remaining values, apply a simple moving average of window size 3 (if enough points).

Return: filtered list after P0, then P1, then P2 applied in strict priority order.

Include docstring, type hints, and edge case handling.
```

**DeepSeek-chat output:**
```python
from typing import List, Optional
import numpy as np

def deadband_filter(sensor_readings: List[float]) -> List[float]:
    """
    Apply Deadband Protocol (P0/P1/P2) to sensor readings.
    
    P0: Remove values outside [-100.0, 100.0] (invalid range).
    P1: Keep only values within [-50.0, 50.0] (operational range).
    P2: Apply moving average (window=3) to remaining values.
    
    Args:
        sensor_readings: List of raw sensor values.
    
    Returns:
        Filtered list after P0, P1, P2 applied in strict priority.
    """
    if not sensor_readings:
        return []
    
    # P0: Map negative space (remove invalid)
    p0_filtered = [x for x in sensor_readings if -100.0 <= x <= 100.0]
    
    # P1: Find safe channels (operational range)
    p1_filtered = [x for x in p0_filtered if -50.0 <= x <= 50.0]
    
    # P2: Optimize within channels (moving average)
    if len(p1_filtered) < 3:
        return p1_filtered  # Not enough points for moving average
    
    p2_filtered = []
    for i in range(len(p1_filtered)):
        if i < 2:
            # Not enough history, use raw value
            p2_filtered.append(p1_filtered[i])
        else:
            window = p1_filtered[i-2:i+1]
            avg = sum(window) / 3.0
            p2_filtered.append(avg)
    
    return p2_filtered
```

```python
from typing import List

def deadband_filter(readings: List[float]) -> List[float]:
    """Deadband protocol filter for sensor readings.
    
    Steps:
    1. P0: Remove readings outside [-100, 100].
    2. P1: Keep readings inside [-50, 50].
    3. P2: Apply moving average with window 3.
    
    Parameters:
        readings: list of float sensor readings.
        
    Returns:
        list of filtered readings.
    """
    # P0
    p0 = [r for r in readings if -100 <= r <= 100]
    
    # P1
    p1 = [r for r in p0 if -50 <= r <= 50]
    
    # P2
    if len(p1) < 3:
        return p1
    
    result = []
    for i in range(len(p1)):
        if i == 0:
            result.append(p1[0])
        elif i == 1:
            result.append((p1[0] + p1[1]) / 2)
        else:
            result.append((p1[i-2] + p1[i-1] + p1[i]) / 3)
    
    return result
```

### **Function 2: tile_validator**

**Prompt:**
```
Write a Python function `tile_validator` that validates a training tile dictionary against fleet schema.

Expected tile structure:
{
    "agent_id": str,
    "cycle": int,
    "content": str,
    "timestamp": str (ISO format),
    "room_id": str,
    "tags": list[str] (optional)
}

Validation rules:
- All required fields present
- agent_id starts with "zeroclaw-"
- cycle >= 0
- content length between 10 and 10000 chars
- timestamp valid ISO format (use datetime.fromisoformat)
- room_id starts with "room-"
- tags if present must be non-empty strings

Return: (bool, list[str]) where bool is validity, list contains error messages.
Include docstring and type hints.
```

**DeepSeek-chat output:**
```python
from typing import Dict, List, Tuple, Any
from datetime import datetime

def tile_validator(tile: Dict[str, Any]) -> Tuple[bool, List[str]]:
    """
    Validate a training tile against fleet schema.
    
    Args:
        tile: Dictionary containing tile data.
    
    Returns:
        Tuple of (is_valid, error_messages).
    """
    errors = []
    required_fields = ["agent_id", "cycle", "content", "timestamp", "room_id"]
    
    # Check required fields
    for field in required_fields:
        if field not in tile:
            errors.append(f"Missing required field: {field}")
    
    if errors:
        return (False, errors)
    
    # Validate agent_id
    if not isinstance(tile["agent_id"], str) or not tile["agent_id"].startswith("zeroclaw-"):
        errors.append("agent_id must be string starting with 'zeroclaw-'")
    
    # Validate cycle
    if not isinstance(tile["cycle"], int) or tile["cycle"] < 0:
        errors.append("cycle must be non-negative integer")
    
    # Validate content
    if not isinstance(tile["content"], str):
        errors.append("content must be string")
    elif len(tile["content"]) < 10 or len(tile["content"]) > 10000:
        errors.append("content length must be between 10 and 10000 characters")
    
    # Validate timestamp
    try:
        datetime.fromisoformat(tile["timestamp"])
    except (ValueError, TypeError):
        errors.append("timestamp must be valid ISO format string")
    
    # Validate room_id
    if not isinstance(tile["room_id"], str) or not tile["room_id"].startswith("room-"):
        errors.append("room_id must be string starting with 'room-'")
    
    # Validate tags if present
    if "tags" in tile:
        if not isinstance(tile["tags"], list):
            errors.append("tags must be list if present")
        else:
            for i, tag in enumerate(tile["tags"]):
                if not isinstance(tag, str):
                    errors.append(f"tag at index {i} must be string")
                elif not tag.strip():
                    errors.append(f"tag at index {i} must be non-empty")
    
    return (len(errors) == 0, errors)
```
