**Prompt:**
```
Write a Python function `deadband_filter(values, deadband=0.1)` that applies a deadband filter to a list of float values.

Deadband filter rules:
- If absolute value of element < deadband: set to 0.0
- Otherwise: keep original value
- Return new list of same length
- Include type hints and a brief docstring.
- Add a simple example in the docstring.
```

**DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(values: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply a deadband filter to a list of float values.
    
    Parameters:
    -----------
    values : List[float]
        Input list of floating-point numbers.
    deadband : float, optional
        Deadband threshold (default 0.1). Values with absolute magnitude
        below this threshold are set to zero.
    
    Returns:
    --------
    List[float]
        Filtered list where |value| < deadband → 0.0, else unchanged.
    
    Example:
    --------
    >>> deadband_filter([0.05, -0.2, 0.15, -0.09], deadband=0.1)
    [0.0, -0.2, 0.15, 0.0]
    """
    if not isinstance(values, list):
        raise TypeError("values must be a list")
    if not isinstance(deadband, (int, float)):
        raise TypeError("deadband must be numeric")
    if deadband < 0:
        raise ValueError("deadband must be non-negative")
    
    result = []
    for v in values:
        if abs(v) < deadband:
            result.append(0.0)
        else:
            result.append(v)
    return result
```

Write a Python function `parse_fleet_log(log_line: str)` that parses a fleet log line and returns a dictionary.

Log format: "YYYY-MM-DD HH:MM:SS [AGENT] [LEVEL] message"

**DeepSeek-chat output:**
```python
from datetime import datetime
from typing import Optional, Dict, Any
import re

def parse_fleet_log(log_line: str) -> Optional[Dict[str, Any]]:
    """
    Parse a fleet log line into structured components.
    
    Expected format: "YYYY-MM-DD HH:MM:SS [AGENT] [LEVEL] message"
    
    Parameters:
    -----------
    log_line : str
        Single line from fleet log.
    
    Returns:
    --------
    Optional[Dict[str, Any]]
        Dictionary with keys: timestamp (datetime), agent (str), level (str), message (str).
        Returns None if line doesn't match expected format.
    
    Example:
    --------
    >>> parse_fleet_log("2026-04-20 14:30:22 [Alchemist] [INFO] Submitted tile 347")
    {'timestamp': datetime.datetime(2026, 4, 20, 14, 30, 22),
     'agent': 'Alchemist',
     'level': 'INFO',
     'message': 'Submitted tile 347'}
    """
    pattern = r'^(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) \[([^\]]+)\] \[([^\]]+)\] (.+)$'
    match = re.match(pattern, log_line.strip())
    
    if not match:
        return None
    
    date_str, agent, level, message = match.groups()
    
    try:
        timestamp = datetime.strptime(date_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        return None
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "level": level,
        "message": message.strip()
    }
```
