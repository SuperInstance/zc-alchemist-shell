## **2. Function 1: deadband_filter()**

**Prompt:**
```
Write a Python function `deadband_filter(readings: List[float], deadband: float) -> List[float]` that applies the fleet's Deadband Protocol to sensor readings.

Context: The Deadband Protocol has three priority levels:
- P0: Identify "negative space" – readings that are invalid (NaN, infinite) or out of safe range (-100.0 to 100.0). Replace these with None.
- P1: Find "safe channels" – readings within deadband of zero (-deadband to +deadband). Set these to 0.0.
- P2: "Optimize within channels" – for valid readings outside deadband, return them unchanged.

Strict priority: P0 → P1 → P2. Never skip P0 or P1.

Input: List of float readings, deadband threshold (positive float).
Output: List of filtered floats (or None for invalid).
Include docstring and example.
```

### **DeepSeek-chat output:**
```python
from typing import List, Optional

def deadband_filter(readings: List[float], deadband: float) -> List[Optional[float]]:
    """
    Apply the fleet's Deadband Protocol to sensor readings.
    
    P0 (negative space): Replace invalid readings (NaN, inf, out of range [-100, 100]) with None.
    P1 (safe channels): Readings within deadband of zero (-deadband to +deadband) become 0.0.
    P2 (optimize): Valid readings outside deadband remain unchanged.
    
    Strict priority: P0 → P1 → P2.
    
    Args:
        readings: List of raw sensor readings.
        deadband: Positive float threshold around zero for P1 filtering.
    
    Returns:
        List of filtered values (float or None).
    
    Example:
        >>> deadband_filter([1.5, -0.5, 150.0, float('nan'), 0.2], 0.3)
        [1.5, 0.0, None, None, 0.0]
    """
    SAFE_MIN, SAFE_MAX = -100.0, 100.0
    filtered = []
    
    for val in readings:
        # P0: Map negative space (invalid readings)
        if not isinstance(val, (int, float)):
            filtered.append(None)
            continue
        if val != val:  # NaN check
            filtered.append(None)
            continue
        if val == float('inf') or val == float('-inf'):
            filtered.append(None)
            continue
        if val < SAFE_MIN or val > SAFE_MAX:
            filtered.append(None)
            continue
        
        # P1: Find safe channels (within deadband of zero)
        if -deadband <= val <= deadband:
            filtered.append(0.0)
            continue
        
        # P2: Optimize within channels (keep valid reading)
        filtered.append(val)
    
    return filtered
```
