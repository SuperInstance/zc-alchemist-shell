### **Function 1: deadband_filter**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input: `values` (list of floats), `p0_threshold` (float), `p1_min` (float), `p1_max` (float)
Process:
- P0 (negative space): Remove any values where abs(value) < p0_threshold (too small to act).
- P1 (safe channels): Keep only values where p1_min <= value <= p1_max.
- P2 (optimize): Return the value closest to the midpoint of [p1_min, p1_max].

Return: Filtered value (float) or None if no values survive P0+P1.
Include a brief docstring.
```

**DeepSeek-chat output:**
```python
def deadband_filter(values, p0_threshold, p1_min, p1_max):
    """
    Apply Deadband Protocol (P0/P1/P2) to filter and select a value.

    P0: Remove values whose absolute value is below p0_threshold (negative space).
    P1: Keep only values within the safe channel [p1_min, p1_max].
    P2: From remaining values, pick the one closest to the channel midpoint.

    Args:
        values (list[float]): Input decision values.
        p0_threshold (float): Minimum absolute value to pass P0.
        p1_min (float): Lower bound of safe channel.
        p1_max (float): Upper bound of safe channel.

    Returns:
        float or None: Selected value, or None if no values survive.
    """
    # P0: filter out values in negative space
    p0_candidates = [v for v in values if abs(v) >= p0_threshold]
    if not p0_candidates:
        return None

    # P1: keep only values in safe channel
    p1_candidates = [v for v in p0_candidates if p1_min <= v <= p1_max]
    if not p1_candidates:
        return None

    # P2: optimize — choose value closest to channel midpoint
    midpoint = (p1_min + p1_max) / 2
    selected = min(p1_candidates, key=lambda v: abs(v - midpoint))
    return selected
```

def deadband_filter(values, p0_threshold, p1_min, p1_max):
    """
    Implements the Deadband Protocol (P0/P1/P2) for value selection.

    Steps:
    1. P0 (Negative Space): Discard values with absolute value below p0_threshold.
    2. P1 (Safe Channel): Keep values within [p1_min, p1_max].
    3. P2 (Optimize): Select the value closest to the midpoint of the safe channel.

    Parameters:
        values (list of float): Input values.
        p0_threshold (float): Threshold for P0 filtering.
        p1_min (float): Minimum of safe channel.
        p1_max (float): Maximum of safe channel.

    Returns:
        float or None: The selected value, or None if no value passes filters.
    """
    # P0: Remove values in negative space
    after_p0 = [v for v in values if abs(v) >= p0_threshold]
    if not after_p0:
        return None

    # P1: Keep values in safe channel
    after_p1 = [v for v in after_p0 if p1_min <= v <= p1_max]
    if not after_p1:
        return None

    # P2: Optimize: choose closest to midpoint
    midpoint = (p1_min + p1_max) / 2.0
    best_value = min(after_p1, key=lambda x: abs(x - midpoint))
    return best_value

### **Function 2: parse_fleet_context**

**Prompt:**
```
Write a Python function `parse_fleet_context` that parses a markdown fleet-context string (like FLEET-CONTEXT.md) into a structured dictionary.

Input: `text` (str) - markdown content
Extract:
- "senior_agents": list of senior agent names (from "Three senior agents: ...")
- "doctrine": dict with keys "p0", "p1", "p2" descriptions
- "key_technologies": list of technology names with bold markers
- "rules": list of rule strings

Assume sections are marked with ## headings. Return dict with those keys; missing sections become empty lists/dict.
Include error handling for missing patterns.
```

**DeepSeek-chat output:**
```python
import re

def parse_fleet_context(text):
    """
    Parse a fleet-context markdown string into structured data.

    Args:
        text (str): Markdown content similar to FLEET-CONTEXT.md

    Returns:
        dict: With keys:
            - senior_agents: list[str]
            - doctrine: dict with p0, p1, p2
            - key_technologies: list[str]
            - rules: list[str]
    """
    result = {
        "senior_agents": [],
        "doctrine": {"p0": "", "p1": "", "p2": ""},
        "key_technologies": [],
        "rules": []
    }

    # Extract senior agents
    agent_match = re.search(r"Three senior agents:\s*(.+?)(?:\n|$)", text, re.IGNORECASE)
    if agent_match:
        agents_text = agent_match.group(1)
        # Simple split by commas, remove markers like parentheses
        agents = [a.strip().rstrip('.)') for a in re.split(r'[,;]', agents_text)]
        result["senior_agents"] = [a for a in agents if a]

    # Extract doctrine lines
    doctrine_section = re.search(r"Fleet Doctrine.*?\n```(.*?)```", text, re.DOTALL | re.IGNORECASE)
    if doctrine_section:
        doctrine_text = doctrine_section.group(1)
        lines = doctrine_text.strip().split('\n')
        for line in lines:
            line = line.strip()
            if line.startswith('P0:'):
                result["doctrine"]["p0"] = line[3:].strip()
            elif line.startswith('P1:'):
                result["doctrine"]["p1"] = line[3:].strip()
            elif line.startswith('P2:'):
                result["doctrine"]["p2"] = line[3:].strip()

    # Extract key technologies (bold items)
    tech_matches = re.findall(r"\*\*([^*]+)\*\*", text)
    # Filter to likely technologies (after "Key Technologies" section)
    # Simple heuristic: take first 7 bold items after that heading
    result["key_technologies"] = tech_matches[:7] if tech_matches else []

    # Extract rules (bullet points after "Rules" heading)
    rules_section = re.search(r"## Rules\s*\n(.*?)(?=##|\Z)", text, re.DOTALL | re.IGNORECASE)
    if rules_section:
        rules_text = rules_section.group(1)
        # Capture lines starting with dash or asterisk
        rule_lines = re.findall(r"^\s*[-*]\s*(.+)$", rules_text, re.MULTILINE)
        result["rules"] = [r.strip() for r in rule_lines if r.strip()]

    return result
```
