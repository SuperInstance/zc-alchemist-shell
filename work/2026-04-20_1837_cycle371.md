### **Function 1: Deadband Filter**

**Prompt to both models:**
```
Write a Python function `apply_deadband_protocol` that takes a list of float sensor readings and returns a filtered list according to the Cocapn fleet's Deadband Protocol.

The Deadband Protocol has three priority levels:
P0: Map negative space (identify readings that are invalid or dangerous, e.g., NaN, infinity, or beyond physical limits [-100.0, 100.0]).
P1: Find safe channels (readings within valid range but outside the "normal" band [-10.0, 10.0] are marked as warnings).
P2: Optimize within channels (normal readings are passed through unchanged).

Return a list of tuples: (original_value, status), where status is one of: "P0_REJECTED", "P1_WARNING", "P2_NORMAL".

Include type hints, a docstring, and example usage.
```

**DeepSeek-chat output:**
```python
from typing import List, Tuple

def apply_deadband_protocol(readings: List[float]) -> List[Tuple[float, str]]:
    """
    Apply the Cocapn fleet Deadband Protocol to sensor readings.
    
    P0: Reject invalid/dangerous readings (NaN, infinity, outside [-100.0, 100.0]).
    P1: Warn for readings in valid range but outside normal band [-10.0, 10.0].
    P2: Pass normal readings unchanged.
    
    Args:
        readings: List of float sensor readings.
    
    Returns:
        List of tuples (original_value, status) where status is one of:
        "P0_REJECTED", "P1_WARNING", "P2_NORMAL".
    """
    result = []
    for val in readings:
        # P0: Negative space
        if not isinstance(val, (int, float)) or not float('-inf') < val < float('inf'):
            result.append((val, "P0_REJECTED"))
            continue
        if val < -100.0 or val > 100.0:
            result.append((val, "P0_REJECTED"))
            continue
        
        # P1: Safe channels
        if val < -10.0 or val > 10.0:
            result.append((val, "P1_WARNING"))
            continue
        
        # P2: Optimize within channels
        result.append((val, "P2_NORMAL"))
    
    return result
```

```python
    Protocol:
    - P0 (Negative Space): Identify and reject invalid/dangerous readings.
    - P1 (Safe Channels): Identify readings outside normal operational range but within physical limits.
    - P2 (Optimization): Readings within normal range are passed through.
    
    Parameters:
    sensor_readings: A list of floating-point sensor readings.
    
    Returns:
    A list of tuples, each containing (original_reading, status_string).
    Status strings: "P0_REJECTED", "P1_WARNING", or "P2_NORMAL".
    """
    filtered_results = []

    PHYSICAL_MAX = 100.0

        # P0: Check physical limits
        if reading < PHYSICAL_MIN or reading > PHYSICAL_MAX:
            filtered_results.append((reading, "P0_REJECTED"))
            continue
            
        # P1: Check if outside normal range but within physical limits
        if reading < NORMAL_MIN or reading > NORMAL_MAX:
            filtered_results.append((reading, "P1_WARNING"))
            continue
            
        # P2: Normal reading
        filtered_results.append((reading, "P2_NORMAL"))
    
    return filtered_results
```
