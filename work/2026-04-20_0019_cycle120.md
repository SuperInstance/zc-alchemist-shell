### **Function 1: `clean_and_normalize_text`**

**Prompt:** "Write a Python function `clean_and_normalize_text(text: str) -> str` that takes a raw text string and returns a cleaned version. Cleaning includes: converting to lowercase, removing extra whitespace, removing all punctuation except apostrophes in contractions, and normalizing Unicode characters (NFKD normalization)."

**DeepSeek-chat Implementation:**
```python
import re
import unicodedata

def clean_and_normalize_text(text: str) -> str:
    """
    Clean and normalize a raw text string.
    
    Steps:
    1. Normalize Unicode to NFKD form (compatibility decomposition)
    2. Convert to lowercase
    3. Remove all punctuation except apostrophes in contractions
    4. Collapse multiple whitespace characters to a single space
    5. Strip leading/trailing whitespace
    
    Args:
        text: Raw input string
        
    Returns:
        Cleaned and normalized string
    """
    if not isinstance(text, str):
        raise TypeError(f"Expected string, got {type(text).__name__}")
    
    # Normalize Unicode (NFKD decomposes characters like é to e + combining accent)
    normalized = unicodedata.normalize('NFKD', text)
    
    # Convert to lowercase
    lowercased = normalized.lower()
    
    # Remove punctuation except apostrophes
    # Keep apostrophes only when surrounded by letters (contractions)
    cleaned = re.sub(r"(?<!\w)'|'(?!\w)", "", lowercased)  # Remove lone apostrophes
    cleaned = re.sub(r"[^\w\s']", "", cleaned)  # Remove other punctuation
    
    # Collapse whitespace
    cleaned = re.sub(r"\s+", " ", cleaned)
    
    return cleaned.strip()

### **Function 2: `find_longest_palindromic_substring`**

**Prompt:** "Write an efficient Python function `find_longest_palindromic_substring(s: str) -> str` that returns the longest palindromic substring within `s`. Handle empty strings and single characters. Include time complexity analysis."

**DeepSeek-chat Implementation:**
```python
def find_longest_palindromic_substring(s: str) -> str:
    """
    Find the longest palindromic substring using Manacher's algorithm.
    
    Time Complexity: O(n) where n is length of string
    Space Complexity: O(n)
    
    Args:
        s: Input string
        
    Returns:
        Longest palindromic substring. Returns empty string for empty input.
    """
    if not s:
        return ""
    
    # Transform string with boundaries to handle even-length palindromes
    # e.g., "abc" -> "^#a#b#c#$"
    transformed = '^#' + '#'.join(s) + '#$'
    n = len(transformed)
    p = [0] * n  # Palindrome radii
    center = right = 0
    
    for i in range(1, n - 1):
        # Mirror index
        mirror = 2 * center - i
        
        # Use previously computed radius if within bounds
        if i < right:
            p[i] = min(right - i, p[mirror])
        
        # Attempt to expand palindrome centered at i
        while transformed[i + p[i] + 1] == transformed[i - p[i] - 1]:
            p[i] += 1
        
        # Update center and right boundary if expanded beyond
        if i + p[i] > right:
            center = i
            right = i + p[i]
    
    # Find maximum radius and its center
    max_radius = max(p)
    center_index = p.index(max_radius)
    
    # Extract original substring
    start = (center_index - max_radius) // 2
    return s[start:start + max_radius]

def find_longest_palindromic_substring_simple(s: str) -> str:
    """
    Simpler expand-around-center approach.
    Good for clarity, less efficient for very long strings.
    """
    if not s:
        return ""

        while left >= 0 and right < len(s) and s[left] == s[right]:
            left -= 1
            right += 1
        return s[left + 1:right]

        # Even length palindromes
        even_pal = expand_around_center(i, i + 1)
        if len(even_pal) > len(longest):
            longest = even_pal
