### **Function 1: deadband_filter**

**Prompt:**  
```
Write a Python function `deadband_filter(data, lower, upper)` that implements a deadband filter.
- Input: `data` (list of floats), `lower`, `upper` (float thresholds, lower < upper).
- Logic: For each value in data:
  - If value < lower, map to lower.
  - If value > upper, map to upper.
  - Else, keep value unchanged.
- Return: filtered list.
Include a docstring and a simple example in comments.
```

**DeepSeek-chat output:**
```python
def deadband_filter(data, lower, upper):
    """
    Apply a deadband filter to a list of numeric data.

    Parameters:
    data (list of float): Input data values.
    lower (float): Lower threshold.
    upper (float): Upper threshold.

    Returns:
    list of float: Filtered data where values below lower are set to lower,
                   values above upper are set to upper, others unchanged.

    Example:
    >>> deadband_filter([1.0, 5.0, 10.0, 15.0, 20.0], 5.0, 15.0)
    [5.0, 5.0, 10.0, 15.0, 15.0]
    """
    if lower >= upper:
        raise ValueError("lower must be less than upper")
    
    filtered = []
    for value in data:
        if value < lower:
            filtered.append(lower)
        elif value > upper:
            filtered.append(upper)
        else:
            filtered.append(value)
    return filtered
```

```python
    """
```

```python
import datetime

def parse_fleet_log(log_text):
    """
    Parses a fleet log string in the format "TIMESTAMP | AGENT | STATUS | MESSAGE".

    Args:
        log_text (str): A single line of fleet log.

    Returns:
        dict: A dictionary with keys 'timestamp', 'agent', 'status', 'message'.
              Returns None if the log line is malformed.

    Example:
        >>> log = "2026-04-20 14:30:15 | Alchemist | ACTIVE | Testing model outputs"
        >>> parse_fleet_log(log)
        {'timestamp': datetime.datetime(2026, 4, 20, 14, 30, 15),
         'agent': 'Alchemist',
         'status': 'ACTIVE',
         'message': 'Testing model outputs'}
    """
    if not log_text or ' | ' not in log_text:
        return None
    
    parts = log_text.split(' | ', 3)  # Split into max 4 parts
    if len(parts) != 4:
        return None
    
    timestamp_str, agent, status, message = parts
    try:
        timestamp = datetime.datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        return None
    
    return {
        'timestamp': timestamp,
        'agent': agent.strip(),
        'status': status.strip(),
        'message': message.strip()
    }
```
