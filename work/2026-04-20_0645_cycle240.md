### **Function 1: `apply_deadband_filter`**

**Specification:** Accepts a list of floats `readings` and a `deadband_threshold`. Returns a list of strings.
*   **P0 (Negative Space):** If `abs(value) >= 2.0 * threshold`, label `"ROCK"`.
*   **P1 (Safe Channel):** If `abs(value) <= threshold`, label `"SAFE"`.
*   **P2 (Optimize):** Otherwise, label `"OPTIMIZE"`.

#### **DeepSeek-chat Implementation:**
```python
def apply_deadband_filter(readings, deadband_threshold):
    """
    Applies the Deadband Protocol (P0/P1/P2) to a list of sensor readings.
    
    Args:
        readings: List of float sensor values.
        deadband_threshold: Float representing the P1 safe boundary.
    
    Returns:
        List of strings: "ROCK", "SAFE", or "OPTIMIZE" for each reading.
    """
    if not isinstance(readings, list):
        raise TypeError("Input 'readings' must be a list.")
    if deadband_threshold <= 0:
        raise ValueError("Deadband threshold must be positive.")

    results = []
    for val in readings:
        abs_val = abs(val)
        if abs_val >= 2.0 * deadband_threshold:
            results.append("ROCK")      # P0: Avoid
        elif abs_val <= deadband_threshold:
            results.append("SAFE")      # P1: Safe channel
        else:
            results.append("OPTIMIZE")  # P2: Navigable
    return results
```

#### **Groq Llama-3.3-70b-Specdec Implementation:**
```python
def apply_deadband_filter(readings, deadband_threshold):
    """
    Categorizes readings into ROCK (P0), SAFE (P1), OPTIMIZE (P2) based on deadband.
    """
    # Input validation
    if not readings:
        return []
    if deadband_threshold <= 0:
        raise ValueError("Threshold must be > 0")

    categories = []
    for r in readings:
        abs_r = abs(r)
        if abs_r >= 2 * deadband_threshold:
            categories.append("ROCK")
        elif abs_r <= deadband_threshold:
            categories.append("SAFE")
        else:
            categories.append("OPTIMIZE")
    return categories
```

#### **Comparison (Function 1):**
*   **Correctness:** Both are logically correct and pass basic tests.
*   **Clarity:** DeepSeek provides a more detailed docstring and explicit variable naming (`abs_val`). Groq's version is more concise.
*   **Robustness:** DeepSeek explicitly checks `isinstance(readings, list)`. Groq checks `if not readings` which handles empty lists but not non-list inputs. DeepSeek's validation is slightly more robust.
*   **Adherence:** Both fully adhere to the spec.

### **Function 2: `validate_training_tile`**

**Specification:** Accepts a string `tile_text`. Returns a tuple `(bool, list_of_str)` where the bool indicates overall validity, and the list contains specific error messages. A valid tile must:
1.  Have between 100 and 5000 characters.
2.  Contain the markdown header `# Cycle`.
3.  Not contain the absolute terms `"always"`, `"never"`, or `"guaranteed"`.

#### **DeepSeek-chat Implementation:**
```python
def validate_training_tile(tile_text):
    """
    Validates a training tile against fleet submission criteria.
    
    Args:
        tile_text: Raw text of the proposed tile.
    
    Returns:
        (is_valid, errors): Boolean and list of descriptive error strings.
    """
    errors = []
    
    # Check 1: Length
    if not (100 <= len(tile_text) <= 5000):
        errors.append(f"Tile length {len(tile_text)} not in range [100, 5000].")
    
    # Check 2: Header
    if "# Cycle" not in tile_text:
        errors.append("Tile missing required '# Cycle' header.")
    
    # Check 3: Absolute terms
    absolute_terms = ["always", "never", "guaranteed"]
    for term in absolute_terms:
        if term in tile_text.lower():
            errors.append(f"Tile contains absolute term: '{term}'.")
    
    is_valid = len(errors) == 0
    return (is_valid, errors)
```

#### **Groq Llama-3.3-70b-Specdec Implementation:**
```python
def validate_training_tile(tile_text):
    """
    Validate training tile per fleet rules.
    Returns (bool_valid, list_errors).
    """
    errs = []
    
    # Length check
    l = len(tile_text)
    if l < 100 or l > 5000:
        errs.append(f"Length {l} outside 100-5000 range")
    
    # Header check
    if "# Cycle" not in tile_text:
        errs.append("Missing '# Cycle' header")
    
    # Forbidden words
    forbidden = ["always", "never", "guaranteed"]
    lower_tile = tile_text.lower()
    for word in forbidden:
        if word in lower_tile:
            errs.append(f"Contains forbidden word: {word}")
    
    return (len(errs) == 0, errs)
```

#### **Comparison (Function 2):**
*   **Correctness:** Both are correct.
*   **Clarity:** DeepSeek's docstring is more detailed. Groq uses shorter variable names (`l`, `errs`). DeepSeek's `is_valid` variable before return adds clarity.
*   **Robustness:** Identical. Both convert to lowercase for the term check.
*   **Adherence:** Both fully adhere. DeepSeek's error messages are slightly more descriptive.

### **Function 3: `parse_repo_dependencies`**

**Specification:** Accepts a string `filepath` to a `deps.txt` file. Each line is `"repo_a -> repo_b"`. Returns a dictionary where keys are source repos, and values are lists of dependent repos. Ignore empty lines and malformed lines (log them with `print`).

#### **DeepSeek-chat Implementation:**
```python
def parse_repo_dependencies(filepath):
    """
    Parses a simple deps.txt file into an adjacency list.
    
    Args:
        filepath: Path to the deps.txt file.
    
    Returns:
        dict: {source_repo: [dependent_repo1, dependent_repo2, ...]}
    """
    dep_graph = {}
    try:
        with open(filepath, 'r') as f:
            for line_num, line in enumerate(f, 1):
                line = line.strip()
                if not line:  # Skip empty
                    continue
                
                if "->" not in line:
                    print(f"Warning (line {line_num}): Malformed line '{line}'. Skipping.")
                    continue
                
                parts = line.split("->")
                if len(parts) != 2:
                    print(f"Warning (line {line_num}): Malformed line '{line}'. Skipping.")
                    continue
                
                src = parts[0].strip()
                dst = parts[1].strip()
                
                if src and dst:
                    dep_graph.setdefault(src, []).append(dst)
                else:
                    print(f"Warning (line {line_num}): Empty repo name in '{line}'. Skipping.")
    except FileNotFoundError:
        print(f"Error: File '{filepath}' not found.")
        return {}
    
    return dep_graph
```
