## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

The function should take two arguments:
1. `signals` (list of floats): A list of raw signal values.
2. `deadband_width` (float, default=0.1): The width of the deadband zone.

The Deadband Protocol has three priority levels:
- P0 (Negative Space): If a signal's absolute value is within +/- `deadband_width` of zero, it's considered "in the deadband" and should be mapped to 0.0.
- P1 (Safe Channels): If a signal is outside the deadband, it's in a "safe channel". Its value should be adjusted to the nearest boundary of the deadband.
  - Example: If `deadband_width=0.1`, a signal of 0.15 is adjusted to 0.1. A signal of -0.25 is adjusted to -0.1.
- P2 (Optimize): This function performs the P1 mapping. A future optimization (P2) might be applied elsewhere.

Return a new list of floats with the filtered/adjusted signals.

Include a detailed docstring and type hints. Add 2-3 inline comments explaining the logic.
```

### **DeepSeek-chat Output:**
```python
from typing import List

def deadband_filter(signals: List[float], deadband_width: float = 0.1) -> List[float]:
    """
    Apply the Deadband Protocol (P0/P1) to a list of signal values.

    P0 (Negative Space): Values within +/- deadband_width of zero are mapped to 0.0.
    P1 (Safe Channels): Values outside the deadband are adjusted to the nearest deadband boundary.

    Args:
        signals: List of raw signal values.
        deadband_width: Width of the deadband zone around zero. Default 0.1.

    Returns:
        List of filtered signal values.
    """
    filtered = []
    for s in signals:
        # P0: Check if signal is within the deadband (negative space)
        if abs(s) <= deadband_width:
            filtered.append(0.0)
        else:
            # P1: Adjust to the nearest deadband boundary
            if s > 0:
                adjusted = deadband_width
            else:  # s < 0
                adjusted = -deadband_width
            filtered.append(adjusted)
    return filtered
```

    """
    Implements the Deadband Protocol for signal processing.

    Parameters:
    signals (List[float]): A list of raw signal values.
    deadband_width (float): The width of the deadband zone. Default is 0.1.

    Returns:
    List[float]: A list of filtered signals where:
        - Signals within [-deadband_width, deadband_width] are set to 0.0 (P0).
        - Signals outside this range are clipped to the nearest boundary (P1).

            result.append(0.0)

            result.append(deadband_width)
        else:  # signal < -deadband_width
            # P1: Negative safe channel, clip to upper bound
            result.append(-deadband_width)
    return result
