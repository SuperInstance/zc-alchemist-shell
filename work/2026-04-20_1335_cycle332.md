### **Function 1: deadband_filter**

**Specification:**
- *Description:* Given a list of numeric readings, apply the Deadband Protocol: P0 (negative space) – remove values below a lower threshold; P1 (safe channel) – keep values within a deadband range; P2 (optimize) – return the median of the remaining values.
- *Input:* `readings` (list of floats), `lower_threshold` (float), `upper_threshold` (float).
- *Output:* A single float (median of filtered readings) or `None` if no readings survive filtering.
- *Edge cases:* Empty list, all readings below lower threshold, single reading within deadband.

#### **DeepSeek‑chat output:**
```python
def deadband_filter(readings, lower_threshold, upper_threshold):
    """
    Apply Deadband Protocol to a list of readings.
    P0: Remove values below lower_threshold.
    P1: Keep values within [lower_threshold, upper_threshold].
    P2: Return median of surviving values.
    """
    if not readings:
        return None
    
    # P0: filter out values below lower threshold
    filtered = [r for r in readings if r >= lower_threshold]
    if not filtered:
        return None
    
    # P1: further filter to deadband range
    in_deadband = [r for r in filtered if r <= upper_threshold]
    if not in_deadband:
        return None
    
    # P2: compute median
    sorted_vals = sorted(in_deadband)
    n = len(sorted_vals)
    mid = n // 2
    if n % 2 == 0:
        return (sorted_vals[mid - 1] + sorted_vals[mid]) / 2.0
    else:
        return sorted_vals[mid]
```

#### **Groq Llama‑3.3‑70b output:**

```python
    """
    Apply the Deadband Protocol to filter readings.
    Removes readings below lower_threshold, keeps those within [lower_threshold, upper_threshold],
    and returns the median of the remaining readings.
    """
    # Handle empty input
    if not readings:
        return None
    
    # P0: Remove readings below lower_threshold
    filtered_readings = [r for r in readings if r >= lower_threshold]
    
    # If no readings remain, return None
    if not filtered_readings:
        return None
    
    # P1: Keep readings within the deadband (<= upper_threshold)
    deadband_readings = [r for r in filtered_readings if r <= upper_threshold]
    
    # If no readings in deadband, return None
    if not deadband_readings:
        return None
    
    # P2: Compute median of deadband readings
    sorted_readings = sorted(deadband_readings)
    n = len(sorted_readings)
    mid_index = n // 2
    
    if n % 2 == 0:
        # Even number of readings: average of two middle values
        median_value = (sorted_readings[mid_index - 1] + sorted_readings[mid_index]) / 2
    else:
        # Odd number of readings: middle value
        median_value = sorted_readings[mid_index]
    
    return median_value
```

- Both are correct and implement the protocol exactly.
- DeepSeek’s version is slightly more concise (fewer intermediate variables).
- Groq’s version has more explicit variable names (`deadband_readings`, `mid_index`) and a more verbose docstring.
- Edge‑case handling identical.

### **Function 2: parse_fleet_log**

**Specification:**
- *Description:* Parse a fleet log line like `"[2026‑04‑20 13:45] AGENT=Alchemist CYCLE=332 STATUS=ACTIVE TILES=4"` into a dict.
- *Input:* A string `log_line`.
- *Output:* Dictionary with keys `timestamp`, `agent`, `cycle`, `status`, `tiles`. If a field is missing, map to `None`.
- *Edge cases:* Malformed lines, missing fields, extra whitespace, empty input.

#### **DeepSeek‑chat output:**
```python
def parse_fleet_log(log_line):
    """
    Parse a fleet log line into a structured dictionary.
    Expected format: [timestamp] KEY1=value1 KEY2=value2 ...
    """
    if not log_line or not log_line.strip():
        return {
            'timestamp': None,
            'agent': None,
            'cycle': None,
            'status': None,
            'tiles': None
        }
    
    result = {
        'timestamp': None,
        'agent': None,
        'cycle': None,
        'status': None,
        'tiles': None
    }
    
    # Extract timestamp inside brackets
    import re
    ts_match = re.search(r'\[([^\]]+)\]', log_line)
    if ts_match:
        result['timestamp'] = ts_match.group(1).strip()
    
    # Find all KEY=value pairs
    pairs = re.findall(r'(\w+)=([^\s]+)', log_line)
    for key, val in pairs:
        key_lower = key.lower()
        if key_lower == 'agent':
            result['agent'] = val
        elif key_lower == 'cycle':
            try:
                result['cycle'] = int(val)
            except ValueError:
                result['cycle'] = None
        elif key_lower == 'status':
            result['status'] = val
        elif key_lower == 'tiles':
            try:
                result['tiles'] = int(val)
            except ValueError:
                result['tiles'] = None
    
    return result
```
