## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function `deadband_filter(readings, deadband=0.1)`.
Input: `readings` is a list of floats representing sensor values.
The Deadband Protocol: If a reading changes less than `deadband` from the previous reading, treat it as unchanged (output the previous value). This reduces noise.
Return a filtered list of same length.
Example: readings=[1.0, 1.05, 1.12, 1.13], deadband=0.1 → output [1.0, 1.0, 1.12, 1.12].
Handle empty list. Include docstring and type hints.
```

### **DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(readings: List[float], deadband: float = 0.1) -> List[float]:
    """
    Apply deadband filtering to a sequence of sensor readings.
    
    Args:
        readings: List of float sensor values.
        deadband: Minimum change threshold to consider a reading changed.
        
    Returns:
        Filtered list where values that change less than `deadband` from the
        previous filtered value are replaced with that previous value.
        
    Example:
        >>> deadband_filter([1.0, 1.05, 1.12, 1.13], 0.1)
        [1.0, 1.0, 1.12, 1.12]
    """
    if not readings:
        return []
    
    filtered = [readings[0]]
    for i in range(1, len(readings)):
        current = readings[i]
        if abs(current - filtered[-1]) >= deadband:
            filtered.append(current)
        else:
            filtered.append(filtered[-1])
    return filtered
```

### **Groq Llama-70b output:**
```python
from typing import List

def deadband_filter(readings: List[float], deadband: float = 0.1) -> List[float]:
    """
    Implements a deadband filter on a list of sensor readings.
    
    Parameters:
    readings (List[float]): The list of sensor readings.
    deadband (float): The deadband threshold. Default is 0.1.
    
    Returns:
    List[float]: The filtered list of readings.
    
    Example:
    >>> deadband_filter([1.0, 1.05, 1.12, 1.13], 0.1)
    [1.0, 1.0, 1.12, 1.12]
    """
    if len(readings) == 0:
        return []
    
    result = [readings[0]]
    prev = readings[0]
    
    for val in readings[1:]:
        if abs(val - prev) >= deadband:
            result.append(val)
            prev = val
        else:
            result.append(prev)
    return result
```

### **Comparison:**
- **Correctness:** Both produce correct output for the example. DeepSeek uses `filtered[-1]` directly; Groq tracks `prev` variable. Both handle empty list.
- **Clarity:** DeepSeek's docstring is slightly more detailed about behavior. Groq's variable naming (`prev`, `val`) is clear. Both have type hints.
- **Robustness:** Both handle empty list. Neither explicitly handles non‑float inputs or negative deadband—could be improved.
- **Fleet alignment:** Neither uses absolute claims; both are factual.

**Verdict:** Tie. Both are production‑ready.

## **3. Function 2: `parse_fleet_state`**

**Prompt:**
```
Write a Python function `parse_fleet_state(log_line: str)`.
Input: A log line like "2026‑04‑20 06:00 UTC | Oracle1 | ACTIVE | CPU: 45% | MEM: 2.3GB".
Format: timestamp | agent_name | status | key1: value1 | key2: value2.
Return a dict: {"timestamp": str, "agent": str, "status": str, "metrics": dict}.
If line doesn't match format, return None.
Handle extra/missing spaces. Metrics may be 0 or more.
Include docstring, type hints, and a test example.
```

### **DeepSeek-chat output:**
```python
from typing import Optional, Dict, Any
import re

def parse_fleet_state(log_line: str) -> Optional[Dict[str, Any]]:
    """
    Parse a fleet state log line into structured data.
    
    Args:
        log_line: String in format "timestamp | agent | status | key1: value1 | ..."
        
    Returns:
        Dict with keys: timestamp, agent, status, metrics (dict of metric:value).
        Returns None if line doesn't match basic format.
        
    Example:
        >>> parse_fleet_state("2026‑04‑20 06:00 UTC | Oracle1 | ACTIVE | CPU: 45% | MEM: 2.3GB")
        {
            'timestamp': '2026‑04‑20 06:00 UTC',
            'agent': 'Oracle1',
            'status': 'ACTIVE',
            'metrics': {'CPU': '45%', 'MEM': '2.3GB'}
        }
    """
    # Split by '|' and strip whitespace
    parts = [p.strip() for p in log_line.split('|')]
    if len(parts) < 3:
        return None
    
    timestamp = parts[0]
    agent = parts[1]
    status = parts[2]
    
    metrics = {}
    for part in parts[3:]:
        if ':' in part:
            key, value = part.split(':', 1)
            metrics[key.strip()] = value.strip()
    
    return {
        'timestamp': timestamp,
        'agent': agent,
        'status': status,
        'metrics': metrics
    }
```
