### **Function 1: `deadband_filter`**

**Prompt:**
"Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol. The function takes a list of numeric values `readings` and three threshold parameters: `p0_threshold` (values below this are P0 'negative space'), `p1_min`, and `p1_max` (defines the P1 'safe channel'). It should return a tuple: (`p0_count`, `p1_list`, `p2_list`). `p0_count` is the number of readings in P0. `p1_list` contains readings within the P1 safe channel. `p2_list` contains readings above P1_max (these are P2 'optimization' candidates). Include a docstring and a simple example usage."

#### **DeepSeek-chat Output:**
```python
def deadband_filter(readings, p0_threshold, p1_min, p1_max):
    """
    Implements the Cocapn fleet Deadband Protocol (P0/P1/P2) on sensor readings.

    Parameters:
    readings (list of float): Input sensor readings.
    p0_threshold (float): Values <= this are considered P0 (negative space).
    p1_min (float): Lower bound of P1 safe channel.
    p1_max (float): Upper bound of P1 safe channel.

    Returns:
    tuple: (p0_count, p1_list, p2_list)
        p0_count (int): Number of readings in P0.
        p1_list (list): Readings within [p1_min, p1_max] (P1 safe channel).
        p2_list (list): Readings > p1_max (P2 optimization candidates).
    """
    p0_count = 0
    p1_list = []
    p2_list = []

    for val in readings:
        if val <= p0_threshold:
            p0_count += 1
        elif p1_min <= val <= p1_max:
            p1_list.append(val)
        elif val > p1_max:
            p2_list.append(val)
        # Values between p0_threshold and p1_min are ignored (deadband).

    return p0_count, p1_list, p2_list

# Example usage
if __name__ == "__main__":

    """

    - P0 (negative space): readings <= p0_threshold
    - P1 (safe channel): readings between p1_min and p1_max (inclusive)
    - P2 (optimization): readings > p1_max

    Readings between p0_threshold and p1_min are considered in the deadband
    and are not included in any category.

    Args:
        readings (list): A list of numeric values.
        p0_threshold (float): The threshold for P0.
        p1_min (float): The minimum value for P1.
        p1_max (float): The maximum value for P1.

    Returns:
        tuple: A tuple containing:
            - p0_count (int): The number of readings in P0.
            - p1_list (list): The list of readings in P1.
            - p2_list (list): The list of readings in P2.
    """
    p0_count = 0
    p1_list = []
    p2_list = []

    for reading in readings:
        if reading <= p0_threshold:
            p0_count += 1
        elif reading >= p1_min and reading <= p1_max:
            p1_list.append(reading)
        elif reading > p1_max:
            p2_list.append(reading)
        # If reading is between p0_threshold and p1_min, it is ignored.

    return p0_count, p1_list, p2_list

### **Function 2: `parse_fleet_log`**

**Prompt:**
"Write a Python function `parse_fleet_log(log_line)` that parses a standardized fleet log line. Format: `'[TIMESTAMP] AGENT@CYCLE - MESSAGE'`. Example: `'[2026-04-20T12:34:56Z] Alchemist@317 - Submitted tile 45'`. The function should return a dictionary with keys: `timestamp` (string), `agent` (string), `cycle` (int), `message` (string). If the line doesn't match the format, return `None`. Use regex. Include a docstring."

#### **DeepSeek-chat Output:**
```python
import re

def parse_fleet_log(log_line):
    """
    Parse a standardized Cocapn fleet log line.

    Expected format: '[TIMESTAMP] AGENT@CYCLE - MESSAGE'

    Parameters:
    log_line (str): A single log line.

    Returns:
    dict or None: Dictionary with keys 'timestamp', 'agent', 'cycle', 'message'
                  if format matches, otherwise None.
    """
    pattern = r'^\[([^\]]+)\]\s+([^@]+)@(\d+)\s+-\s+(.+)$'
    match = re.match(pattern, log_line.strip())
    if match:
        timestamp, agent, cycle_str, message = match.groups()
        return {
            'timestamp': timestamp,
            'agent': agent.strip(),
            'cycle': int(cycle_str),
            'message': message.strip()
        }
    return None
