## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol on a 1D list of numeric values.

The protocol has three priority levels:
P0: Map negative space (identify values that are "rocks" - e.g., NaN, infinite, or outside absolute safe bounds).
P1: Find safe channels (identify values within a defined "safe band" around a target).
P2: Optimize within channels (for values in the safe band, apply a smoothing or optimization).

Function signature:
def deadband_filter(signal: List[float], target: float, safe_band: Tuple[float, float], absolute_bounds: Tuple[float, float]) -> Dict[str, Any]:

It should return a dictionary with keys:
- 'p0_rocks': indices of values that violate absolute_bounds or are non-finite.
- 'p1_safe': indices of values within safe_band of target (and not rocks).
- 'p2_optimized': a list where rocks are set to NaN, safe values are passed through, and others are maybe set to target? (You decide a simple P2 logic).
- 'filtered_signal': the resulting P2-optimized list.

Include type hints, a docstring, and basic error checking. Output only the Python code.
```

### **DeepSeek-chat Output:**
```python
from typing import List, Tuple, Dict, Any
import math

def deadband_filter(
    signal: List[float],
    target: float,
    safe_band: Tuple[float, float],
    absolute_bounds: Tuple[float, float]
) -> Dict[str, Any]:
    """
    Apply Deadband Protocol filtering to a 1D signal.

    P0: Identify 'rocks' - values outside absolute bounds or non-finite.
    P1: Identify 'safe' values - within safe_band of target (and not rocks).
    P2: Optimize: rocks -> NaN, safe -> original, others -> target.

    Args:
        signal: Input signal values.
        target: Target value for safe band center.
        safe_band: (lower_offset, upper_offset) from target defining safe band.
        absolute_bounds: (min, max) absolute allowable values.

    Returns:
        Dictionary with:
        - p0_rocks: indices of rock values.
        - p1_safe: indices of safe values.
        - p2_optimized: list of optimized values.
        - filtered_signal: same as p2_optimized (for compatibility).
    """
    # Validate inputs
    if len(safe_band) != 2 or len(absolute_bounds) != 2:
        raise ValueError("safe_band and absolute_bounds must be tuples of length 2")
    if safe_band[0] >= safe_band[1]:
        raise ValueError("safe_band lower must be < upper")

    lower_safe = target + safe_band[0]
    upper_safe = target + safe_band[1]
    abs_min, abs_max = absolute_bounds

    p0_rocks = []
    p1_safe = []
    p2_optimized = []

    for i, val in enumerate(signal):
        # P0: Check for rocks
        is_rock = False
        if not math.isfinite(val):
            is_rock = True
        elif val < abs_min or val > abs_max:
            is_rock = True

        if is_rock:
            p0_rocks.append(i)
            p2_optimized.append(float('nan'))
            continue

        # P1: Check for safe channel
        if lower_safe <= val <= upper_safe:
            p1_safe.append(i)
            p2_optimized.append(val)  # Keep safe value
        else:
            # Not rock, not safe -> apply P2 optimization: move to target
            p2_optimized.append(target)

    return {
        'p0_rocks': p0_rocks,
        'p1_safe': p1_safe,
        'p2_optimized': p2_optimized,
        'filtered_signal': p2_optimized
    }
```

### **Groq Llama-3.3-70b-Specdec Output:**

```python
import math
from typing import List, Tuple, Dict, Any

def deadband_filter(
    signal: List[float],
    target: float,
    safe_band: Tuple[float, float],
    absolute_bounds: Tuple[float, float]
) -> Dict[str, Any]:
    """
    Implements the Deadband Protocol (P0, P1, P2) on a 1D signal.

    P0 (Negative Space): Identifies values that are non-finite or outside absolute bounds.
    P1 (Safe Channels): Identifies values within the safe band around the target.
    P2 (Optimization): Applies optimization:
        - Rocks (P0) are replaced with NaN.
        - Safe values (P1) are kept unchanged.
        - Values outside safe band but within absolute bounds are clamped to the nearest safe band edge.

    Args:
        signal: List of float values representing the input signal.
        target: The target value for the safe band center.
        safe_band: A tuple (lower_offset, upper_offset) defining the safe band relative to target.
        absolute_bounds: A tuple (min, max) defining absolute allowable values.

    Returns:
        A dictionary containing:
            - p0_rocks: Indices of values identified as rocks.
            - p1_safe: Indices of values within the safe band.
            - p2_optimized: The optimized signal after applying P2 rules.
            - filtered_signal: Alias for p2_optimized.
    """
    if len(safe_band) != 2:
        raise ValueError("safe_band must be a tuple of length 2")
    if len(absolute_bounds) != 2:
        raise ValueError("absolute_bounds must be a tuple of length 2")

    lower_safe = target + safe_band[0]
    upper_safe = target + safe_band[1]
    abs_min, abs_max = absolute_bounds

    p0_rocks = []
    p1_safe = []
    p2_optimized = []

    for idx, value in enumerate(signal):
        # P0: Identify rocks (negative space)
        if not math.isfinite(value) or value < abs_min or value > abs_max:
            p0_rocks.append(idx)
            p2_optimized.append(float('nan'))
            continue

        # P1: Identify safe channel
        if lower_safe <= value <= upper_safe:
            p1_safe.append(idx)
            p2_optimized.append(value)
        else:
            # P2: Optimize values outside safe band but within absolute bounds
            # Clamp to nearest safe band edge
            if value < lower_safe:
                optimized_value = lower_safe
            else:  # value > upper_safe
                optimized_value = upper_safe
            p2_optimized.append(optimized_value)

    return {
        "p0_rocks": p0_rocks,
        "p1_safe": p1_safe,
        "p2_optimized": p2_optimized,
        "filtered_signal": p2_optimized
    }
```
