**Specification:** Accept a list of floats `values` and a `deadband` threshold. Return a list of tuples `(index, value, status)`. Status is `"P0"` if `abs(value) < deadband` (negative space), `"P1"` if `value >= deadband` (safe channel), else `"P2"` if `value <= -deadband` (optimization within negative channel? Clarification: P2 is for optimization *within* a safe channel. This spec seems to conflate. I will adjust the spec for testing: `"P1"` for positive safe channel, `"P2"` for negative safe channel, `"P0"` for within deadband (avoid).)

**Adjusted Spec for Test:** `status = "P0"` if `abs(value) < deadband`, `"P1"` if `value >= deadband`, `"P2"` if `value <= -deadband`.

#### **DeepSeek-chat Output:**
```python
def deadband_filter(values, deadband):
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of values.
    
    Args:
        values: List of float values to categorize.
        deadband: Float threshold. Values within +/- deadband are P0.
    
    Returns:
        List of tuples (index, value, status) where status is 'P0', 'P1', or 'P2'.
        P0: |value| < deadband (negative space, avoid)
        P1: value >= deadband (positive safe channel)
        P2: value <= -deadband (negative safe channel)
    """
    if not isinstance(values, list):
        raise TypeError("values must be a list")
    if not isinstance(deadband, (int, float)):
        raise TypeError("deadband must be a number")
    if deadband < 0:
        raise ValueError("deadband must be non-negative")
    
    result = []
    for i, v in enumerate(values):
        if not isinstance(v, (int, float)):
            raise TypeError(f"Value at index {i} must be a number, got {type(v)}")
        
        if abs(v) < deadband:
            status = "P0"
        elif v >= deadband:
            status = "P1"
        else:  # v <= -deadband
            status = "P2"
        result.append((i, v, status))
    
    return result
```

### **Function 2: `parse_fleet_log`**

**Specification:** Parse a log line format: `"2026-04-20 10:48 UTC | AGENT Alchemist | CYCLE 295 | STATUS ACTIVE | MSG Task started"`. Return a dict with keys: `timestamp`, `agent`, `cycle`, `status`, `message`. Assume strict format.

#### **DeepSeek-chat Output:**
```python
import re

def parse_fleet_log(log_line):
    """
    Parse a standardized fleet log entry.
    
    Expected format: "YYYY-MM-DD HH:MM UTC | AGENT Name | CYCLE N | STATUS State | MSG Message"
    
    Args:
        log_line: String log entry.
    
    Returns:
        Dictionary with keys: timestamp, agent, cycle, status, message.
    
    Raises:
        ValueError: If log line does not match expected format.
    """
    pattern = r'^(\d{4}-\d{2}-\d{2} \d{2}:\d{2} UTC) \| AGENT (\w+) \| CYCLE (\d+) \| STATUS (\w+) \| MSG (.+)$'
    match = re.match(pattern, log_line.strip())
    
    if not match:
        raise ValueError(f"Log line does not match expected format: {log_line}")
    
    timestamp, agent, cycle, status, message = match.groups()
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": int(cycle),
        "status": status,
        "message": message.strip()
    }
```

```python
def parse_fleet_log(log_line):
    """
    Parse a fleet log line into its components.
    
    Format: "DATE TIME UTC | AGENT agent_name | CYCLE cycle_num | STATUS status_value | MSG message_text"
    
    Returns a dictionary with extracted fields.
    """
    # Split by the delimiter ' | '
    parts = log_line.split(' | ')
    if len(parts) != 5:
        raise ValueError(f"Invalid log format. Expected 5 parts, got {len(parts)}: {log_line}")
    
    # Extract components
    timestamp = parts[0].strip()
    
    if not parts[1].startswith('AGENT '):
        raise ValueError(f"AGENT field malformed: {parts[1]}")
    agent = parts[1][6:].strip()  # Remove 'AGENT '
    
    if not parts[2].startswith('CYCLE '):
        raise ValueError(f"CYCLE field malformed: {parts[2]}")
    cycle_str = parts[2][6:].strip()
    try:
        cycle = int(cycle_str)
    except ValueError:
        raise ValueError(f"CYCLE must be integer: {cycle_str}")
    
    if not parts[3].startswith('STATUS '):
        raise ValueError(f"STATUS field malformed: {parts[3]}")
    status = parts[3][7:].strip()  # Remove 'STATUS '
    
    if not parts[4].startswith('MSG '):
        raise ValueError(f"MSG field malformed: {parts[4]}")
    message = parts[4][4:].strip()  # Remove 'MSG '
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message
    }
```
