## **2. Function 1: `deadband_filter`**

**Prompt:**
"Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol. The function takes a list of numeric values and three threshold tuples: `p0_threshold`, `p1_threshold`, `p2_threshold`. Each threshold is a tuple `(low, high)`.
- **P0 (Negative Space):** Any value outside the P0 band is rejected (filtered out). Return `None` for these.
- **P1 (Safe Channel):** Values inside the P0 band but outside the P1 band are considered 'safe but suboptimal'. They should be returned, but marked.
- **P2 (Optimization):** Values inside the P1 band are 'optimal'. They should be returned and marked as such.
The function should return a list of tuples: `(original_value, status, clipped_value)`, where `status` is one of `'rejected'`, `'safe'`, `'optimal'`, and `clipped_value` is the original value, or if in P1, optionally clipped to the nearest P1 boundary (your choice, document it). Include type hints, a docstring, and a simple example."

### **DeepSeek-chat Output:**
```python
from typing import List, Tuple, Optional

def deadband_filter(values: List[float],
                    p0_threshold: Tuple[float, float],
                    p1_threshold: Tuple[float, float],
                    p2_threshold: Tuple[float, float]) -> List[Tuple[float, str, Optional[float]]]:
    """
    Implements the Cocapn Deadband Protocol (P0/P1/P2) for numeric filtering.

    Args:
        values: List of input values to filter.
        p0_threshold: (low, high) tuple. Values outside this band are P0 (rejected).
        p1_threshold: (low, high) tuple. Values inside P0 but outside this are P1 (safe).
        p2_threshold: (low, high) tuple. Values inside this are P2 (optimal).

    Returns:
        List of tuples (original_value, status, clipped_value).
        Status: 'rejected', 'safe', or 'optimal'.
        Clipped value: For 'optimal', same as input. For 'safe', clipped to nearest P1 boundary.
        For 'rejected', None.
    """
    p0_low, p0_high = p0_threshold
    p1_low, p1_high = p1_threshold
    p2_low, p2_high = p2_threshold

    result = []
    for v in values:
        # P0 check: outside P0 band -> rejected
        if v < p0_low or v > p0_high:
            result.append((v, 'rejected', None))
            continue

        # P2 check: inside P2 band -> optimal
        if p2_low <= v <= p2_high:
            result.append((v, 'optimal', v))
            continue

        # P1: inside P0 but outside P2 -> safe (clip to nearest P1 boundary)
        # Determine nearest boundary (p1_low or p1_high)
        dist_to_low = abs(v - p1_low)
        dist_to_high = abs(v - p1_high)
        clipped = p1_low if dist_to_low <= dist_to_high else p1_high
        result.append((v, 'safe', clipped))

    return result


# Example usage
if __name__ == "__main__":
    data = [0.5, 1.2, 2.0, 2.8, 3.5, 5.0]
    p0 = (0.0, 4.0)   # Reject <0 or >4
    p1 = (1.0, 3.0)   # Safe channel
    p2 = (1.5, 2.5)   # Optimal zone

    filtered = deadband_filter(data, p0, p1, p2)
    for val, status, clipped in filtered:
        print(f"{val}: {status} -> {clipped}")
```

def deadband_filter(values: List[float],
                    p0_threshold: Tuple[float, float],
                    p1_threshold: Tuple[float, float],
                    p2_threshold: Tuple[float, float]) -> List[Tuple[float, str, Optional[float]]]:
    """
    Apply Deadband Protocol filtering.

    Parameters:
    - values: input numbers
    - p0_threshold: (low, high) for rejection band
    - p1_threshold: (low, high) for safe band
    - p2_threshold: (low, high) for optimal band

    Returns:
    List of (original, status, clipped) where status in ['rejected','safe','optimal']
    """
    p0_low, p0_high = p0_threshold
    p1_low, p1_high = p1_threshold
    p2_low, p2_high = p2_threshold

    out = []
    for v in values:
        if v < p0_low or v > p0_high:
            out.append((v, 'rejected', None))
        elif v >= p2_low and v <= p2_high:
            out.append((v, 'optimal', v))
        else:
            # safe zone: clip to nearest p1 boundary
            if v < p1_low:
                clipped = p1_low
            elif v > p1_high:
                clipped = p1_high
            else:
                # This case shouldn't happen per logic, but fallback
                clipped = v
            out.append((v, 'safe', clipped))
    return out
