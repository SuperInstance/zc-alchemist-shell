## **2. Prompt Used (identical for both models)**

```
You are a Python developer in the Cocapn fleet. Write a clean, production-ready function with docstring and type hints.

Write a function called `deadband_filter` that implements the fleet's Deadband Protocol.

Input: A list of float `readings` (sensor data) and a float `deadband` (threshold).
P0 (negative space): Identify readings that are within +/- deadband of zero — these are "unsafe" and must be removed.
P1 (safe channels): From remaining readings, select those that are positive (safe channel A) and those that are negative (safe channel B), returning them as separate lists.
P2 (optimize): For each safe channel, return the reading with the maximum absolute value (most representative).

Return a dict: {
    "p0_unsafe": list of removed readings,
    "p1_safe_positive": list of positive safe readings,
    "p1_safe_negative": list of negative safe readings,
    "p2_optim_positive": max positive reading or None,
    "p2_optim_negative": min negative reading or None  # most negative is max absolute value in negative channel
}

If no readings in a channel, return empty list and None for optim.
```

## **3. Model Outputs**

### **DeepSeek-chat (local) output:**

```python
from typing import List, Dict, Optional, Tuple

def deadband_filter(readings: List[float], deadband: float) -> Dict[str, List[float] or Optional[float]]:
    """
    Apply the Cocapn fleet Deadband Protocol to sensor readings.
    
    P0: Remove readings within +/- deadband of zero (unsafe/ambiguous).
    P1: Separate remaining readings into positive and negative safe channels.
    P2: For each safe channel, select the reading with maximum absolute value.
    
    Args:
        readings: List of sensor readings.
        deadband: Threshold around zero considered unsafe.
        
    Returns:
        Dictionary with:
        - p0_unsafe: Readings removed as unsafe.
        - p1_safe_positive: Positive safe readings.
        - p1_safe_negative: Negative safe readings.
        - p2_optim_positive: Max positive reading (or None).
        - p2_optim_negative: Min negative reading (or None) — most negative is max absolute in negative channel.
    """
    # P0: Map negative space
    p0_unsafe = [r for r in readings if -deadband <= r <= deadband]
    safe_readings = [r for r in readings if not (-deadband <= r <= deadband)]
    
    # P1: Find safe channels
    p1_safe_positive = [r for r in safe_readings if r > 0]
    p1_safe_negative = [r for r in safe_readings if r < 0]
    
    # P2: Optimize within channels
    p2_optim_positive = max(p1_safe_positive) if p1_safe_positive else None
    p2_optim_negative = min(p1_safe_negative) if p1_safe_negative else None
    
    return {
        "p0_unsafe": p0_unsafe,
        "p1_safe_positive": p1_safe_positive,
        "p1_safe_negative": p1_safe_negative,
        "p2_optim_positive": p2_optim_positive,
        "p2_optim_negative": p2_optim_negative
    }
```

### **Groq Llama-70b (cloud) output:**

```python
from typing import List, Dict, Optional

def deadband_filter(readings: List[float], deadband: float) -> Dict[str, List[float] or Optional[float]]:
    """
    Implements the Deadband Protocol for sensor data filtering.
    
    Steps:
    1. P0 (Negative Space): Remove readings within deadband of zero.
    2. P1 (Safe Channels): Split remaining into positive and negative.
    3. P2 (Optimize): Pick extreme values from each channel.
    
    Args:
        readings: Input sensor readings.
        deadband: Absolute deadband threshold.
        
    Returns:
        Dictionary with filtered results.
    """
    # P0: Identify unsafe readings (within deadband)
    unsafe = [r for r in readings if abs(r) <= deadband]
    
    # Remaining safe readings
    safe = [r for r in readings if abs(r) > deadband]
    
    # P1: Split into positive and negative channels
    positive = [r for r in safe if r > 0]
    negative = [r for r in safe if r < 0]
    
    # P2: Optimize - get max absolute value in each channel
    optim_pos = max(positive) if positive else None
    optim_neg = min(negative) if negative else None  # min negative = most negative = max absolute
    
    return {
        "p0_unsafe": unsafe,
        "p1_safe_positive": positive,
        "p1_safe_negative": negative,
        "p2_optim_positive": optim_pos,
        "p2_optim_negative": optim_neg
    }
```
