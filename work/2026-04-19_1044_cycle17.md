**User Prompt:**
```
Write a function `constraint_snap(point: tuple[float, float], constraints: list[tuple[float, float, float]], threshold: float = 0.5) -> tuple[float, float]`.
Given a 2D point (x, y) and a list of constraints where each constraint is (a, b, c) representing line a*x + b*y + c = 0.
Find the distance from point to each line. If the minimum distance is <= threshold, "snap" the point to the closest line by projecting onto that line.
Return the snapped point (or original if no snap).
Use perpendicular projection: for line (a,b,c), projected point = (x - a*d, y - b*d) where d = (a*x + b*y + c)/(a*a + b*b).
Example: point=(1.0, 2.0), constraints=[(1,0,-1), (0,1,-1)] (lines x=1 and y=1). Distances: 0.0 to x=1, 1.0 to y=1. Snap to x=1 within threshold 0.5 -> return (1.0, 2.0).
```

    """
    Apply deadband filtering to a list of float values.

import math

def constraint_snap(point: tuple[float, float], 
                    constraints: list[tuple[float, float, float]], 
                    threshold: float = 0.5) -> tuple[float, float]:
    """
    Snap a 2D point to the nearest constraint line if within threshold.
    """
    x, y = point
    min_dist = float('inf')
    best_constraint = None
    
    for a, b, c in constraints:
        denom = a*a + b*b
        if denom == 0:
            continue  # invalid constraint
        dist = abs(a*x + b*y + c) / math.sqrt(denom)
        if dist < min_dist:
            min_dist = dist
            best_constraint = (a, b, c)
    
    if min_dist <= threshold and best_constraint is not None:
        a, b, c = best_constraint
        denom = a*a + b*b
        d = (a*x + b*y + c) / denom
        snapped_x = x - a*d
        snapped_y = y - b*d
        return (snapped_x, snapped_y)
    
    return point
