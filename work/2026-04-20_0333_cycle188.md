### **Function 1: Deadband Filter**

**Requirement:** Write a function `apply_deadband_protocol` that takes a list of float sensor readings and returns a processed list. The protocol: P0 (Negative Space): Remove any readings that are `<= 0.0`. P1 (Safe Channel): Keep only readings where `0.0 < reading <= 100.0`. P2 (Optimize): For the remaining readings, apply a simple optimization: if a reading is `> 75.0`, scale it down by 10%. Return the final list.

#### **DeepSeek-chat Output:**
```python
from typing import List

def apply_deadband_protocol(sensor_readings: List[float]) -> List[float]:
    """
    Apply the fleet's Deadband Protocol (P0/P1/P2) to a list of sensor readings.

    Protocol:
    - P0 (Negative Space): Remove readings <= 0.0
    - P1 (Safe Channel): Keep readings where 0.0 < reading <= 100.0
    - P2 (Optimize): Scale readings > 75.0 down by 10%

    Parameters:
    -----------
    sensor_readings : List[float]
        Raw list of sensor readings.

    Returns:
    --------
    List[float]
        Processed list of readings after applying the protocol.
    """
    # P0: Map negative space - remove invalid/negative readings
    p0_filtered = [r for r in sensor_readings if r > 0.0]

    # P1: Find safe channels - keep only readings in valid operational range
    p1_filtered = [r for r in p0_filtered if r <= 100.0]

    # P2: Optimize within channels - apply scaling to high readings
    p2_optimized = []
    for reading in p1_filtered:
        if reading > 75.0:
            # Scale down by 10%
            optimized_reading = reading * 0.9
            p2_optimized.append(round(optimized_reading, 2))
        else:
            p2_optimized.append(reading)

    return p2_optimized
```

```python
from typing import List

def apply_deadband_protocol(sensor_readings: List[float]) -> List[float]:
    """
    Applies the Deadband Protocol (P0/P1/P2) to sensor readings.

    Steps:
    1. P0 (Negative Space): Remove readings <= 0.0
    2. P1 (Safe Channel): Keep readings where 0.0 < reading <= 100.0
    3. P2 (Optimize): For readings > 75.0, scale down by 10%

    Parameters:
    -----------
    sensor_readings : List[float]
        A list of floating-point sensor readings.

    Returns:
    --------
    List[float]
        The processed list of sensor readings after applying the protocol.
    """
    if not isinstance(sensor_readings, list):
        raise TypeError("sensor_readings must be a list")

    # P0: Remove non-positive readings
    p0_readings = [reading for reading in sensor_readings if isinstance(reading, (int, float)) and reading > 0.0]

    # P1: Keep readings in safe range (0, 100]
    p1_readings = [reading for reading in p0_readings if reading <= 100.0]

    # P2: Optimize readings > 75.0 by scaling down 10%
    result = []
    for reading in p1_readings:
        if reading > 75.0:
            reading = reading * 0.9
        result.append(round(reading, 2))

    return result
```

### **Function 2: Training Tile Validator**

**Requirement:** Write a function `validate_training_tile` that takes a dictionary `tile` and returns a tuple `(bool, str)` indicating validity and an error message if invalid. A valid tile must have: a string key `"id"` starting with 'tile_', a string key `"content"` of non-zero length, a list key `"tags"` containing only strings, and an optional key `"metadata"` which must be a dict if present.

#### **DeepSeek-chat Output:**
```python
from typing import Dict, Any, Tuple, List

def validate_training_tile(tile: Dict[str, Any]) -> Tuple[bool, str]:
    """
    Validate if a dictionary represents a valid training tile for the PLATO system.

    A valid tile must contain:
    - 'id': string starting with 'tile_'
    - 'content': non-empty string
    - 'tags': list containing only strings
    - 'metadata': optional, but if present must be a dictionary

    Parameters:
    -----------
    tile : Dict[str, Any]
        The tile dictionary to validate.

    Returns:
    --------
    Tuple[bool, str]
        (is_valid, error_message). If valid, error_message is empty string.
    """
    # Check required keys exist
    required_keys = {"id", "content", "tags"}
    if not required_keys.issubset(tile.keys()):
        missing = required_keys - set(tile.keys())
        return False, f"Missing required keys: {missing}"

    # Validate 'id'
    if not isinstance(tile["id"], str):
        return False, "'id' must be a string"
    if not tile["id"].startswith("tile_"):
        return False, "'id' must start with 'tile_'"

    # Validate 'content'
    if not isinstance(tile["content"], str):
        return False, "'content' must be a string"
    if len(tile["content"].strip()) == 0:
        return False, "'content' cannot be empty"

    # Validate 'tags'
    if not isinstance(tile["tags"], list):
        return False, "'tags' must be a list"
    if not all(isinstance(tag, str) for tag in tile["tags"]):
        return False, "All elements in 'tags' must be strings"

    # Validate optional 'metadata'
    if "metadata" in tile and not isinstance(tile["metadata"], dict):
        return False, "'metadata' must be a dictionary if present"

    return True, ""
```
