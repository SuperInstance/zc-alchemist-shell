## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input: 
- `values`: list of floats (could be empty)
- `p0_threshold`: float, the "negative space" bound (absolute). Any value with abs(value) < p0_threshold is considered unsafe (P0 violation).
- `p1_min`, `p1_max`: floats, the safe channel bounds. Values within [p1_min, p1_max] are safe (P1).
- `p2_target`: float, the optimal target within P1.

Logic:
1. P0: Remove any values where abs(value) < p0_threshold (too close to zero/unsafe).
2. P1: Keep only values where p1_min <= value <= p1_max (safe channel).
3. P2: From remaining values, pick the one closest to p2_target. If multiple, pick the first.

Return: 
- Filtered value (float) or None if no values survive P0+P1.

Include docstring, type hints, and a brief example in comments.
```

### **DeepSeek-chat output:**
```python
from typing import List, Optional

def deadband_filter(
    values: List[float],
    p0_threshold: float,
    p1_min: float,
    p1_max: float,
    p2_target: float
) -> Optional[float]:
    """
    Implements the Cocapn fleet Deadband Protocol (P0, P1, P2) for numeric filtering.

    Steps:
    1. P0 (negative space): Remove values too close to zero (abs(v) < p0_threshold).
    2. P1 (safe channel): Keep values within [p1_min, p1_max].
    3. P2 (optimization): Choose value closest to p2_target.

    Args:
        values: Input list of floats.
        p0_threshold: Absolute bound for P0 exclusion.
        p1_min: Lower bound of safe channel.
        p1_max: Upper bound of safe channel.
        p2_target: Optimal target within safe channel.

    Returns:
        Filtered float or None if no values survive P0+P1.

    Example:
        >>> deadband_filter([-0.5, 1.2, 3.0, 0.1], 0.2, 0.5, 4.0, 2.0)
        1.2  # P0 removes 0.1, P1 keeps 1.2 and 3.0, P2 picks 1.2 (closer to 2.0)
    """
    if not values:
        return None

    # P0: Map negative space
    p0_safe = [v for v in values if abs(v) >= p0_threshold]
    if not p0_safe:
        return None

    # P1: Find safe channels
    p1_safe = [v for v in p0_safe if p1_min <= v <= p1_max]
    if not p1_safe:
        return None

    # P2: Optimize within channels
    best_value = min(p1_safe, key=lambda x: abs(x - p2_target))
    return best_value
```

    """

    Parameters:
    - values: list of floats to filter
    - p0_threshold: values with absolute value less than this are discarded (P0)
    - p1_min, p1_max: safe channel bounds; values outside [p1_min, p1_max] are discarded (P1)

    Returns:
    - The value from the input list that survives P0 and P1 and is closest to p2_target.
    - Returns None if no values survive P0 and P1.

    Example:
    >>> deadband_filter([-0.1, 0.5, 1.5, 2.5], 0.2, 0.0, 2.0, 1.0)
    0.5
    """

        return None

    return result
