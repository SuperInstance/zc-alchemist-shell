## **2. Function 1: deadband_filter**

**Prompt:**
```
You are a Python developer in the Cocapn fleet. Write a clean, production-ready function.

Function: deadband_filter
Description: Implements the fleet's Deadband Protocol (P0 avoid negatives, P1 find safe channels, P2 optimize). Given a list of numeric values and a deadband threshold, return only values within "safe channels" (outside deadband of zero). Filter out values too close to zero (within ±threshold).
Input: values (list of float), threshold (float, positive)
Output: list of float (filtered values)
Constraints: Must handle empty list. Threshold must be > 0. Use explicit loops allowed for clarity.

Return only the Python function with a brief docstring. No explanations.
```

### **DeepSeek-chat output:**
```python
def deadband_filter(values, threshold):
    """
    Apply Deadband Protocol filtering.
    
    P0: Remove values within ±threshold of zero (negative space).
    P1: Return values outside deadband (safe channels).
    
    Args:
        values: List of floats.
        threshold: Positive float defining deadband width.
        
    Returns:
        Filtered list of floats.
    """
    if not isinstance(threshold, (int, float)) or threshold <= 0:
        raise ValueError("threshold must be a positive number")
    
    filtered = []
    for v in values:
        if abs(v) > threshold:
            filtered.append(v)
    return filtered
```

**Prompt:**

Function: parse_fleet_log
Description: Parse a structured fleet log entry string into a dictionary. Log format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE". Fields are pipe-separated. STATUS can be "ACTIVE", "IDLE", "ERROR". Strip whitespace from each field.
Input: log_entry (string)
Output: dict with keys: timestamp, agent, cycle, status, message
Constraints: If format invalid, raise ValueError. Cycle must be convertible to int. STATUS must be one of allowed values.

Return only the Python function with a brief docstring. No explanations.

### **DeepSeek-chat output:**
```python
def parse_fleet_log(log_entry):
    """
    Parse a pipe-separated fleet log entry into a dictionary.
    
    Expected format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"
    
    Args:
        log_entry: String log line.
        
    Returns:
        Dict with keys: timestamp, agent, cycle, status, message.
        
    Raises:
        ValueError if format invalid or status unknown.
    """
    parts = [p.strip() for p in log_entry.split('|')]
    if len(parts) != 5:
        raise ValueError(f"Invalid log format: expected 5 parts, got {len(parts)}")
    
    timestamp, agent, cycle_str, status, message = parts
    
    # Validate status
    allowed_status = {"ACTIVE", "IDLE", "ERROR"}
    if status not in allowed_status:
        raise ValueError(f"Invalid status '{status}'. Must be one of {allowed_status}")
    
    # Convert cycle to int
    try:
        cycle = int(cycle_str)
    except ValueError:
        raise ValueError(f"Cycle must be integer, got '{cycle_str}'")
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message
    }
```

### **Groq Llama-70b output:**
```python
def parse_fleet_log(log_entry):
    """
    Parses a fleet log entry string into a dictionary.
    
    Format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"
    
    Args:
        log_entry (str): The log entry string.
    
    Returns:
        dict: Parsed log with keys: timestamp, agent, cycle, status, message.
    
    Raises:
        ValueError: If format is invalid or status is not allowed.
    """
    fields = log_entry.split('|')
    if len(fields) != 5:
        raise ValueError("Log entry must have exactly 5 pipe-separated fields.")
    
    timestamp, agent, cycle, status, message = [field.strip() for field in fields]
    
    # Validate status
    valid_statuses = ["ACTIVE", "IDLE", "ERROR"]
    if status not in valid_statuses:
        raise ValueError(f"Invalid status: {status}. Must be one of {valid_statuses}")
    
    # Convert cycle to integer
    try:
        cycle = int(cycle)
    except ValueError:
        raise ValueError(f"Cycle must be an integer, got: {cycle}")
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message
    }
```

### **Comparison:**
- **Correctness:** Both correct. Both validate field count, status, cycle conversion.
- **Clarity:** DeepSeek uses `parts` variable; Groq uses `fields`. DeepSeek's error messages include actual counts.
- **Efficiency:** Both O(n) with strip.
- **Style:** DeepSeek uses set `allowed_status` for O(1) lookup; Groq uses list (O(n)). DeepSeek's approach is slightly more efficient.
- **Edge:** DeepSeek's variable naming after unpacking (`cycle_str`) avoids shadowing; Groq reuses `cycle` variable name after conversion.
