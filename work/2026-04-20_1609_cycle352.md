## **2. Function 1: Deadband Filter**

**Prompt:**
```
Write a Python function `apply_deadband` that implements the Cocapn fleet's Deadband Protocol.

Inputs:
- `signal`: list of floats (time-series signal)
- `p0_threshold`: float (negative space boundary)
- `p1_min`, `p1_max`: floats (safe channel boundaries)

Logic:
P0: Any value where abs(signal[i]) < p0_threshold → set to 0.0 (negative space)
P1: Values between p1_min and p1_max → keep unchanged (safe channel)
P2: Values outside P1 range → clamp to nearest P1 boundary (optimize within channel)

Return: processed list of floats same length as signal.

Include docstring and example usage.
```

### **DeepSeek-chat output:**
```python
def apply_deadband(signal, p0_threshold, p1_min, p1_max):
    """
    Apply the Deadband Protocol (P0/P1/P2) to a time-series signal.
    
    Parameters:
    signal (list[float]): Input signal values.
    p0_threshold (float): P0 deadband threshold. Values with |value| < p0_threshold are set to zero.
    p1_min (float): Lower bound of P1 safe channel.
    p1_max (float): Upper bound of P1 safe channel.
    
    Returns:
    list[float]: Processed signal after applying deadband logic.
    
    Protocol:
    - P0 (negative space): |value| < p0_threshold → 0.0
    - P1 (safe channel): p1_min ≤ value ≤ p1_max → keep unchanged
    - P2 (optimize): otherwise → clamp to nearest P1 boundary
    """
    processed = []
    for value in signal:
        # P0: Map negative space
        if abs(value) < p0_threshold:
            processed.append(0.0)
        # P1: Safe channel
        elif p1_min <= value <= p1_max:
            processed.append(value)
        # P2: Optimize within channel (clamp)
        else:
            if value < p1_min:
                processed.append(p1_min)
            else:  # value > p1_max
                processed.append(p1_max)
    return processed
```

```python
def apply_deadband(signal, p0_threshold, p1_min, p1_max):
    """
    Implements the Deadband Protocol for signal processing.
    
    Args:
        signal: A list of float values representing the input signal.
        p0_threshold: Threshold for P0 (negative space). Values with absolute value less than this are set to zero.
        p1_min: Minimum value of the P1 safe channel.
        p1_max: Maximum value of the P1 safe channel.
    
    Returns:
        A list of processed float values.
    """
    result = []
    for val in signal:
        abs_val = abs(val)
        # P0: negative space
        if abs_val < p0_threshold:
            result.append(0.0)
        # P1: safe channel
        elif p1_min <= val <= p1_max:
            result.append(val)
        # P2: optimize by clamping
        else:
            if val < p1_min:
                result.append(p1_min)
            elif val > p1_max:
                result.append(p1_max)
            else:
                # This case shouldn't happen due to above conditions, but for safety
                result.append(val)
    return result
```
