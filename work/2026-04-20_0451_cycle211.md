## **3. Function 1: Deadband Filter**

**Prompt details:**
- **function_description:** Apply the Deadband Protocol to a list of sensor readings. P0: identify readings below a danger threshold. P1: identify readings within a safe operating range. P2: from the safe readings, return the optimal (median) value.
- **function_name:** `apply_deadband_protocol`
- **input_description:** `readings: List[float], danger_threshold: float, safe_min: float, safe_max: float`
- **output_description:** `Dict with keys: 'p0_readings', 'p1_readings', 'p2_optimal'`

### **DeepSeek-chat output:**

```python
from typing import List, Dict

def apply_deadband_protocol(
    readings: List[float],
    danger_threshold: float,
    safe_min: float,
    safe_max: float
) -> Dict[str, List[float] | float]:
    """
    Apply Cocapn fleet Deadband Protocol to sensor readings.
    
    P0 (negative space): Identify readings below danger threshold (what NOT to use).
    P1 (safe channels): Identify readings within safe operating range (where we CAN be).
    P2 (optimize): From safe readings, select the median as optimal path.
    
    Args:
        readings: List of sensor readings.
        danger_threshold: Readings below this are in P0 danger zone.
        safe_min: Minimum safe value for P1 range.
        safe_max: Maximum safe value for P1 range.
    
    Returns:
        Dictionary with:
        - 'p0_readings': List of readings in P0 (danger zone).
        - 'p1_readings': List of readings in P1 (safe channel).
        - 'p2_optimal': Median of P1 readings (optimal value), or None if no safe readings.
    """
    # P0: Map negative space
    p0_readings = [r for r in readings if r < danger_threshold]
    
    # P1: Find safe channels
    p1_readings = [r for r in readings if safe_min <= r <= safe_max]
    
    # P2: Optimize within channels
    p2_optimal = None
    if p1_readings:
        sorted_readings = sorted(p1_readings)
        mid = len(sorted_readings) // 2
        p2_optimal = sorted_readings[mid] if len(sorted_readings) % 2 == 1 else \
                     (sorted_readings[mid - 1] + sorted_readings[mid]) / 2
    
    return {
        'p0_readings': p0_readings,
        'p1_readings': p1_readings,
        'p2_optimal': p2_optimal
    }

# Example usage
if __name__ == "__main__":
    sensor_data = [1.2, 3.4, 0.5, 7.8, 2.9, 0.3, 5.6]
    result = apply_deadband_protocol(sensor_data, danger_threshold=1.0, safe_min=2.0, safe_max=6.0)
    print(f"P0 (danger): {result['p0_readings']}")
    print(f"P1 (safe): {result['p1_readings']}")
    print(f"P2 (optimal): {result['p2_optimal']}")
```

### **Groq Llama-70b output:**

```python
from typing import List, Dict, Optional

def apply_deadband_protocol(
    readings: List[float],
    danger_threshold: float,
    safe_min: float,
    safe_max: float
) -> Dict[str, List[float] | Optional[float]]:
    """
    Implements the Cocapn Fleet Deadband Protocol for sensor data triage.
    
    Protocol:
    - P0 (Negative Space): Identify readings that are hazardous (below danger_threshold).
    - P1 (Safe Channels): Identify readings within the acceptable operational range [safe_min, safe_max].
    - P2 (Optimization): From the safe readings, compute the median as the optimal operating point.
    
    This mimics the fleet's decision-making where safety (P0) is prioritized over optimization (P2).
    """
    # P0: Negative space – readings to avoid
    p0 = [r for r in readings if r < danger_threshold]
    
    # P1: Safe channels – readings within operational bounds
    p1 = [r for r in readings if (r >= safe_min) and (r <= safe_max)]
    
    # P2: Optimization – median of safe readings
    p2 = None
    if p1:
        sorted_p1 = sorted(p1)
        n = len(sorted_p1)
        if n % 2 == 1:
            p2 = sorted_p1[n // 2]
        else:
            mid1 = sorted_p1[n // 2 - 1]
            mid2 = sorted_p1[n // 2]
            p2 = (mid1 + mid2) / 2.0
    
    return {
        "p0_readings": p0,
        "p1_readings": p1,
        "p2_optimal": p2
    }

# Example usage
if __name__ == "__main__":
    # Simulated sensor data from a fleet monitor
    test_readings = [0.8, 2.5, 3.0, 1.1, 4.5, 0.9, 5.0]
    danger = 1.0
    low = 2.0
    high = 5.0
    
    outcome = apply_deadband_protocol(test_readings, danger, low, high)
    print("Deadband Protocol Results:")
    print(f"P0 (Hazardous): {outcome['p0_readings']}")
    print(f"P1 (Safe): {outcome['p1_readings']}")
    print(f"P2 (Optimal): {outcome['p2_optimal']}")
```
