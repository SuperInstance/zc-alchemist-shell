## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter(values: List[float], p0_thresh: float = 0.1, p1_margin: float = 0.05) -> List[str]`.
Apply the deadband protocol:
- P0: If abs(value) < p0_thresh → "P0: avoid" (too close to zero, negative space)
- P1: If p0_thresh <= abs(value) <= (p0_thresh + p1_margin) → "P1: safe channel"
- P2: Otherwise → "P2: optimize"
Return list of strings.
Include docstring and example.
```

### **DeepSeek-chat output:**
```python
from typing import List

def deadband_filter(values: List[float], p0_thresh: float = 0.1, p1_margin: float = 0.05) -> List[str]:
    """
    Apply deadband protocol (P0/P1/P2) to a list of float values.
    
    Parameters:
    -----------
    values : List[float]
        Input values to classify.
    p0_thresh : float, default=0.1
        Threshold for P0 (negative space).
    p1_margin : float, default=0.05
        Additional margin for P1 safe channel.
        
    Returns:
    --------
    List[str]
        Classification for each value: "P0: avoid", "P1: safe channel", or "P2: optimize".
    
    Example:
    --------
    >>> deadband_filter([0.05, 0.12, 0.2, -0.08, -0.15])
    ['P0: avoid', 'P1: safe channel', 'P2: optimize', 'P0: avoid', 'P2: optimize']
    """
    results = []
    p1_max = p0_thresh + p1_margin
    
    for v in values:
        abs_v = abs(v)
        if abs_v < p0_thresh:
            results.append("P0: avoid")
        elif abs_v <= p1_max:
            results.append("P1: safe channel")
        else:
            results.append("P2: optimize")
    
    return results
```

### **Groq Llama-70b output:**

```python
from typing import List

def deadband_filter(values: List[float], p0_thresh: float = 0.1, p1_margin: float = 0.05) -> List[str]:
    """
    Categorize values according to the deadband protocol.
    
    Args:
        values: A list of floating point numbers.
        p0_thresh: The threshold for P0 (default 0.1).
        p1_margin: The margin for P1 (default 0.05).
    
    Returns:
        A list of strings indicating the category for each value.
    """
    categories = []
    for val in values:
        abs_val = abs(val)
        if abs_val < p0_thresh:
            categories.append("P0: avoid")
        elif abs_val <= p0_thresh + p1_margin:
            categories.append("P1: safe channel")
        else:
            categories.append("P2: optimize")
    return categories

# Example usage:

    sample = [0.05, 0.12, 0.2, -0.08, -0.15]
```

**Comparison:**
- **Correctness:** Both correct. Logic identical.
- **Code quality:** DeepSeek includes intermediate variable `p1_max` for clarity. Groq computes inline. Both readable.
- **Conciseness:** Groq slightly shorter (no intermediate var). DeepSeek more explicit.
- **Extras:** DeepSeek provides detailed docstring with example in Google format. Groq uses simpler docstring, adds `if __name__ == "__main__"` block.
- **Edge cases:** Neither handles NaN/inf. Both handle empty list.

**Verdict:** Tie. DeepSeek better documentation; Groq slightly cleaner loop.

```python
import subprocess
import os
from typing import Dict, Any, List

def git_repo_health(repo_path: str) -> Dict[str, Any]:
    """
    Check health metrics of a local git repository.
    
    Parameters:
    -----------
    repo_path : str
        Path to the repository.
        
    Returns:
    --------
    Dict[str, Any]
        Dictionary containing:
        - is_repo: bool
        - is_clean: bool (True if no uncommitted changes)
        - large_files: List[str] (paths > 10MB)
        - branch: str (current branch)
        - commit_count: int (total commits on current branch)
        
    Raises:
    -------
    ValueError
        If repo_path is not a git repository.
    """
    # Check if path exists and is a git repo
    git_dir = os.path.join(repo_path, '.git')
    if not os.path.isdir(git_dir):
        raise ValueError(f"Not a git repository: {repo_path}")
    
    def run_git(cmd: str) -> str:
        """Run git command and return stripped stdout."""
        try:
            result = subprocess.run(
                cmd, shell=True, cwd=repo_path,
                capture_output=True, text=True, timeout=5
            )
            if result.returncode != 0:
                return ""
            return result.stdout.strip()
        except subprocess.TimeoutExpired:
            return ""
    
    # Check if working directory is clean
    status = run_git("git status --porcelain")
    is_clean = status == ""
    
    # Find files > 10MB
    large_files = []
    find_cmd = "git ls-files | while read f; do if [ $(stat -f%z \"$f\" 2>/dev/null || stat -c%s \"$f\") -gt 10485760 ]; then echo \"$f\"; fi; done"
    # Simpler cross-platform approach using Python
    for root, dirs, files in os.walk(repo_path):
        for f in files:
            if f == '.git':
                continue
            fp = os.path.join(root, f)
            try:
                if os.path.getsize(fp) > 10 * 1024 * 1024:  # 10MB
                    large_files.append(os.path.relpath(fp, repo_path))
            except OSError:
                pass
    
    # Get current branch
    branch = run_git("git branch --show-current")
    if not branch:
        branch = "detached"
    
    # Get commit count
    commit_count = run_git("git rev-list --count HEAD")
    commit_count = int(commit_count) if commit_count.isdigit() else 0
    
    return {
        "is_repo": True,
        "is_clean": is_clean,
        "large_files": large_files,
        "branch": branch,
        "commit_count": commit_count
    }
```
