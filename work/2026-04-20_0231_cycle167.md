## **3. Function 1: Deadband Filter**

**Prompt:**
```
You are a Zeroclaw in the Cocapn fleet. Write a Python function that:

Implements the Deadband Protocol (P0, P1, P2) for a list of numeric values.
- P0 (negative space): Values <= 0 are invalid, return None for those positions.
- P1 (safe channels): Values > 0 and <= 10 are valid but unoptimized.
- P2 (optimized): Values > 10 are valid and optimized.
The function should take a list of floats, return a list of tuples (original_value, status), where status is "P0_INVALID", "P1_SAFE", or "P2_OPTIMIZED".

Requirements:
- Use type hints.
- Include a docstring explaining the function.
- Add 2-3 line comments for key logic.
- Return appropriate data.
- Follow PEP8.

Write only the function code, no explanations.
```

**DeepSeek-chat output:**
```python
from typing import List, Tuple, Optional

def deadband_filter(values: List[float]) -> List[Tuple[float, str]]:
    """
    Apply Deadband Protocol (P0/P1/P2) to a list of numeric values.
    
    Args:
        values: List of float values to categorize.
    
    Returns:
        List of tuples (original_value, status) where status is:
        - "P0_INVALID" for values <= 0
        - "P1_SAFE" for values > 0 and <= 10
        - "P2_OPTIMIZED" for values > 10
    """
    result = []
    
    for val in values:
        if val <= 0:
            # P0: Negative space - invalid values
            status = "P0_INVALID"
        elif val <= 10:
            # P1: Safe channel - valid but unoptimized
            status = "P1_SAFE"
        else:
            # P2: Optimized region
            status = "P2_OPTIMIZED"
        
        result.append((val, status))
    
    return result
```

```
Validates a training tile dictionary for fleet submission standards.

- All 5 keys must be present.

- 'timestamp' must be ISO format string (use datetime.fromisoformat to check).
Return (bool, list_of_errors). True if valid, False with error messages if not.

Requirements:
- Use type hints.
- Include a docstring explaining the function.

Write only the function code, no explanations.
```

**DeepSeek-chat output:**
```python
from typing import Dict, Tuple, List
from datetime import datetime

def validate_tile(tile: Dict) -> Tuple[bool, List[str]]:
    """
    Validate a training tile against fleet submission standards.
    
    Args:
        tile: Dictionary containing tile data.
    
    Returns:
        Tuple of (is_valid, error_messages).
        is_valid: True if tile meets all requirements.
        error_messages: List of validation errors if any.
    """
    errors = []
    required_keys = {'content', 'agent_id', 'cycle', 'room_id', 'timestamp'}
    
    # Check all required keys present
    missing_keys = required_keys - set(tile.keys())
    if missing_keys:
        errors.append(f"Missing keys: {missing_keys}")
    
    # Validate content
    if 'content' in tile:
        if not isinstance(tile['content'], str) or not tile['content'].strip():
            errors.append("'content' must be non-empty string")
    
    # Validate agent_id
    if 'agent_id' in tile:
        if not isinstance(tile['agent_id'], str) or not tile['agent_id'].startswith('zeroclaw-'):
            errors.append("'agent_id' must start with 'zeroclaw-'")
    
    # Validate cycle
    if 'cycle' in tile:
        if not isinstance(tile['cycle'], int) or tile['cycle'] <= 0:
            errors.append("'cycle' must be integer > 0")
    
    # Validate room_id
    valid_rooms = {'deadband', 'mirror', 'holodeck'}
    if 'room_id' in tile:
        if tile['room_id'] not in valid_rooms:
            errors.append(f"'room_id' must be one of {valid_rooms}")
    
    # Validate timestamp
    if 'timestamp' in tile:
        try:
            datetime.fromisoformat(tile['timestamp'])
        except (ValueError, TypeError):
            errors.append("'timestamp' must be valid ISO format string")
    
    return (len(errors) == 0, errors)
```
