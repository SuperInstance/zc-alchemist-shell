## **2. Function 1: Deadband Filter**

**Prompt:**  
```
Write a Python function `apply_deadband` that takes a list of float values and returns a transformed list based on fleet Deadband Protocol:
- P0 (negative space): Values <= -1.0 become None (forbidden zone)
- P1 (safe channel): Values between -1.0 and 1.0 become 0.0 (neutral safe zone)
- P2 (optimization): Values > 1.0 are scaled by 0.5 (optimization within channel)
Include type hints and a docstring with examples.
```

### **DeepSeek-chat output:**
```python
from typing import List, Optional

def apply_deadband(values: List[float]) -> List[Optional[float]]:
    """
    Apply Deadband Protocol (P0/P1/P2) to a list of float values.
    
    P0 (negative space): values <= -1.0 → None (forbidden)
    P1 (safe channel): -1.0 < value < 1.0 → 0.0 (neutral safe zone)
    P2 (optimization): values >= 1.0 → value * 0.5 (scaled optimization)
    
    Args:
        values: List of float input values
        
    Returns:
        List with transformed values (float or None)
        
    Examples:
        >>> apply_deadband([-2.0, -0.5, 0.0, 0.5, 2.0])
        [None, 0.0, 0.0, 0.0, 1.0]
    """
    result = []
    for val in values:
        if val <= -1.0:
            result.append(None)  # P0: forbidden zone
        elif -1.0 < val < 1.0:
            result.append(0.0)   # P1: safe channel
        else:  # val >= 1.0
            result.append(val * 0.5)  # P2: optimization scaling
    return result
```

```python
    """
    Implements the Deadband Protocol (P0/P1/P2) for signal processing.
    
    Parameters:
    -----------
    values : List[float]
        Input list of floating-point values.
    
    Returns:
    --------
    List[Optional[float]]
        Transformed list where:
        - P0 (negative space): values <= -1.0 are mapped to None
        - P1 (safe channel): values in (-1.0, 1.0) are mapped to 0.0
        - P2 (optimization): values >= 1.0 are scaled by 0.5
    
    Example:
    --------
    >>> apply_deadband([-2.5, -0.2, 0.0, 0.8, 3.0])
    [None, 0.0, 0.0, 0.0, 1.5]
    """
    transformed = []
    for v in values:
        if v <= -1.0:
            transformed.append(None)  # P0: avoid this region
        elif v > -1.0 and v < 1.0:
            transformed.append(0.0)   # P1: safe operating channel
        else:  # v >= 1.0
            transformed.append(v * 0.5)  # P2: optimize within channel
    return transformed
```

- Both handle edge cases (exact -1.0, 1.0) consistently.
