## **2. Function 1: `deadband_filter`**

**Prompt:**
"Write a Python function `deadband_filter(values, deadband=0.1)`. It implements the Cocapn fleet's Deadband Protocol (P0: avoid, P1: safe channel, P2: optimize). Given a list of floats `values`, it should return a new list where any value whose absolute value is less than `deadband` is set to 0.0 (P0: map negative space). All other values should be passed through unchanged. Include a docstring and a simple example in a `if __name__ == "__main__":` block."

### **DeepSeek-chat Output:**
```python
def deadband_filter(values, deadband=0.1):
    """
    Apply the Deadband Protocol (P0) to a list of numeric values.
    
    Parameters:
    values (list of float): Input list of values.
    deadband (float): Threshold for the deadband. Absolute values less than this are mapped to zero.
    
    Returns:
    list of float: Filtered list where |value| < deadband -> 0.0, else value.
    """
    # P0: Map negative space (what NOT to do) - values within deadband are nullified.
    filtered = []
    for v in values:
        if abs(v) < deadband:
            filtered.append(0.0)
        else:
            filtered.append(v)
    return filtered
```

## **3. Function 2: `parse_fleet_log`**

**Prompt:**
"Write a Python function `parse_fleet_log(log_line)` that parses a semi-structured fleet log line and returns a dictionary. Log format: `'[TIMESTAMP] AGENT@CYCLE - MESSAGE'` Example: `'[2026-04-19T23:00:00Z] Alchemist@90 - Submitted tile 4 for review.'` The function should extract `timestamp` (as string), `agent`, `cycle` (as int), and `message`. Assume the format is always valid. Use regex for extraction. Include a docstring."

### **DeepSeek-chat Output:**
```python
import re

def parse_fleet_log(log_line):
    """
    Parse a fleet log line into structured components.
    
    Format: [TIMESTAMP] AGENT@CYCLE - MESSAGE
    
    Parameters:
    log_line (str): A single log line.
    
    Returns:
    dict: Keys: 'timestamp', 'agent', 'cycle', 'message'.
    """
    pattern = r'\[([^\]]+)\]\s+([^@]+)@(\d+)\s+-\s+(.+)'
    match = re.match(pattern, log_line)
    if not match:
        return {}  # Or raise an error, but prompt says assume valid
    
    timestamp, agent, cycle, message = match.groups()
    return {
        'timestamp': timestamp,
        'agent': agent.strip(),
        'cycle': int(cycle),
        'message': message.strip()
    }

# Example use (not in main block as not requested, but for testing):
if __name__ == "__main__":
    test_log = "[2026-04-19T23:00:00Z] Alchemist@90 - Submitted tile 4 for review."
    parsed = parse_fleet_log(test_log)
    print(parsed)
```
