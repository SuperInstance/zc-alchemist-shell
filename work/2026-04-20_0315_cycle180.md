## **2. Function 1: `deadband_filter`**

**Prompt to both models:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

The function takes a list of floats `values` and three threshold parameters: `p0_threshold`, `p1_low`, `p1_high`.

P0 (Negative Space): Any value where abs(value) < p0_threshold is considered "rock" and must be mapped to 0.0.
P1 (Safe Channel): Any value between p1_low and p1_high (inclusive) is in the safe channel and should be passed through unchanged.
P2 (Optimize): Any value outside the P1 channel but not a P0 rock should be clamped to the nearest P1 boundary (p1_low or p1_high).

The protocol is strict priority: P0 overrides P1, P1 overrides P2.

Return a new list of processed values.

Include a docstring and example usage.
```

### **DeepSeek-chat Output:**
```python
def deadband_filter(values, p0_threshold, p1_low, p1_high):
    """
    Apply the Cocapn fleet Deadband Protocol (P0/P1/P2) to a list of values.

    Parameters:
    values (list of float): Input values to process.
    p0_threshold (float): Threshold for P0 (negative space). |value| < p0_threshold -> 0.0.
    p1_low (float): Lower bound of P1 safe channel.
    p1_high (float): Upper bound of P1 safe channel.

    Returns:
    list of float: Processed values after applying P0, P1, P2 in strict priority.
    """
    result = []
    for v in values:
        # P0: Map negative space (rocks) to zero
        if abs(v) < p0_threshold:
            result.append(0.0)
            continue

        # P1: Check if in safe channel
        if p1_low <= v <= p1_high:
            result.append(v)
            continue

        # P2: Clamp to nearest P1 boundary
        if v < p1_low:
            result.append(p1_low)
        else:  # v > p1_high
            result.append(p1_high)
    return result
```

```python
        if val < p1_low:
            filtered_values.append(p1_low)
        else:  # val > p1_high
            filtered_values.append(p1_high)
```
