## **2. Function 1: `deadband_filter`**

**Prompt to Models:**
```
Write a Python function `deadband_filter(data, lower_bound, upper_bound)`.
It applies a deadband filter to input `data`. `data` can be a single number (int/float) or a list/array of numbers.
If a value is strictly between `lower_bound` and `upper_bound` (exclusive), it is set to 0.
Otherwise, the value passes through unchanged.
Return the filtered data in the same type/shape as the input.
Include a docstring.
```

### **DeepSeek-chat Output:**
```python
def deadband_filter(data, lower_bound, upper_bound):
    """
    Apply a deadband filter to input data.

    Parameters:
    data: int, float, or list/array of numbers.
    lower_bound: Lower threshold.
    upper_bound: Upper threshold.

    Returns:
    Filtered data where values inside (lower_bound, upper_bound) are set to 0.
    """
    import numpy as np

    # Ensure bounds are ordered
    if lower_bound > upper_bound:
        lower_bound, upper_bound = upper_bound, lower_bound

    # Handle single value
    if isinstance(data, (int, float)):
        if lower_bound < data < upper_bound:
            return 0
        return data

    # Handle list or array
    if isinstance(data, list):
        result = []
        for val in data:
            if lower_bound < val < upper_bound:
                result.append(0)
            else:
                result.append(val)
        return result
    elif isinstance(data, np.ndarray):
        # Vectorized operation for efficiency
        result = data.copy()
        mask = (result > lower_bound) & (result < upper_bound)
        result[mask] = 0
        return result
    else:
        raise TypeError("Input data must be int, float, list, or numpy array.")
```

```python
def deadband_filter(data, lower_bound, upper_bound):
    """
    Apply a deadband filter to the input data.

    Args:
        data: A numeric scalar or a sequence (list, tuple, numpy array) of numeric values.
        lower_bound: The lower bound of the deadband.
        upper_bound: The upper bound of the deadband.

    Returns:
        The filtered data with values inside the deadband set to zero.
        The output type matches the input type (scalar or sequence).
    """
    import numpy as np

    # Swap bounds if they are reversed
    if lower_bound > upper_bound:
        lower_bound, upper_bound = upper_bound, lower_bound

    # Handle scalar input
    if isinstance(data, (int, float, np.number)):
        if lower_bound < data < upper_bound:
            return 0
        else:
            return data

    # Handle sequence input
    # Convert to numpy array for uniform handling, but preserve original type if possible
    arr = np.asarray(data)
    filtered = arr.copy()
    mask = (filtered > lower_bound) & (filtered < upper_bound)
    filtered[mask] = 0

    # Convert back to original type
    if isinstance(data, list):
        return filtered.tolist()
    elif isinstance(data, tuple):
        return tuple(filtered.tolist())
    else:
        # Assume it's a numpy array or something compatible
        return filtered
```

### **DeepSeek-chat Output:**
```python
import re

def parse_fleet_log_line(line):
    """
    Parse a standardized fleet log line.

    Format: "TIMESTAMP [LEVEL] component message"
    Example: "2026-04-20T03:45:12Z [INFO] plato-torch Room initialized"

    Returns:
        dict with keys: timestamp, level, component, message
        or None if line doesn't match.
    """
    pattern = r'^(\S+)\s+\[(\w+)\]\s+(\S+)\s+(.+)$'
    match = re.match(pattern, line.strip())
    if not match:
        return None

    timestamp, level, component, message = match.groups()
    return {
        'timestamp': timestamp,
        'level': level,
        'component': component,
        'message': message
    }
```
