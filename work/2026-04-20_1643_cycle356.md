### **Function 1: `validate_deadband_protocol`**

**Prompt:** "Write a Python function called `validate_deadband_protocol` that takes a list of action strings. Each action string starts with 'P0:', 'P1:', or 'P2:'. The function must validate that the sequence follows the Deadband Protocol: all P0 actions must come before any P1, and all P1 before any P2. Return a tuple where the first element is a boolean (True if valid, False if violation) and the second element is a list of strings describing any violations found (empty list if valid). Include a docstring."

**DeepSeek-chat Output:**
```python
def validate_deadband_protocol(action_sequence):
    """
    Validates a sequence of actions against the Deadband Protocol.

    The Deadband Protocol requires strict priority ordering:
    P0 (negative space mapping) > P1 (safe channel finding) > P2 (optimization).
    In a sequence, all P0 actions must precede any P1 actions, and all P1 actions
    must precede any P2 actions.

    Args:
        action_sequence (list of str): A list of action strings, each prefixed
                                        with 'P0:', 'P1:', or 'P2:'.

    Returns:
        tuple: (is_valid (bool), violations (list of str))
               violations contains human-readable descriptions of any protocol breaches.
    """
    violations = []
    # Track the highest priority phase we've seen
    current_phase = 'P0'
    phase_order = {'P0': 0, 'P1': 1, 'P2': 2}

    for i, action in enumerate(action_sequence):
        if not (action.startswith('P0:') or action.startswith('P1:') or action.startswith('P2:')):
            violations.append(f"Action at index {i} '{action}' does not start with a valid phase prefix (P0:/P1:/P2:).")
            continue

        # Extract phase from action
        action_phase = action.split(':')[0]

        # Check if we're moving backwards in phase order
        if phase_order[action_phase] < phase_order[current_phase]:
            violations.append(
                f"Protocol violation at index {i}: '{action_phase}' action '{action}' "
                f"appears after '{current_phase}' actions, which is not allowed."
            )
        # Update current_phase if we've moved forward
        elif phase_order[action_phase] > phase_order[current_phase]:
            current_phase = action_phase

    is_valid = len(violations) == 0
    return (is_valid, violations)
```

### **Function 2: `parse_fleet_tile`**

**Prompt:** "Write a Python function `parse_fleet_tile` that takes a string `tile_text` (content of a fleet training tile). Parse it to extract: 1. cycle number (looks for 'Cycle' followed by a number), 2. agent role (looks for a line starting with '**Role:**'), 3. phase (looks for 'Phase' followed by a number), 4. a list of tasks (lines that start with '- ' or '* ' after a 'Tasks:' section). Return a dict with these keys. If a field is not found, use None. Use the re module for robust extraction. Include error handling."

**DeepSeek-chat Output:**
```python
import re

def parse_fleet_tile(tile_text):
    """
    Parses a raw fleet training tile text to extract structured metadata.

    Args:
        tile_text (str): The full text content of a tile.

    Returns:
        dict: A dictionary with keys:
            - 'cycle': int or None
            - 'role': str or None
            - 'phase': int or None
            - 'tasks': list of str (empty list if none found)
    """
    result = {
        'cycle': None,
        'role': None,
        'phase': None,
        'tasks': []
    }

    # Extract cycle number (e.g., "Cycle 356" or "Cycle: 123")
    cycle_match = re.search(r'Cycle[:\s]*(\d+)', tile_text, re.IGNORECASE)
    if cycle_match:
        try:
            result['cycle'] = int(cycle_match.group(1))
        except ValueError:
            pass  # Keep as None if conversion fails

    # Extract agent role (e.g., "**Role:** Alchemist")
    role_match = re.search(r'\*\*Role:\*\*\s*(\w+)', tile_text)
    if role_match:
        result['role'] = role_match.group(1)

    # Extract phase (e.g., "Phase 4" or "Phase: 2")
    phase_match = re.search(r'Phase[:\s]*(\d+)', tile_text, re.IGNORECASE)
    if phase_match:
        try:
            result['phase'] = int(phase_match.group(1))
        except ValueError:
            pass

    # Extract tasks: Find a "Tasks:" section, then capture bullet points
    # This is a simple approach; a more robust parser would handle markdown.
    lines = tile_text.split('\n')
    in_task_section = False
    for line in lines:
        stripped = line.strip()
        # Start of task section
        if re.match(r'^#*\s*Tasks?:', stripped, re.IGNORECASE):
            in_task_section = True
