### **Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that applies the Deadband Protocol to a list of float values.

The Deadband Protocol has three priority levels:
P0: Map negative space (what NOT to do) — here, filter out values that are NaN or infinite.
P1: Find safe channels (where you CAN be) — here, keep only values within a specified safe range [min_val, max_val].
P2: Optimize within channels (best path) — here, if the filtered list is non-empty, return the value closest to a target; otherwise return None.

Parameters:
- values: List[float]
- min_val: float
- max_val: float
- target: float

Return: Optional[float]

Include type hints and a docstring. Use only standard library.
```

**DeepSeek-chat output:**
```python
from typing import List, Optional
import math

def deadband_filter(values: List[float], min_val: float, max_val: float, target: float) -> Optional[float]:
    """
    Apply the Deadband Protocol to filter and select a value.

    P0: Remove NaN and infinite values.
    P1: Keep only values within [min_val, max_val].
    P2: Return the value closest to target from the filtered list, or None if empty.

    Args:
        values: List of input float values.
        min_val: Minimum allowed value for safe channel.
        max_val: Maximum allowed value for safe channel.
        target: Target value for optimization.

    Returns:
        The value closest to target from the filtered list, or None.
    """
    # P0: Map negative space - remove non-finite values
    finite_vals = []
    for v in values:
        if not math.isfinite(v):
            continue
        finite_vals.append(v)

    # P1: Find safe channels - keep values within range
    safe_vals = [v for v in finite_vals if min_val <= v <= max_val]

    # P2: Optimize within channels - find closest to target
    if not safe_vals:
        return None

    closest = safe_vals[0]
    min_dist = abs(closest - target)
    for v in safe_vals[1:]:
        dist = abs(v - target)
        if dist < min_dist:
            min_dist = dist
            closest = v

    return closest
```

```python
import math

def deadband_filter(values: List[float], min_val: float, max_val: float, target: float) -> Optional[float]:
    """
    Applies the Deadband Protocol to a list of float values.

    The protocol:
    1. P0 (Negative Space): Filter out non-finite values (NaN, inf).
    2. P1 (Safe Channels): Keep only values within [min_val, max_val].
    3. P2 (Optimization): From the remaining values, select the one closest to `target`.

    Args:
        values: A list of float values.
        min_val: The minimum value of the safe channel.
        max_val: The maximum value of the safe channel.
        target: The target value for optimization.

    Returns:
        The value from the filtered list that is closest to `target`, or None if the filtered list is empty.
    """
    # P0: Remove non-finite values
    filtered = [v for v in values if math.isfinite(v)]

    # P1: Keep values within safe channel
    filtered = [v for v in filtered if min_val <= v <= max_val]

    # P2: Find value closest to target
    if not filtered:
        return None

    # Use min with key function to find closest
    return min(filtered, key=lambda x: abs(x - target))
```

```python
from typing import Optional, Dict, Any

def parse_fleet_log(log_line: str) -> Optional[Dict[str, Any]]:
    """
    Parse a structured fleet log line into a dictionary.

    Expected format: "YYYY-MM-DD HH:MM:SS | AGENT | CYCLE | STATUS | MESSAGE"

    Args:
        log_line: A log line string.

    Returns:
        Dictionary with keys: timestamp, agent, cycle, status, message.
        Returns None if format is invalid.
    """
    parts = log_line.split(" | ")
    if len(parts) != 5:
        return None

    timestamp, agent, cycle_str, status, message = parts

    try:
        cycle = int(cycle_str)
    except ValueError:
        return None

    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message
    }
```
