## **2. Function 1: Deadband Filter**

**Prompt:**
```
Write a Python function `apply_deadband_protocol` that takes a list of floats `sensor_readings` and returns a list of strings.

Doctrine:
- P0 (negative space): If reading <= -10.0, return "P0: ALARM"
- P1 (safe channel): If -10.0 < reading < 10.0, return "P1: SAFE"
- P2 (optimize): If reading >= 10.0, return "P2: OPTIMIZE"

Edge cases: empty list, None input, non‑float values.
Include docstring and example.
```

### **DeepSeek-chat output:**
```python
def apply_deadband_protocol(sensor_readings):
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of sensor readings.

    Parameters
    ----------
    sensor_readings : list of float or None
        List of sensor values. Can be empty or None.

    Returns
    -------
    list of str
        List of protocol labels for each reading.

    Examples
    --------
    >>> apply_deadband_protocol([-15.0, -5.0, 0.0, 5.0, 15.0])
    ['P0: ALARM', 'P1: SAFE', 'P1: SAFE', 'P1: SAFE', 'P2: OPTIMIZE']
    """
    if sensor_readings is None:
        return []

    results = []
    for val in sensor_readings:
        try:
            num = float(val)
        except (TypeError, ValueError):
            results.append("ERROR: Invalid reading")
            continue

        if num <= -10.0:
            results.append("P0: ALARM")
        elif num < 10.0:
            results.append("P1: SAFE")
        else:  # num >= 10.0
            results.append("P2: OPTIMIZE")
    return results
```

### **Groq Llama-70b output:**
```python
def apply_deadband_protocol(sensor_readings):
    """
    Applies the Deadband Protocol to sensor readings.

    Args:
        sensor_readings (list): A list of numerical sensor readings.

    Returns:
        list: A list of strings indicating the protocol level for each reading.
    """
    if not sensor_readings:
        return []

    protocol_results = []
    for reading in sensor_readings:
        if reading is None:
            protocol_results.append("ERROR: None reading")
            continue
        try:
            reading_float = float(reading)
        except ValueError:
            protocol_results.append("ERROR: Invalid reading")
            continue

        if reading_float <= -10.0:
            protocol_results.append("P0: ALARM")
        elif reading_float < 10.0:
            protocol_results.append("P1: SAFE")
        else:
            protocol_results.append("P2: OPTIMIZE")

    return protocol_results
```

### **Comparison:**
- **Correctness:** Both handle edge cases (None, non‑float). DeepSeek explicitly checks `sensor_readings is None`; Groq uses `if not sensor_readings` which treats `None` as falsy but would also treat empty list the same—acceptable.
- **Clarity:** DeepSeek’s docstring includes a concrete example, which is useful. Groq’s docstring is more concise. Both follow PEP8.
- **Robustness:** DeepSeek catches `TypeError` (e.g., if val is a dict) in addition to `ValueError`. Groq only catches `ValueError`. DeepSeek’s approach is slightly more defensive.
- **Fleet alignment:** Both correctly implement P0/P1/P2 thresholds.

**Verdict:** DeepSeek’s version is marginally more robust due to broader error handling and the example. Both are production‑ready.

## **3. Function 2: Training Tile Validator**

**Prompt:**
```
Write a Python function `validate_training_tile` that checks if a dictionary `tile` meets fleet submission standards.

Required keys: `title`, `content`, `author`, `timestamp`, `tags`.
`content` must be non‑empty string, length between 50 and 5000 chars.
`tags` must be a list of strings, each tag length 1‑20 chars, max 10 tags.
`timestamp` must be an ISO‑8601 string (e.g., "2026‑04‑20T10:00:00Z").

Return a tuple (bool, list_of_errors).
Include validation for extra keys (only allowed keys are the required ones).
```

### **DeepSeek-chat output:**
```python
import re
from datetime import datetime

def validate_training_tile(tile):
    """
    Validate a training tile against fleet submission standards.

    Parameters
    ----------
    tile : dict
        The tile candidate.

    Returns
    -------
    (bool, list)
        True if valid, False otherwise, and a list of error messages.
    """
    if not isinstance(tile, dict):
        return False, ["Tile must be a dictionary"]

    required_keys = {"title", "content", "author", "timestamp", "tags"}
    allowed_keys = required_keys  # no extra keys allowed

    # Check for missing or extra keys
    tile_keys = set(tile.keys())
    missing = required_keys - tile_keys
    extra = tile_keys - allowed_keys

    errors = []
    if missing:
        errors.append(f"Missing keys: {sorted(missing)}")
    if extra:
        errors.append(f"Extra keys not allowed: {sorted(extra)}")

    # Validate each field if present
    if "title" in tile:
        if not isinstance(tile["title"], str) or not tile["title"].strip():
            errors.append("Title must be a non‑empty string")

    if "content" in tile:
        content = tile["content"]
        if not isinstance(content, str):
            errors.append("Content must be a string")
        else:
            if len(content) < 50:
                errors.append("Content too short (min 50 chars)")
            elif len(content) > 5000:
                errors.append("Content too long (max 5000 chars)")

    if "author" in tile:
        if not isinstance(tile["author"], str) or not tile["author"].strip():
            errors.append("Author must be a non‑empty string")

    if "timestamp" in tile:
        ts = tile["timestamp"]
        if not isinstance(ts, str):
            errors.append("Timestamp must be a string")
        else:
            try:
                datetime.fromisoformat(ts.replace('Z', '+00:00'))
            except ValueError:
                errors.append("Timestamp must be valid ISO‑8601 format")

    if "tags" in tile:
        tags = tile["tags"]
        if not isinstance(tags, list):
            errors.append("Tags must be a list")
        else:
            if len(tags) > 10:
                errors.append("Too many tags (max 10)")
            for i, tag in enumerate(tags):
                if not isinstance(tag, str):
                    errors.append(f"Tag {i} must be a string")
                elif not (1 <= len(tag) <= 20):
                    errors.append(f"Tag {i} length must be 1‑20 chars")

    return (len(errors) == 0, errors)
```
