## **2. Function 1: `deadband_filter`**

**Spec:** Given a list of float values and a deadband threshold `epsilon`, return a new list where any value within `±epsilon` of zero is set to 0.0 (P0: avoid noise), values outside the deadband are kept (P1: safe channel), and optionally apply a simple scaling (P2: optimize) if a `scale` factor is provided.

### **2.A DeepSeek‑chat version**
```python
def deadband_filter(values, epsilon, scale=1.0):
    """
    Apply the Deadband Protocol to a list of values.
    
    P0: Values within ±epsilon of zero are forced to zero (negative space).
    P1: Values outside the deadband are kept (safe channel).
    P2: If scale != 1.0, apply scaling as a simple optimization.
    
    Args:
        values (list[float]): Input values.
        epsilon (float): Deadband threshold (>= 0).
        scale (float): Optional scaling factor for P2 optimization.
    
    Returns:
        list[float]: Filtered values.
    """
    if epsilon < 0:
        raise ValueError("epsilon must be non‑negative")
    
    result = []
    for v in values:
        if -epsilon <= v <= epsilon:
            # P0: map negative space → zero
            result.append(0.0)
        else:
            # P1: safe channel → keep original
            # P2: optional scaling
            result.append(v * scale)
    return result
```

### **2.B Groq Llama‑70b version**

```python
def deadband_filter(values, epsilon, scale=1.0):
    """
    Implements the Deadband Protocol: zero out small signals, scale others.
    
    Parameters:
    - values: list of floats
    - epsilon: deadband width (non‑negative)
    - scale: multiplier for values outside deadband
    
    Returns:
    - list of processed floats
    """
    if epsilon < 0:
        raise ValueError("Epsilon cannot be negative")
    
    filtered = []
    for val in values:
        if abs(val) <= epsilon:
            filtered.append(0.0)
        else:
            filtered.append(val * scale)
    return filtered
```

```python
        for v in values
    ]
```

## **3. Function 2: `parse_fleet_log`**

**Spec:** Parse a semi‑structured log line like:
`"2026‑04‑20 06:30 [WARN] JetsonClaw1 — GPU temp 85C (threshold 80)"`
Return a dict with keys: `timestamp`, `level`, `agent`, `message`, `metrics` (optional dict of extracted numbers).

### **3.A DeepSeek‑chat version**
```python
import re
from datetime import datetime

def parse_fleet_log(log_line):
    """
    Parse a fleet log line into structured fields.
    
    Expected format:
        YYYY‑MM‑DD HH:MM [LEVEL] Agent — Message
    
    Extracts numeric metrics from message if present.
    """
    pattern = r'(\d{4}\-\d{2}\-\d{2} \d{2}:\d{2}) \[(\w+)\] (\w+) — (.+)'
    match = re.match(pattern, log_line)
    if not match:
        return {"error": "format mismatch"}
    
    ts, level, agent, msg = match.groups()
    
    # Extract numbers (e.g., "85C", "80")
    metrics = {}
    num_pattern = r'(\d+(?:\.\d+)?)'
    numbers = re.findall(num_pattern, msg)
    if numbers:
        metrics = {f"value_{i}": float(n) for i, n in enumerate(numbers)}
    
    return {
        "timestamp": ts,
        "level": level,
        "agent": agent,
        "message": msg,
        "metrics": metrics
    }
```

### **3.B Groq Llama‑70b version**

```python
import re

def parse_fleet_log(line):
    """
    Parses a standardized fleet log entry.
    
    Format: date time [LEVEL] Agent — descriptive text
    Returns dictionary with parsed components.
    """
    regex = r'^(\d{4}\-\d{2}\-\d{2} \d{2}:\d{2}) \[(\w+)\] (\w+) — (.+)$'
    m = re.match(regex, line)
    if not m:
        return {"error": "Invalid log format"}
    
    ts, lvl, agent, msg = m.groups()
    
    # find all numbers in the message
    nums = re.findall(r'\b\d+(?:\.\d+)?\b', msg)
    metrics = {f"metric_{idx}": float(val) for idx, val in enumerate(nums)}
    
    return {
        "timestamp": ts,
        "level": lvl,
        "agent": agent,
        "message": msg,
        "metrics": metrics
    }
```
