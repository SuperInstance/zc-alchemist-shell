### **Function 1: `validate_deadband_protocol`**

**Prompt to Models:**
"Write a Python function called `validate_deadband_protocol` that takes a list of strings as input. The strings can only be "P0", "P1", or "P2". Return a tuple: (bool, str). The bool is True if valid, False if invalid. The str is an empty string if valid, or a specific error message if invalid (e.g., "Skipped P1", "Invalid sequence order", "Repeated P0 after P1 started"). #### **DeepSeek-chat (My Implementation):**
```python
def validate_deadband_protocol(priority_sequence):
    """
    Validates if a sequence of priority strings follows the Deadband Protocol.

    Rules:
    1. Only "P0", "P1", "P2" are allowed.
    2. Sequence must be strictly P0 -> P1 -> P2.
    3. No priority can be skipped (e.g., cannot go P0 -> P2).
    4. Once a higher priority is started, a lower priority cannot reappear
       (e.g., P0, P1, P0 is invalid).

    Args:
        priority_sequence (list[str]): List of priority strings.

    Returns:
        tuple[bool, str]: (is_valid, error_message). error_message is empty if valid.
    """
    allowed = {"P0", "P1", "P2"}
    expected_order = ["P0", "P1", "P2"]
    current_stage_index = -1  # Index in expected_order of the last valid priority seen

    for i, priority in enumerate(priority_sequence):
        # Check if priority is allowed
        if priority not in allowed:
            return False, f"Invalid priority '{priority}' at position {i}"

        # Find index in expected order
        priority_index = expected_order.index(priority)

        # Rule: Cannot skip a priority
        if priority_index > current_stage_index + 1:
            skipped = expected_order[current_stage_index + 1:priority_index]
            return False, f"Skipped priority(ies) {skipped} at position {i}"

        # Rule: Cannot go back to a lower priority once a higher one is started
        if priority_index < current_stage_index:
            return False, f"Repeated lower priority {priority} after {expected_order[current_stage_index]} at position {i}"

        # Update current stage
        current_stage_index = priority_index

    return True, ""
```

```python
    """
    Validate Deadband Protocol sequence.

    Parameters:
    sequence (list): List of strings like 'P0', 'P1', 'P2'

    Returns:
    tuple: (bool, str) where bool indicates validity, str has error message if any.
    """
    if not sequence:
        return True, "Empty sequence is valid"

    order = {"P0": 0, "P1": 1, "P2": 2}
    seen = set()
    max_seen = -1

    for idx, item in enumerate(sequence):
        if item not in order:
            return False, f"Invalid item {item} at index {idx}"

        current = order[item]

        # Check for repeats after progression
        if current < max_seen:
            return False, f"Priority regression: {item} after higher priority at index {idx}"

        # Check for skips
        if current > max_seen + 1:
            skipped = [p for p in ["P0", "P1", "P2"] if order[p] == max_seen + 1]
            return False, f"Skipped {skipped[0]} at index {idx}"

        max_seen = current
        seen.add(item)

    return True, ""
```
