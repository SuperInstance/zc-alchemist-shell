## **2. Prompt Used**

```
You are a Python expert. Write a clean, well-documented function for the following task.

### Function 1: deadband_filter
Implement the Fleet's Deadband Protocol (P0, P1, P2).
- Input: A list of floats `values` and a float `threshold`.
- P0 (Negative Space): Identify values where abs(value) < threshold. These are "rocks" – return None for them.
- P1 (Safe Channels): For values where abs(value) >= threshold, identify their sign (positive or negative channel).
- P2 (Optimize): Within each channel (positive/negative), return the value with the largest magnitude.
- Return: A tuple: (filtered_positive_value, filtered_negative_value). If a channel has no valid values, return None for that channel.
Example: values=[-5, -1, 2, 10], threshold=2 -> P0 removes -1 and 2 (abs<2). P1 channels: negative:[-5], positive:[10]. P2: max magnitude in each -> (-5, 10).

### Function 2: parse_fleet_log
Parse a structured log line from the Cocapn fleet.
- Input: A string `log_line` with format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE".
- Return: A dictionary with keys: timestamp, agent, cycle, status, message.
- The fields are separated by " | ". Handle extra whitespace.
- If the line does not match the expected format, return a dict with a single key "error" and the original line as the value.

### Function 3: constraint_snap
A simple 2D geometric snapping function.
- Input: A point `(x, y)` as a tuple, and a list of reference points `[(x1, y1), (x2, y2), ...]`.
- Also input a `tolerance` float.
- Find the reference point within the tolerance (Euclidean distance).
- If exactly one reference point is within tolerance, "snap" the input point to those coordinates.
- If zero or multiple points are within tolerance, return the original point unchanged.
- Return: The snapped (or original) point as a tuple (x, y).

Provide only the Python code for these three functions.

### **DeepSeek-chat Output**

def deadband_filter(values, threshold):
    """
    Apply Deadband Protocol (P0, P1, P2) to filter values.
    
    Args:
        values: List of float values
        threshold: Minimum absolute value to pass P0
        
    Returns:
        Tuple (max_positive, max_negative) where each can be None
    """
    # P0: Identify rocks (values inside deadband)
    safe_values = [v for v in values if abs(v) >= threshold]
    
    # P1: Separate channels
    positive_channel = [v for v in safe_values if v > 0]
    negative_channel = [v for v in safe_values if v < 0]
    
    # P2: Optimize within channels
    max_positive = max(positive_channel) if positive_channel else None
    max_negative = min(negative_channel) if negative_channel else None  # Most negative = largest magnitude
    
    return (max_positive, max_negative)


def parse_fleet_log(log_line):
    """
    Parse a structured fleet log line.
    
    Args:
        log_line: String in format "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"
        
    Returns:
        Dictionary with parsed fields or error dict
    """
    # Split by pipe and strip whitespace
    parts = [part.strip() for part in log_line.split('|')]
    
    # Check format
    if len(parts) != 5:
        return {'error': log_line}
    
    return {
        'timestamp': parts[0],
        'agent': parts[1],
        'cycle': parts[2],
        'status': parts[3],
        'message': parts[4]
    }


def constraint_snap(point, reference_points, tolerance):
    """
    Snap a point to the nearest reference point within tolerance.
    
    Args:
        point: Tuple (x, y)
        reference_points: List of tuples [(x1, y1), ...]
        tolerance: Maximum distance for snapping
        
    Returns:
        Snapped point as tuple (x, y)
    """
    x, y = point
    candidates = []
    
    for ref_x, ref_y in reference_points:
        distance = ((x - ref_x) ** 2 + (y - ref_y) ** 2) ** 0.5
        if distance <= tolerance:
            candidates.append((ref_x, ref_y))
    
    # Snap only if exactly one candidate
    if len(candidates) == 1:
        return candidates[0]
    return point
```

def deadband_filter(values, threshold):
    positive_candidates = []
    negative_candidates = []
    for v in values:
        if abs(v) < threshold:
            continue  # P0: skip rocks
        if v > 0:
            positive_candidates.append(v)
        else:
            negative_candidates.append(v)
    # P2: optimize
    pos = max(positive_candidates) if positive_candidates else None
    neg = min(negative_candidates) if negative_candidates else None
    return pos, neg

def parse_fleet_log(log_line):
    try:
        timestamp, agent, cycle, status, message = [s.strip() for s in log_line.split('|')]
        return {
            'timestamp': timestamp,
            'agent': agent,
            'cycle': cycle,
            'status': status,
            'message': message
        }
    except ValueError:
        return {'error': log_line}

def constraint_snap(point, reference_points, tolerance):
    import math
    x, y = point
    snap_candidate = None
    count = 0
    for rx, ry in reference_points:
        dist = math.sqrt((x - rx)**2 + (y - ry)**2)
        if dist <= tolerance:
            snap_candidate = (rx, ry)
            count += 1
    if count == 1:
        return snap_candidate
    return point
