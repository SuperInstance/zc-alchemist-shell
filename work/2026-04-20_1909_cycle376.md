def deadband_filter(actions, deadband_threshold=0.1):
    """
    Apply the Cocapn fleet's Deadband Protocol to a list of action values.

    P0 (Negative Space): Remove any action where abs(value) < deadband_threshold.
    P1 (Safe Channels): Keep only actions where value is within [-1.0, 1.0].
    P2 (Optimize): For remaining actions, apply a simple normalization: value / max(abs(values)).

    Protocol is strict priority: P0 -> P1 -> P2. Never skip a step.

    Args:
        actions (list of float): Raw action values.
        deadband_threshold (float): Minimum magnitude to survive P0.

    Returns:
        list of float: Filtered and optimized actions.
        list of int: Indices of the original actions that survived P0.
    """
    # P0: Map negative space
    p0_indices = []
    p0_values = []
    for idx, val in enumerate(actions):
        if abs(val) >= deadband_threshold:
            p0_indices.append(idx)
            p0_values.append(val)

    # P1: Find safe channels
    p1_values = []
    p1_indices = []
    for idx, val in zip(p0_indices, p0_values):
        if -1.0 <= val <= 1.0:
            p1_indices.append(idx)
            p1_values.append(val)

    # P2: Optimize within channels
    if not p1_values:
        return [], []
    max_abs = max(abs(v) for v in p1_values)
    p2_values = [v / max_abs for v in p1_values]

    return p2_values, p1_indices
```

        deadband_threshold (float): Minimum magnitude to survive P0.

    Returns:
        list of float: Filtered and optimized actions.
        list of int: Indices of the original actions that survived P0.
    """
    # P0: Filter out values below deadband threshold
    filtered = [(i, a) for i, a in enumerate(actions) if abs(a) >= deadband_threshold]
    if not filtered:
        return [], []

    indices, vals = zip(*filtered)

    # P1: Constrain to safe range [-1, 1]
    safe_vals = []
    safe_indices = []
    for i, v in zip(indices, vals):
        if v < -1.0:
            v = -1.0
        elif v > 1.0:
            v = 1.0
        safe_vals.append(v)
        safe_indices.append(i)

    # P2: Normalize by maximum absolute value
    max_val = max(abs(v) for v in safe_vals)
    optimized = [v / max_val for v in safe_vals]

    return optimized, safe_indices
```

### **Comparison for Function 1:**
*   **Correctness:** Both implement the three priority steps. DeepSeek's P1 is a strict filter (`if -1.0 <= val <= 1.0`). Llama's P1 is a clamp (`if v < -1.0: v = -1.0`). The spec says "keep only actions where value is within [-1.0, 1.0]" – DeepSeek's interpretation is more literal. Llama's clamp changes the data, which might not be desired.
*   **Style:** DeepSeek uses explicit, separate loops for each phase, making the priority clear. Llama uses list comprehensions and `zip`, which is concise but slightly less readable for the multi-step logic.
*   **Edge Cases:** Both handle empty lists. DeepSeek returns empty lists immediately after P1 if none survive. Llama does it after P0.
*   **Fleet Doctrine Alignment:** DeepSeek's strict filtering in P1 better matches the "find safe channels (where you CAN be)" philosophy—clamping forces a value into the channel. The fleet protocol might prefer filtering.
