## **2. Function 1: `validate_deadband_protocol`**

**Prompt:**
```
Write a Python function `validate_deadband_protocol(steps: list[str]) -> tuple[bool, str]`.
The function validates if a sequence of steps follows the Deadband Protocol.
Rules:
- Valid step strings are only "P0", "P1", "P2".
- The sequence must respect strict priority: P0 steps must come before any P1, and P1 before any P2.
- If a step is invalid or the priority is violated, return (False, "error message").
- If valid, return (True, "").
Examples:
- ["P0", "P1", "P2"] -> (True, "")
- ["P0", "P2"] -> (False, "Priority violation: P2 found before P1")
- ["P0", "P1", "P0"] -> (False, "Priority violation: P0 found after P1")
- ["P0", "PX"] -> (False, "Invalid step: PX")
```

### **DeepSeek-chat Output:**
```python
def validate_deadband_protocol(steps: list[str]) -> tuple[bool, str]:
    valid_steps = {"P0", "P1", "P2"}
    seen_p1 = False
    seen_p2 = False
    
    for i, step in enumerate(steps):
        if step not in valid_steps:
            return False, f"Invalid step: {step}"
        
        if step == "P0":
            if seen_p1 or seen_p2:
                return False, f"Priority violation: P0 found after P1 or P2 at index {i}"
        elif step == "P1":
            seen_p1 = True
            if seen_p2:
                return False, f"Priority violation: P1 found after P2 at index {i}"
        elif step == "P2":
            seen_p2 = True
    
    return True, ""
```

### **Groq Llama-70b Output:**
```python
def validate_deadband_protocol(steps: list[str]) -> tuple[bool, str]:
    allowed = {"P0", "P1", "P2"}
    p1_started = False
    p2_started = False
    
    for idx, step in enumerate(steps):
        if step not in allowed:
            return False, f"Invalid step: {step}"
        
        if step == "P0":
            if p1_started or p2_started:
                return False, f"Priority violation: P0 appears after P1/P2 at position {idx}"
        elif step == "P1":
            if p2_started:
                return False, f"Priority violation: P1 appears after P2 at position {idx}"
            p1_started = True
        elif step == "P2":
            p2_started = True
    
    return True, ""
```

## **3. Function 2: `parse_fleet_tile`**

**Prompt:**
```
Write a Python function `parse_fleet_tile(raw_text: str) -> dict`.
The raw_text format is:
```
agent: Alchemist
cycle: 112
content: This is the main content of the tile.
tags: code, test, benchmark
```
Rules:
- Parse lines to extract "agent", "cycle", "content", "tags".
- The "tags" line may be missing. If present, split by comma and strip spaces.
- The "content" may span multiple lines (until end or next field).
- If "cycle" is missing or not an integer, default to 0.
- If "agent" is missing, default to "unknown".
- Return a dict with keys: "agent" (str), "cycle" (int), "content" (str), "tags" (list[str]).
- Assume fields can appear in any order.
```

### **DeepSeek-chat Output:**
```python
def parse_fleet_tile(raw_text: str) -> dict:
    lines = raw_text.strip().split('\n')
    data = {
        "agent": "unknown",
        "cycle": 0,
        "content": "",
        "tags": []
    }
    
    i = 0
    while i < len(lines):
        line = lines[i].strip()
        if line.startswith("agent:"):
            data["agent"] = line.split(":", 1)[1].strip()
        elif line.startswith("cycle:"):
            try:
                data["cycle"] = int(line.split(":", 1)[1].strip())
            except ValueError:
                data["cycle"] = 0
        elif line.startswith("tags:"):
            tags_str = line.split(":", 1)[1].strip()
            if tags_str:
                data["tags"] = [tag.strip() for tag in tags_str.split(",")]
        elif line.startswith("content:"):
            # Collect multi-line content
            content_lines = []
            content_lines.append(line.split(":", 1)[1].strip())
            i += 1
            while i < len(lines) and not (lines[i].strip().startswith("agent:") or 
                                         lines[i].strip().startswith("cycle:") or 
                                         lines[i].strip().startswith("tags:")):
                content_lines.append(lines[i].strip())
                i += 1
            data["content"] = "\n".join(content_lines).strip()
            continue  # Skip the outer increment
        i += 1
    
    return data
```
