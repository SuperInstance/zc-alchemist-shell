## **3. Function 1: deadband_filter**

**Task Description:** Implement a deadband filter for sensor readings. Given a list of floats `values` and a deadband threshold `epsilon`, return a new list where any value within `±epsilon` of the previous *output* value is replaced by that previous output (holds its value). This mimics control systems deadband behavior. Start with first value as is.

**Input:** `values: List[float]`, `epsilon: float`
**Output:** `List[float]`
**Edge cases:** `epsilon < 0`, empty list, single element.

### **DeepSeek-chat output (simulated):**

```python
def deadband_filter(values: list[float], epsilon: float) -> list[float]:
    """
    Apply a deadband filter to a sequence of sensor readings.
    
    The Deadband Protocol (P0/P1/P2) inspires this: P0 avoids jitter,
    P1 finds stable channels, P2 optimizes within them.
    
    Args:
        values: List of sensor readings.
        epsilon: Deadband threshold. Must be >= 0.
    
    Returns:
        Filtered list where values within ±epsilon of previous output
        are replaced by that previous output.
    
    Example:
        >>> deadband_filter([1.0, 1.05, 1.15, 1.08], 0.1)
        [1.0, 1.0, 1.15, 1.15]
    """
    if epsilon < 0:
        raise ValueError("epsilon must be non-negative")
    if not values:
        return []
    
    filtered = [values[0]]
    for v in values[1:]:
        if abs(v - filtered[-1]) <= epsilon:
            filtered.append(filtered[-1])  # hold previous
        else:
            filtered.append(v)
    return filtered
```

```python
from typing import List

def deadband_filter(values: List[float], epsilon: float) -> List[float]:
    """
    Implements a deadband filter for sensor data.
    
    Parameters:
    values (List[float]): The input sensor readings.
    epsilon (float): The deadband threshold. If the difference between
                     the current reading and the last output is <= epsilon,
                     the last output is repeated.
    
    Returns:
    List[float]: The filtered values.
    
    Example:
    >>> deadband_filter([1.0, 1.05, 1.2, 1.18], 0.1)
    [1.0, 1.0, 1.2, 1.2]
    """
    if epsilon < 0:
        raise ValueError("Epsilon cannot be negative")
    if len(values) == 0:
        return []
    
    result = [values[0]]
    last_output = values[0]
    
    for i in range(1, len(values)):
        current = values[i]
        if abs(current - last_output) <= epsilon:
            result.append(last_output)
        else:
            result.append(current)
            last_output = current
    return result
```

## **4. Function 2: parse_fleet_log**

**Task Description:** Parse a fleet log line format: `"2026-04-20 08:15 [AGENT] [LEVEL] message @key=value key2=value2"`. Return a dict with keys: `timestamp`, `agent`, `level`, `message`, `kv_pairs` (dict). If format invalid, return None.

**Input:** `log_line: str`
**Output:** `Optional[Dict[str, Any]]`
**Edge cases:** missing brackets, no kv pairs, extra spaces, malformed kv.

### **DeepSeek-chat output (simulated):**

```python
import re
from typing import Optional, Dict, Any

def parse_fleet_log(log_line: str) -> Optional[Dict[str, Any]]:
    """
    Parse a structured fleet log line.
    
    Fleet logs follow: TIMESTAMP [AGENT] [LEVEL] message @k=v k2=v2
    Used by Oracle1, JetsonClaw1, Forgemaster for monitoring.
    
    Args:
        log_line: A single log line.
    
    Returns:
        Dict with keys: timestamp, agent, level, message, kv_pairs.
        Returns None if format cannot be parsed.
    
    Example:
        >>> parse_fleet_log('2026-04-20 08:15 [Alchemist] [INFO] Task started @cycle=259')
        {'timestamp': '2026-04-20 08:15', 'agent': 'Alchemist', 'level': 'INFO',
         'message': 'Task started', 'kv_pairs': {'cycle': '259'}}
    """
    pattern = r'^(\d{4}-\d{2}-\d{2} \d{2}:\d{2}) \[([^\]]+)\] \[([^\]]+)\] (.+?)(?: @(.+))?$'
    match = re.match(pattern, log_line.strip())
    if not match:
        return None
    
    timestamp, agent, level, message, kv_str = match.groups()
    kv_pairs = {}
    if kv_str:
        for pair in kv_str.split():
            if '=' in pair:
                k, v = pair.split('=', 1)
                kv_pairs[k] = v
    
    return {
        'timestamp': timestamp,
        'agent': agent,
        'level': level,
        'message': message.strip(),
        'kv_pairs': kv_pairs
    }
```
