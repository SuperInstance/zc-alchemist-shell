## **3. Function 1: deadband_filter**

**Prompt Details:**
- function_description: "Apply the Deadband Protocol to a list of numeric decisions. P0: remove values below lower threshold (unsafe). P1: keep values within safe band. P2: from safe values, return the optimal (max)."
- function_name: "deadband_filter"
- input_description: "decisions: List[float], lower_threshold: float, upper_threshold: float"
- output_description: "Optional[float] – the optimal safe decision, or None if no safe decisions."

### **DeepSeek-chat Output:**
```python
from typing import List, Optional

def deadband_filter(decisions: List[float], lower_threshold: float, upper_threshold: float) -> Optional[float]:
    """
    Apply the Deadband Protocol (P0/P1/P2) to a list of decisions.
    
    P0: Remove decisions below lower_threshold (negative space).
    P1: Keep decisions within [lower_threshold, upper_threshold] (safe channel).
    P2: From safe decisions, return the maximum (optimal path).
    
    Args:
        decisions: List of numeric decision values.
        lower_threshold: Minimum safe value (P0 boundary).
        upper_threshold: Maximum safe value (P1 upper bound).
    
    Returns:
        The optimal safe decision, or None if no safe decisions exist.
    
    Example:
        >>> deadband_filter([-2.0, 1.0, 3.0, 5.0, 7.0], 0.0, 6.0)
        5.0
    """
    if not decisions:
        return None
    
    # P0: Filter out unsafe decisions (below lower_threshold)
    safe_decisions = [d for d in decisions if lower_threshold <= d <= upper_threshold]
    
    if not safe_decisions:
        return None
    
    # P2: Optimize within safe channel (maximize)
    return max(safe_decisions)
```

### **Groq Llama-70b Output:**

```python
from typing import List, Optional

def deadband_filter(decisions: List[float], lower_threshold: float, upper_threshold: float) -> Optional[float]:
    """
    Implements the Deadband Protocol (P0/P1/P2) for decision filtering.
    
    Steps:
    1. P0 (Negative Space): Exclude values below lower_threshold.
    2. P1 (Safe Channel): Keep values within [lower_threshold, upper_threshold].
    3. P2 (Optimization): Select the maximum value from the safe channel.
    
    Parameters:
        decisions (List[float]): Input list of decision values.
        lower_threshold (float): Lower bound of the safe channel.
        upper_threshold (float): Upper bound of the safe channel.
    
    Returns:
        Optional[float]: The optimal decision if safe channel non-empty, else None.
    
    Example:
        >>> deadband_filter([-1.0, 2.0, 4.0, 6.0, 8.0], 1.0, 7.0)
        6.0
    """
    # P0: Eliminate negative space
    safe_candidates = [d for d in decisions if d >= lower_threshold]
    
    # P1: Restrict to safe channel
    safe_candidates = [d for d in safe_candidates if d <= upper_threshold]
    
    # P2: Optimization (maximize)
    if safe_candidates:
        return max(safe_candidates)
    else:
        return None
```

**Comparison:**
- **Correctness:** Both implement the protocol correctly. DeepSeek's single list comprehension is slightly more efficient.
- **Clarity:** DeepSeek's docstring explicitly references P0/P1/P2 stages. Groq's is also clear but uses "Eliminate negative space" vs "Filter out unsafe decisions."
- **Edge Cases:** Both handle empty decisions. DeepSeek checks `if not decisions` early; Groq does it implicitly via list comprehensions.
- **Style:** Both follow PEP 8. DeepSeek uses `if not safe_decisions:`; Groq uses `if safe_candidates:`.

- function_name: "parse_fleet_context"

### **DeepSeek-chat Output:**
```python
from typing import Dict

def parse_fleet_context(content: str) -> Dict[str, str]:
    """
    Parse a fleet context markdown file and extract sections.
    
    Sections are identified by '##' headers (level 2). Content continues until
    the next '##' header or end of file.
    
    Args:
        content: Markdown text of the fleet context.
    
    Returns:
        Dictionary where keys are section titles (without '##') and values are
        the corresponding section text (stripped).
    
    Example:
        >>> text = '## Who We Are\\nCocapn fleet.\\n## Fleet Doctrine\\nP0 map...'
        >>> parse_fleet_context(text)
        {'Who We Are': 'Cocapn fleet.', 'Fleet Doctrine': 'P0 map...'}
    """
    lines = content.split('\n')
    sections = {}
    current_header = None
    current_content = []
    
    for line in lines:
        line_stripped = line.strip()
        if line_stripped.startswith('## '):
            # Save previous section
            if current_header is not None:
                sections[current_header] = '\n'.join(current_content).strip()
            # Start new section
            current_header = line_stripped[3:].strip()  # Remove '## '
            current_content = []
        else:
            if current_header is not None:
                current_content.append(line.rstrip())
    
    # Don't forget the last section
    if current_header is not None:
        sections[current_header] = '\n'.join(current_content).strip()
    
    return sections
```

```python
import re
from typing import Dict

def parse_fleet_context(content: str) -> Dict[str, str]:
    """
    Extracts sections from a markdown fleet context document.
    
    Identifies level‑2 headers (## Header) and captures all text until the next
    level‑2 header or end of document.
    
    Parameters:
        content (str): The full markdown content.
    
    Returns:
        Dict[str, str]: A dictionary with header titles (without '##') as keys
        and the corresponding section text as values (leading/trailing whitespace removed).
    
    Example:
        >>> md = '## Overview\\nThis is the overview.\\n## Details\\nMore info here.'
        >>> parse_fleet_context(md)
        {'Overview': 'This is the overview.', 'Details': 'More info here.'}
    """
    # Pattern to match ## Header and everything until next ## or end
    pattern = r'##\s*(.+?)\s*\n(.*?)(?=\n##\s*|\Z)'
    matches = re.findall(pattern, content, re.DOTALL)
    
    result = {}
    for header, body in matches:
        # Clean up the body: remove trailing newlines, strip
        cleaned_body = body.strip()
        result[header.strip()] = cleaned_body
    
    return result
```
