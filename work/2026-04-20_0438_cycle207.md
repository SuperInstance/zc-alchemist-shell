**Prompt given to both models:**

```
Write a Python function `parse_log_line` that takes a string log line in the format:
"YYYY-MM-DD HH:MM:SS LEVEL [MODULE] MESSAGE"
and returns a dict with keys: timestamp (datetime), level (str), module (str), message (str).
Handle malformed lines gracefully (return None). Include docstring and type hints.
```

#### **DeepSeek-chat output:**
```python
from datetime import datetime
from typing import Optional, Dict, Any
import re

def parse_log_line(log_line: str) -> Optional[Dict[str, Any]]:
    """
    Parse a structured log line into its components.

    Expected format: "YYYY-MM-DD HH:MM:SS LEVEL [MODULE] MESSAGE"
    
    Args:
        log_line: A string log entry.

    Returns:
        Dictionary with keys: timestamp (datetime), level (str), module (str), message (str).
        Returns None if line cannot be parsed.
    """
    pattern = r'^(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) (\w+) \[([^\]]+)\] (.+)$'
    match = re.match(pattern, log_line.strip())
    if not match:
        return None
    
    try:
        timestamp_str, level, module, message = match.groups()
        timestamp = datetime.strptime(timestamp_str, '%Y-%m-%d %H:%M:%S')
        return {
            'timestamp': timestamp,
            'level': level,
            'module': module,
            'message': message.strip()
        }
    except ValueError:
        return None
```

### **Function 2: `deadband_filter`**

**Prompt given:**
```
Write a Python function `deadband_filter` that applies the Deadband Protocol to a list of sensor readings.
P0: Remove any readings that are NaN or outside valid range [0, 100].
P1: Find "safe channels" – readings that are within 10% of the median of remaining values.
P2: From safe channels, return the reading closest to the median.
Return the filtered value or None if no safe reading exists. Include docstring and type hints.
```

#### **DeepSeek-chat output:**
```python
import numpy as np
from typing import Optional, List

def deadband_filter(readings: List[float]) -> Optional[float]:
    """
    Apply Deadband Protocol (P0/P1/P2) to sensor readings.
    
    P0: Remove NaN and readings outside [0, 100].
    P1: Find safe channels – readings within 10% of median of remaining values.
    P2: Return reading closest to median from safe channels.
    
    Args:
        readings: List of float sensor readings.
    
    Returns:
        Filtered reading (float) or None if no safe reading exists.
    """
    # P0: Filter out invalid readings
    valid = [r for r in readings if isinstance(r, (int, float)) and 0 <= r <= 100]
    if not valid:
        return None
    
    # Calculate median of valid readings
    median_val = np.median(valid)
    
    # P1: Safe channels (within 10% of median)
    tolerance = 0.1 * median_val
    safe = [r for r in valid if abs(r - median_val) <= tolerance]
    
    if not safe:
        return None
    
    # P2: Closest to median
    return min(safe, key=lambda x: abs(x - median_val))
```

```python
from typing import List, Optional
import math

def deadband_filter(readings: List[float]) -> Optional[float]:
    """
    Applies the Deadband Protocol to sensor readings.
    
    P0: Remove readings that are NaN or outside [0, 100].
    P1: Find safe channels – readings within 10% of median of remaining values.
    P2: From safe channels, return reading closest to median.
    
    Args:
        readings: List of sensor readings.
        
    Returns:
        Filtered reading or None if no safe reading exists.
    """
    # P0: Remove invalid readings
    valid_readings = []
    for r in readings:
        if isinstance(r, (int, float)) and not math.isnan(r) and 0 <= r <= 100:
            valid_readings.append(r)
    
    if not valid_readings:
        return None
    
    # Calculate median
    sorted_readings = sorted(valid_readings)
    n = len(sorted_readings)
    if n % 2 == 0:
        median = (sorted_readings[n//2 - 1] + sorted_readings[n//2]) / 2
    else:
        median = sorted_readings[n//2]
    
    # P1: Safe channels (within 10% of median)
    tolerance = 0.1 * median
    safe_channels = [r for r in valid_readings if abs(r - median) <= tolerance]
    
    if not safe_channels:
        return None
    
    # P2: Closest to median
    closest = min(safe_channels, key=lambda x: abs(x - median))
    return closest
```
