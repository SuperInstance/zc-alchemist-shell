**Prompt:**
```
Write a Python function called `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input: A list of floats `values` and a float `deadband_threshold`.
Protocol:
- P0 (Negative Space): Identify values where abs(value) < deadband_threshold. These are "rocks" to avoid. Return them as `p0_rocks`.

- P2 (Optimization Zone): Identify values where abs(value) >= 2 * deadband_threshold. These are where optimization can occur. Return them as `p2_optimize`.

The function should return a tuple: (p0_rocks, p1_channels, p2_optimize).

Include a docstring and type hints.
```

### **Function 2: `parse_fleet_log`**

**Prompt:**
```

Log Format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"

The function should:
- Take a single string `log_line`.
- Split on the " | " delimiter.

- Return a dictionary with keys: `timestamp`, `agent`, `cycle`, `status`, `message`.
- The `cycle` should be converted to an integer.
- If the format is invalid, raise a `ValueError` with a descriptive message.

Include a docstring, type hints, and a simple example in the docstring.
```

**DeepSeek-chat Output:**
```python
from typing import Dict, Any

def parse_fleet_log(log_line: str) -> Dict[str, Any]:
    """
    Parse a standardized Cocapn fleet agent log line.

    Expected format: "TIMESTAMP | AGENT | CYCLE | STATUS | MESSAGE"

    Parameters:
    -----------
    log_line : str
        A single log line string.

    Returns:
    --------
    Dict[str, Any]
        Dictionary with keys: 'timestamp', 'agent', 'cycle', 'status', 'message'.
        'cycle' is converted to integer.

    Raises:
    -------
    ValueError
        If the log line does not contain exactly 5 parts separated by " | ".

    Example:
    --------
    >>> parse_fleet_log("2026-04-20T19:10:00Z | Alchemist | 378 | ACTIVE | Task started.")
    {'timestamp': '2026-04-20T19:10:00Z', 'agent': 'Alchemist', 'cycle': 378,
     'status': 'ACTIVE', 'message': 'Task started.'}
    """
    parts = log_line.split(" | ")
    if len(parts) != 5:
        raise ValueError(f"Log line must have exactly 5 parts separated by ' | '. Got {len(parts)} parts: {log_line}")

    timestamp, agent, cycle_str, status, message = parts
    try:
        cycle = int(cycle_str)
    except ValueError:
        raise ValueError(f"Cycle must be an integer. Got '{cycle_str}'")

    return {
        "timestamp": timestamp,
        "agent": agent,
        "cycle": cycle,
        "status": status,
        "message": message
    }
```
