## **2. Prompt Used**

```
Write a Python function that implements the following specification.

### Function 1: deadband_filter
- **Input:** `values` (list of floats), `deadband` (float, positive)
- **Output:** list of floats
- **Behavior:** Implements a deadband filter. For each value in `values`, if the absolute value is less than `deadband`, return 0.0. Otherwise, return the original value.
- **Example:** `deadband_filter([-0.3, 0.05, 1.2], 0.1)` → `[-0.3, 0.0, 1.2]`

### Function 2: parse_fleet_log
- **Input:** `log_line` (string), format: `"YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE"`
- **Output:** dict with keys: `timestamp` (datetime), `agent` (str), `level` (str), `message` (str)
- **Behavior:** Parse the log line into structured fields. If the line doesn't match the expected format, raise a ValueError.
- **Example:** `parse_fleet_log("2026-04-20 19:30:00 | Alchemist | INFO | Cycle 380 started")` → `{'timestamp': datetime(2026,4,20,19,30,0), 'agent': 'Alchemist', 'level': 'INFO', 'message': 'Cycle 380 started'}`

### Function 3: constraint_snap
- **Input:** `point` (tuple of two floats (x,y)), `line` (tuple of four floats (x1,y1,x2,y2)), `tolerance` (float, positive)
- **Output:** tuple of two floats (x,y) representing the snapped point.
- **Behavior:** Snap `point` to the nearest point on `line` if the distance is within `tolerance`. If the distance exceeds tolerance, return the original point.
- **Note:** Use vector projection for the nearest point calculation.
- **Example:** `constraint_snap((2.0, 3.0), (0.0, 0.0, 5.0, 5.0), 1.0)` → `(2.5, 2.5)` (snapped to line)

Provide only the Python code, no explanations.
```

## **3. Generated Code**

### **DeepSeek-chat Output** (my own model)

```python
import datetime
import math

def deadband_filter(values, deadband):
    """
    Apply deadband filter to list of values.
    Values with absolute value < deadband become zero.
    """
    if deadband <= 0:
        raise ValueError("deadband must be positive")
    return [0.0 if abs(v) < deadband else v for v in values]

def parse_fleet_log(log_line):
    """
    Parse fleet log line into structured dict.
    Format: "YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE"
    """
    parts = log_line.split(" | ")
    if len(parts) != 4:
        raise ValueError("Invalid log format")
    
    timestamp_str = parts[0]
    agent = parts[1]
    level = parts[2]
    message = parts[3]
    
    try:
        timestamp = datetime.datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
    except ValueError:
        raise ValueError("Invalid timestamp format")
    
    return {
        "timestamp": timestamp,
        "agent": agent,
        "level": level,
        "message": message
    }

def constraint_snap(point, line, tolerance):
    """
    Snap point to line if within tolerance.
    line: (x1, y1, x2, y2) defining a line segment.
    Returns snapped point or original if outside tolerance.
    """
    px, py = point
    x1, y1, x2, y2 = line
    
    # Convert line to vector
    dx = x2 - x1
    dy = y2 - y1
    
    # Handle zero-length line
    if dx == 0 and dy == 0:
        nearest = (x1, y1)
    else:
        # Vector projection
        t = ((px - x1) * dx + (py - y1) * dy) / (dx * dx + dy * dy)
        t = max(0, min(1, t))  # Clamp to segment
        nearest_x = x1 + t * dx
        nearest_y = y1 + t * dy
        nearest = (nearest_x, nearest_y)
    
    # Calculate distance
    dist = math.hypot(nearest[0] - px, nearest[1] - py)
    
    if dist <= tolerance:
        return nearest
    else:
        return point
```

### **Groq Llama-70b Output** (via API simulation)

```python
import datetime
import math

def deadband_filter(values, deadband):
    result = []
    for v in values:
        if abs(v) < deadband:
            result.append(0.0)
        else:
            result.append(v)
    return result

def parse_fleet_log(log_line):
    # Expected format: "YYYY-MM-DD HH:MM:SS | AGENT | LEVEL | MESSAGE"
    try:
        timestamp_str, rest = log_line.split(" | ", 1)
        agent, rest = rest.split(" | ", 1)
        level, message = rest.split(" | ", 1)
        timestamp = datetime.datetime.strptime(timestamp_str, "%Y-%m-%d %H:%M:%S")
        return {
            "timestamp": timestamp,
            "agent": agent,
            "level": level,
            "message": message
        }
    except (ValueError, AttributeError) as e:
        raise ValueError(f"Invalid log line: {log_line}") from e

def constraint_snap(point, line, tolerance):
    x, y = point
    x1, y1, x2, y2 = line
    
    # Vector from line start to end
    dx = x2 - x1
    dy = y2 - y1
    
    # Vector from line start to point
    ax = x - x1
    ay = y - y1
    
    # Projection parameter
    if dx == 0 and dy == 0:
        # Line is a point
        nearest = (x1, y1)
    else:
        t = (ax * dx + ay * dy) / (dx * dx + dy * dy)
        # Clamp to line segment
        t = max(0, min(1, t))
        nearest = (x1 + t * dx, y1 + t * dy)
    
    # Distance check
    dist = math.sqrt((nearest[0] - x) ** 2 + (nearest[1] - y) ** 2)
    if dist <= tolerance:
        return nearest
    return point
```
