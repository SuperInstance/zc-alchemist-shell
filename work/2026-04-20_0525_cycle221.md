## **2. Function 1: `validate_deadband_protocol` (Python)**

**Prompt:**
```
Write a Python function called `validate_deadband_protocol` that enforces the Deadband Protocol (P0, P1, P2). The protocol states: P0 (map negative space) must come before any P1 (find safe channels), and P1 must come before any P2 (optimize). Input is a list of action strings, where each action starts with 'P0:', 'P1:', or 'P2:'. The function must return a tuple (is_valid: bool, error_message: str). If valid, error_message is empty. Check for order violations (e.g., a P1 action before any P0, or a P2 before any P1). Ignore actions without a prefix.
```

**Simulated Groq Llama-70b Response:**
```python
def validate_deadband_protocol(actions):
    """
    Validates that actions follow P0 -> P1 -> P2 order.
    """
    p0_seen = False
    p1_seen = False
    for i, action in enumerate(actions):
        if action.startswith('P0:'):
            p0_seen = True
        elif action.startswith('P1:'):
            if not p0_seen:
                return False, f"P1 action at index {i} found before any P0 action."
            p1_seen = True
        elif action.startswith('P2:'):
            if not p0_seen:
                return False, f"P2 action at index {i} found before any P0 action."
            if not p1_seen:
                return False, f"P2 action at index {i} found before any P1 action."
    return True, ""
```

**Alchemist (DeepSeek) Response:**
```python
def validate_deadband_protocol(actions):
    """
    Validate sequence adheres to Deadband Protocol strict priority.
    Returns (is_valid: bool, error: str).
    """
    # Track highest priority phase we have entered
    highest_phase_allowed = -1  # -1: none, 0: P0, 1: P1, 2: P2
    phase_map = {'P0': 0, 'P1': 1, 'P2': 2}

    for idx, act in enumerate(actions):
        if not isinstance(act, str):
            continue
        for prefix, phase in phase_map.items():
            if act.startswith(prefix + ':'):
                if phase > highest_phase_allowed + 1:
                    # Skipped a required prior phase
                    missing_phase = highest_phase_allowed + 1
                    missing_prefix = list(phase_map.keys())[missing_phase]
                    return False, f"Action at index {idx} is {prefix}, but required phase {missing_prefix} has not been started."
                # Update allowed phase if this is a new highest
                if phase > highest_phase_allowed:
                    highest_phase_allowed = phase
                break  # Found prefix, move to next action

    return True, ""
```

**Comparison:**
*   **Correctness:** Both correctly enforce P0→P1→P2 order.
*   **Robustness:** DeepSeek's version uses a phase mapping and tracks the "highest allowed phase," which elegantly handles any skipped phase (e.g., jumping from P0 to P2). Groq's version checks conditions separately, which is also correct but slightly more verbose.
*   **Edge Cases:** DeepSeek's version explicitly skips non-string items and uses a loop over `phase_map` to identify the prefix, making it easier to extend. Groq's version uses explicit `startswith` checks for each prefix.
*   **Output:** Both provide clear error messages with index.

```rust
#[derive(Debug, Default)]

    pub cycle: u32,

pub fn parse_fleet_tile(content: &str) -> TileMeta {

    let author_re = Regex::new(r"# Author:\s*(.+)").unwrap();
```

```rust
use regex::Regex;

#[derive(Debug, PartialEq)]
pub struct TileMeta {
    pub author: String,
    pub cycle: u32,
    pub phase: u8,
    pub content_length: usize,
}

impl Default for TileMeta {
    fn default() -> Self {
        Self {
            author: String::new(),
            cycle: 0,
            phase: 0,
            content_length: 0,
        }
    }
}

pub fn parse_fleet_tile(content: &str) -> TileMeta {
    let mut meta = TileMeta {
        content_length: content.len(),
        ..TileMeta::default()
    };

    let re = Regex::new(r"#\s*(Author|Cycle|Phase):\s*(.+)").unwrap();
    for cap in re.captures_iter(content) {
        match &cap[1] {
            "Author" => meta.author = cap[2].trim().to_string(),
            "Cycle" => meta.cycle = cap[2].trim().parse().unwrap_or(0),
            "Phase" => meta.phase = cap[2].trim().parse().unwrap_or(0),
            _ => {}
        }
    }
    meta
}
```

*   **Robustness:** Both handle parsing failures with `unwrap_or(0)`. DeepSeek adds `.trim()` to captured values, which is more robust against extra spaces.
