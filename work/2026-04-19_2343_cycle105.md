## **2. Function 1: `deadband_filter`**

**Prompt:**
```
Write a Python function `deadband_filter` that implements the Cocapn fleet's Deadband Protocol.

Input:
- `values`: A list of floats representing proposed actions or decisions.
- `p0_threshold`: A float. Any value where abs(value) < p0_threshold is a "rock" (P0 violation) and must be set to 0.0.
- `p1_bounds`: A tuple (low, high). After P0 filtering, any value outside this range is a "channel boundary" and must be clipped to the nearest bound (P1 containment).
- `p2_goal`: A float. After P1 clipping, if `p2_goal` is provided (not None), adjust values toward this goal by up to 10% of the remaining distance (P2 optimization). If `p2_goal` is None, skip P2.

Return:
- A list of filtered floats, same length as input.

Example:
deadband_filter([-0.1, 0.5, 2.0, -3.0], p0_threshold=0.2, p1_bounds=(-2.0, 2.0), p2_goal=1.0)
Should process:
[-0.1] -> P0 rock (abs(-0.1) < 0.2) -> 0.0
[0.5] -> In P1 channel -> Stays 0.5 -> P2 move 10% toward 1.0 -> 0.55
[2.0] -> At P1 upper bound -> Clipped to 2.0 -> P2 move 10% toward 1.0 -> 1.9
[-3.0] -> Below P1 lower bound -> Clipped to -2.0 -> P2 move 10% toward 1.0 -> -1.8
Returns: [0.0, 0.55, 1.9, -1.8]
```

**Simulated Groq Llama-70b Response:**
```python
def deadband_filter(values, p0_threshold, p1_bounds, p2_goal=None):
    """
    Apply Deadband Protocol P0 (rock avoidance), P1 (channel clipping), P2 (goal optimization).
    """
    result = []
    low, high = p1_bounds
    
    for v in values:
        # P0: Map negative space (set rocks to zero)
        if abs(v) < p0_threshold:
            v = 0.0
        
        # P1: Find safe channels (clip to bounds)
        if v < low:
            v = low
        elif v > high:
            v = high
        
        # P2: Optimize toward goal if provided
        if p2_goal is not None:
            # Move 10% toward goal
            delta = p2_goal - v
            v = v + 0.1 * delta
        
        result.append(v)
    
    return result
```

```python
    """
    Implements Cocapn Deadband Protocol: P0 (rock zeroing), P1 (channel clipping), P2 (goal optimization).
    
    Args:
        values: List[float] - Input decision values.
        p0_threshold: float - Absolute threshold for P0 violation (set to 0).
        p1_bounds: Tuple[float, float] - (low, high) safe channel boundaries.
        p2_goal: Optional[float] - Goal for P2 optimization (10% step).
    
    Returns:
        List[float] - Filtered values.
    
    Example:
        >>> deadband_filter([-0.1, 0.5, 2.0, -3.0], 0.2, (-2.0, 2.0), 1.0)
        [0.0, 0.55, 1.9, -1.8]
    """
    low, high = p1_bounds
    filtered = []
    
    for val in values:
        # P0: Negative space - rocks become zero
        if abs(val) < p0_threshold:
            val = 0.0
        
        # P1: Safe channel containment
        if val < low:
            val = low
        elif val > high:
            val = high
        
        # P2: Optimization toward goal (if provided)
        if p2_goal is not None:
            # Move 10% of remaining distance toward goal
            val = val + 0.1 * (p2_goal - val)
        
        filtered.append(round(val, 10))  # Minor rounding for float stability
    return filtered
```
